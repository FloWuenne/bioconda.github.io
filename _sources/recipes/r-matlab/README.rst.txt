Emulate MATLAB code using R.

======== ===========

Versions 1.0.2
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-matlab
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-matlab

and update with::

   conda update r-matlab



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-matlab.
