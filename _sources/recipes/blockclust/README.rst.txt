blockclust
==========

Efficient clustering and classification of non\-coding RNAs from short read RNA\-seq profiles.

Versions      1.1.0
License       GPL

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install blockclust

and update with::

   conda update blockclust



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/blockclust.
