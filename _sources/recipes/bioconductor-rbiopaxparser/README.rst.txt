bioconductor-rbiopaxparser
==========================

|downloads|

Parses BioPAX files and represents them in R\, at the moment BioPAX level 2 and level 3 are supported.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/rBiopaxParser.html
Versions      2.20.0, 2.18.0, 2.16.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rbiopaxparser



Links         biotools: :biotools:`rbiopaxparser`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rbiopaxparser

and update with::

   conda update bioconductor-rbiopaxparser



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rbiopaxparser.
