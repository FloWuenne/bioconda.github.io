perl-try-tiny-retry
===================

|downloads|

Just like Try\:\:Tiny\, but with retry instead of try.

Versions      0.002
License       Apache-2.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-try-tiny-retry

and update with::

   conda update perl-try-tiny-retry



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-try-tiny-retry.
