.. _`r-phewas`:

r-phewas
========

|downloads|

Phenome Wide Association Studies \(PheWAS\) \- Functions to perform Phenome Wide Association Studies \(PheWAS\). These functions include the conversion of ICD9 codes to PheWAS codes \(v1.2\)\, statistical analysis\, and plotting.

============= ===========
Home          https://github.com/PheWAS/PheWAS
Versions      0.12.1
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-phewas



Links         doi: :doi:`10.1093/bioinformatics/btu197`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-phewas

and update with::

   conda update r-phewas



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-phewas.
