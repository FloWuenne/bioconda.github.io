.. _`bioconductor-fis`:

bioconductor-fis
================

|downloads|

Data set containing two complete lists of identified functional interaction partners in Human. Data are derived from Reactome and BioGRID databases.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/FIs.html
Versions      1.8.0, 1.6.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-fis/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-fis

and update with::

   conda update bioconductor-fis



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-fis.
