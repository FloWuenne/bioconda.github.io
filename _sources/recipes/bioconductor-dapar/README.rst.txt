bioconductor-dapar
==================

|downloads|

This package contains a collection of functions for the visualisation and the statistical analysis of proteomic data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/DAPAR.html
Versions      1.14.4
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-dapar/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-dapar

and update with::

   conda update bioconductor-dapar



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-dapar.
