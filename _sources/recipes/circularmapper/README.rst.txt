circularmapper
==============

|downloads|

 A method to improve mappings on circular genomes\, using the BWA mapper.

============= ===========
Home          https://github.com/apeltzer/CircularMapper
Versions      1.93.4
License       GPLv3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//circularmapper/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install circularmapper

and update with::

   conda update circularmapper



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/circularmapper.
