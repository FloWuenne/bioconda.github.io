Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install stacks

and update with::

   conda update stacks



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/stacks.
