metabat2
========

|downloads|

MetaBAT2 is a program for binning metagenome\-assembled contigs into bins\/MAGs.

============= ===========
Home          https://bitbucket.org/berkeleylab/metabat
Versions      2.12.1
License       Academic and Research License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/metabat2



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install metabat2

and update with::

   conda update metabat2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/metabat2.
