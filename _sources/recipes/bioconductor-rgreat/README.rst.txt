bioconductor-rgreat
===================

|downloads|

This package makes GREAT \(Genomic Regions Enrichment of Annotations Tool\) analysis automatic by constructing a HTTP POST request according to user\'s input and automatically retrieving results from GREAT web server.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/rGREAT.html
Versions      1.12.1, 1.11.1, 1.8.0
License       MIT + LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rgreat



Links         biotools: :biotools:`rgreat`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rgreat

and update with::

   conda update bioconductor-rgreat



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rgreat.
