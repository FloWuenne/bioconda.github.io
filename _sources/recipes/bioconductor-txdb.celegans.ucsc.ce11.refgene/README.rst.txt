Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-txdb.celegans.ucsc.ce11.refgene

and update with::

   conda update bioconductor-txdb.celegans.ucsc.ce11.refgene
