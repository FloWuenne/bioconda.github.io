bioconductor-bsgenome.scerevisiae.ucsc.saccer1
==============================================

|downloads|

Saccharomyces cerevisiae \(Yeast\) full genome as provided by UCSC \(sacCer1\, Oct. 2003\) and stored in Biostrings objects.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.scerevisiae.ucsc.saccer1

and update with::

   conda update bioconductor-bsgenome.scerevisiae.ucsc.saccer1

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.scerevisiae.ucsc.saccer1.
