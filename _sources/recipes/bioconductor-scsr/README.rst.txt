bioconductor-scsr
=================

|downloads|

Corrects genome\-wide siRNA screens for seed mediated off\-target effect. Suitable functions to identify the effective seeds\/miRNAs and to visualize their effect are also provided in the package.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/scsR.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-scsr/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-scsr

and update with::

   conda update bioconductor-scsr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-scsr.
