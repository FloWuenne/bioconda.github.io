bioconductor-lymphoseq
======================

.. conda:recipe:: bioconductor-lymphoseq
   :replaces_section_title:

   This R package analyzes high\-throughput sequencing of T and B cell receptor complementarity determining region 3 \(CDR3\) sequences generated by Adaptive Biotechnologies\' ImmunoSEQ assay.  Its input comes from tab\-separated value \(.tsv\) files exported from the ImmunoSEQ analyzer.

   :homepage: https://bioconductor.org/packages/3.8/bioc/html/LymphoSeq.html
   :license: Artistic-2.0
   :recipe: /`bioconductor-lymphoseq <https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-lymphoseq>`_/`meta.yaml <https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-lymphoseq/meta.yaml>`_
