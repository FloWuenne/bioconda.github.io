.. _`r-seqminer`:

r-seqminer
==========

|downloads|

Integrate sequencing data \(Variant call format\, e.g. VCF or BCF\) or meta\-analysis results in R.

============= ===========
Home          https://cran.r-project.org/web/packages/seqminer/index.html
Versions      6.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-seqminer



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-seqminer

and update with::

   conda update r-seqminer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-seqminer.
