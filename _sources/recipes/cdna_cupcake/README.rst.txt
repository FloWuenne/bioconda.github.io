cdna_cupcake
============

|downloads|

cDNA\_Cupcake is a miscellaneous collection of Python and R scripts used for analyzing sequencing data.

============= ===========
Home          https://github.com/Magdoll/cDNA_Cupcake
Versions      5.8, 5.3
License       BSD-3-Clause-Clear
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/cdna_cupcake



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cdna_cupcake

and update with::

   conda update cdna_cupcake



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cdna_cupcake.
