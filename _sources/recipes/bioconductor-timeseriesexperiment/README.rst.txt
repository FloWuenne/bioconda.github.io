.. _`bioconductor-timeseriesexperiment`:

bioconductor-timeseriesexperiment
=================================

|downloads|

Visualization and analysis toolbox for short time course data which includes dimensionality reduction\, clustering\, two\-sample differential expression testing and gene ranking techniques. The package also provides methods for retrieving enriched pathways.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/TimeSeriesExperiment.html
Versions      
License       LGPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-timeseriesexperiment



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-timeseriesexperiment

and update with::

   conda update bioconductor-timeseriesexperiment



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-timeseriesexperiment.
