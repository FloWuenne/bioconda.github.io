bioconductor-charge
===================

|downloads|

Identifies genomic duplications or deletions from gene expression data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/CHARGE.html
Versions      1.2.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-charge/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-charge

and update with::

   conda update bioconductor-charge



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-charge.
