r-wicket
========

|downloads|

Utilities to generate bounding boxes from \'WKT\' \(Well\-Known Text\) objects and R data types\, validate \'WKT\' objects and convert object types from the \'sp\' package into \'WKT\' representations.

============= ===========
Home          https://github.com/ropensci/wicket
Versions      0.4.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/r-wicket/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-wicket

and update with::

   conda update r-wicket



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-wicket.
