connectome-workbench
====================

|downloads|

Connectome Workbench is an open source\, freely available visualization and discovery tool used to map neuroimaging data\, especially data generated by the Human Connectome Project

============= ===========
Home          https://www.humanconnectome.org/software/connectome-workbench
Versions      1.3.2, 1.3.1
License       GPL-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/connectome-workbench/meta.yaml


Development   https://github.com/Washington-University/workbench


============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install connectome-workbench

and update with::

   conda update connectome-workbench



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/connectome-workbench.
