sharedmem

Dispatch your trivially parallizable jobs with sharedmem.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sharedmem

and update with::

   conda update sharedmem
