snp-dists
=========

|downloads|

Convert a FASTA alignment to SNP distance matrix

======== ===========
Home     https://github.com/tseemann/snp-dists
Versions 0.2
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/snp-dists
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install snp-dists

and update with::

   conda update snp-dists



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/snp-dists.
