r-proxy
=======

|downloads|

Provides an extensible framework for the efficient calculation of auto- and cross-proximities, along with implementations of the most popular ones. 

======== ===========
Home     https://CRAN.R-project.org/package=proxy
Versions 0.4_16
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-proxy
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-proxy

and update with::

   conda update r-proxy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-proxy.
