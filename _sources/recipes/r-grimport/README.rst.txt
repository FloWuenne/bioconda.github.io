r-grimport
==========

|downloads|

Functions for converting\, importing\, and drawing PostScript  pictures in R plots.

============= ===========
Home          https://CRAN.R-project.org/package=grImport
Versions      0.9_0, 0.9_1, 0.9_1.1
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-grimport



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-grimport

and update with::

   conda update r-grimport



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-grimport.
