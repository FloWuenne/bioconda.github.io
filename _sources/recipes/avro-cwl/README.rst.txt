.. _`avro-cwl`:

avro-cwl
========

|downloads|

Avro is a serialization and RPC framework. This package is a fork of regular avro made by the CWL team in order to fix some issues \(https\:\/\/github.com\/common\-workflow\-language\/cwltool\/issues\/524\)

============= ===========
Home          https://pypi.python.org/pypi?:action=display&name=avro-cwl
Versions      1.8.9, 1.8.4
License       Apache 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/avro-cwl



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install avro-cwl

and update with::

   conda update avro-cwl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/avro-cwl.
