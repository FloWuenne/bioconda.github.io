nanoplot
========

Plotting suite for Oxford Nanopore sequencing data and alignments

Home          https://github.com/wdecoster/NanoPlot
Versions      0.16.2, 0.16.4, 1.0.0, 1.1.0, 1.2.2, 1.8.1
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nanoplot



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanoplot

and update with::

   conda update nanoplot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanoplot.
