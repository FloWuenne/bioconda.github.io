bioconductor-genomicscores
==========================

|downloads|

Provide infrastructure to store and access genomewide position-specific scores within R and Bioconductor.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/GenomicScores.html
Versions 1.0.2
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-genomicscores
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genomicscores

and update with::

   conda update bioconductor-genomicscores



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genomicscores.
