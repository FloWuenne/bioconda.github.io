crux-toolkit
============

|downloads|

A cross\-platform suite of analysis tools for interpreting protein mass spectrometry data

============= ===========
Home          http://crux.ms
Versions      3.2
License       Apache-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/crux-toolkit


Development   https://github.com/crux-toolkit/crux-toolkit


============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install crux-toolkit

and update with::

   conda update crux-toolkit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/crux-toolkit.
