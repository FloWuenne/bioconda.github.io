bioconductor-flowviz
====================

|downloads|

Provides visualization tools for flow cytometry data.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/flowViz.html
Versions 1.38.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowviz
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowviz

and update with::

   conda update bioconductor-flowviz



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowviz.
