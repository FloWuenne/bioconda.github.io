Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install atropos

and update with::

   conda update atropos



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/atropos.
