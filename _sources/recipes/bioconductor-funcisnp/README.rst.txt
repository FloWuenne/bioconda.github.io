bioconductor-funcisnp
=====================

|downloads|

FunciSNP integrates information from GWAS\, 1000genomes and chromatin feature to identify functional SNP in coding or non\-coding regions.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/FunciSNP.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-funcisnp/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-funcisnp

and update with::

   conda update bioconductor-funcisnp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-funcisnp.
