gdc
===

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gdc

and update with::

   conda update gdc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gdc.
