pgma-simple
===========

|downloads|

Pgma is a simple program for building WPGMA trees\.

======== ===========
Home     https://github.com/BackofenLab/GraphClust
Versions 0.1
License  GNUv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/pgma-simple

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pgma-simple

and update with::

   conda update pgma-simple



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pgma-simple.
