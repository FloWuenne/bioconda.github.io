bioconductor-tximeta
====================

|downloads|

Transcript quantification import from Salmon with automatic population of metadata and transcript ranges. Filtered\, combined\, or de novo transcriptomes can be linked to the appropriate sources with linkedTxomes and shared for reproducible analyses.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/tximeta.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-tximeta/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-tximeta

and update with::

   conda update bioconductor-tximeta



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-tximeta.
