r-gwasexacthw
=============

|downloads|

This package contains a function to do exact Hardy\-Weinburg testing \(using Fisher\'s test\) for SNP genotypes as typically obtained in a Genome Wide Association Study \(GWAS\).

============= ===========
Home          https://CRAN.R-project.org/package=GWASExactHW
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-gwasexacthw



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-gwasexacthw

and update with::

   conda update r-gwasexacthw



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-gwasexacthw.
