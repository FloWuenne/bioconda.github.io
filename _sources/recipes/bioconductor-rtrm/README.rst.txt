bioconductor-rtrm
=================

|downloads|

rTRM identifies transcriptional regulatory modules \(TRMs\) from protein\-protein interaction networks.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/rTRM.html
Versions      1.20.0, 1.18.0, 1.16.0, 1.14.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-rtrm/meta.yaml



Links         biotools: :biotools:`rtrm`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rtrm

and update with::

   conda update bioconductor-rtrm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rtrm.
