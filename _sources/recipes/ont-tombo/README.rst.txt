ont-tombo
=========

|downloads|

Detection of modified bases from raw nanopore sequencing data.

======== ===========
Home     https://github.com/nanoporetech/tombo
Versions 1.0, 1.1, 1.1.1
License  mpl-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ont-tombo
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ont-tombo

and update with::

   conda update ont-tombo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ont-tombo.
