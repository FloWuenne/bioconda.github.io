pydnase
=======

|downloads|

DNase\-seq analysis library

============= ===========
Home          http://jpiper.github.io/pyDNase
Versions      0.3.0, 0.2.6
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//pydnase/meta.yaml

Documentation http://pythonhosted.org/pyDNase/index.html



============= ===========

A suite of tools for analysing DNase\-seq data.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pydnase

and update with::

   conda update pydnase



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pydnase.
