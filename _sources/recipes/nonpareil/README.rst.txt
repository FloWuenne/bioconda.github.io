nonpareil
=========

|downloads|

Estimate average coverage and create Nonpareil curves for metagenomic datasets

============= ===========
Home          http://nonpareil.readthedocs.io/
Versions      2.4, 2.4.01, 3.1.1, 3.2
License       Artistic License 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nonpareil/2.4



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nonpareil

and update with::

   conda update nonpareil



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nonpareil.
