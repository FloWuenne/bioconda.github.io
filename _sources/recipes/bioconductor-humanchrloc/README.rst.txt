bioconductor-humanchrloc
========================

|downloads|

Annotation data file for humanCHRLOC assembled using data from public data repositories

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/humanCHRLOC.html
Versions      2.1.6
License       The Artistic License, Version 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-humanchrloc/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-humanchrloc

and update with::

   conda update bioconductor-humanchrloc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-humanchrloc.
