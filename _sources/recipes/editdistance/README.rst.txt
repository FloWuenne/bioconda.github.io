editdistance
============

|downloads|

Fast implementation of the edit distance (Levenshtein distance)

======== ===========
Home     https://www.github.com/aflc/editdistance
Versions 0.3.1, 0.4
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/editdistance
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install editdistance

and update with::

   conda update editdistance



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/editdistance.
