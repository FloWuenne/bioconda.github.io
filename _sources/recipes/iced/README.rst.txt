.. _`iced`:

iced
====

|downloads|

The python module iced implements the ICE normalization of hic data.

============= ===========
Home          https://github.com/hiclib/iced
Versions      0.5.0, 0.4.2
License       new BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/iced



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install iced

and update with::

   conda update iced



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/iced.
