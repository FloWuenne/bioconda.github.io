============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-ngsplot

and update with::

   conda update r-ngsplot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-ngsplot.
