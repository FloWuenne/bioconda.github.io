karect
======

Read error correction tool based on multiple alignment.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install karect

and update with::

   conda update karect



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/karect.
