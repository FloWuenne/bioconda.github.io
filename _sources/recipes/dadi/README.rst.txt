.. _`dadi`:

dadi
====

|downloads|

DADI implements methods for demographic history and selection inference from genetic data\, based on diffusion approximations to the allele frequency spectrum.

============= ===========
Home          https://bitbucket.org/gutenkunstlab/dadi
Versions      1.7.0
License       Custom
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//dadi/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dadi

and update with::

   conda update dadi



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/dadi.
