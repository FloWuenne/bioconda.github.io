bioconductor-bsgenome.ptroglodytes.ucsc.pantro3
===============================================

|downloads|

Full genome sequences for Pan troglodytes \(Chimp\) as provided by UCSC \(panTro3\, Oct. 2010\) and stored in Biostrings objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Ptroglodytes.UCSC.panTro3.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-bsgenome.ptroglodytes.ucsc.pantro3



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.ptroglodytes.ucsc.pantro3

and update with::

   conda update bioconductor-bsgenome.ptroglodytes.ucsc.pantro3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.ptroglodytes.ucsc.pantro3.
