bioconductor-rcyjs
==================

|downloads|

Interactive viewing and exploration of graphs\, connecting R to Cytoscape.js\, using websockets.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/RCyjs.html
Versions

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rcyjs

and update with::

   conda update bioconductor-rcyjs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rcyjs.
