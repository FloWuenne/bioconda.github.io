bioconductor-foldgo
===================

|downloads|

FoldGO is a package designed to annotate gene sets derived from expression experiments and identify fold\-change\-specific GO terms.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/FoldGO.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-foldgo



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-foldgo

and update with::

   conda update bioconductor-foldgo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-foldgo.
