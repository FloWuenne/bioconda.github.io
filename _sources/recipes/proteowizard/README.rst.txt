Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install proteowizard

and update with::

   conda update proteowizard

A Docker container is available at https://quay.io/repository/biocontainers/proteowizard.
