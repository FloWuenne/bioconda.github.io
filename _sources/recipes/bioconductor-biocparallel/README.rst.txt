bioconductor-biocparallel
=========================

|downloads|

This package provides modified versions and novel implementation of functions for parallel evaluation\, tailored to use with Bioconductor objects.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/BiocParallel.html
Versions      1.14.2, 1.12.0, 1.10.1, 1.6.6, 1.5.0, 1.4.3, 1.4.0, 1.2.22
License       GPL-2 | GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-biocparallel



Links         biotools: :biotools:`biocparallel`, doi: :doi:`10.1214/14-STS476`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-biocparallel

and update with::

   conda update bioconductor-biocparallel



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-biocparallel.
