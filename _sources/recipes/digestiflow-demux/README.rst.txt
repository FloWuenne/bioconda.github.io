digestiflow-demux
=================

|downloads|

A command line client tool to perform semiautomatic demultiplexing of Illumina flowcells using data from Digestiflow Web.

============= ===========
Home          https://github.com/bihealth/digestiflow-demux
Versions      0.2.0, 0.1.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/digestiflow-demux/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install digestiflow-demux

and update with::

   conda update digestiflow-demux



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/digestiflow-demux.
