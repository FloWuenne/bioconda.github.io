bioconductor-cytoml
===================

|downloads|

Uses platform\-specific implemenations of the GatingML2.0 standard to exchange gated cytometry data with other software platforms.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/CytoML.html
Versions

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cytoml

and update with::

   conda update bioconductor-cytoml



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cytoml.
