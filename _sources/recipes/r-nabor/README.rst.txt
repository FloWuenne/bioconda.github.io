.. _`r-nabor`:

r-nabor
=======

|downloads|

An R wrapper for \'libnabo\'\, an exact or approximate k nearest neighbour library which is optimised for low dimensional spaces \(e.g. 3D\)

============= ===========
Home          https://CRAN.R-project.org/package=nabor
Versions      0.5.0
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-nabor



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-nabor

and update with::

   conda update r-nabor



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-nabor.
