Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanofilt

and update with::

   conda update nanofilt



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanofilt.
