bioconductor-tcgawgbsdata.hg19
==============================

|downloads|

Data package for WGBS Data in TCGA. Data is stored as SummarizedExperiment Format. See vignette on how to extract the data and perform differential methylation analysis.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/tcgaWGBSData.hg19.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-tcgawgbsdata.hg19



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-tcgawgbsdata.hg19

and update with::

   conda update bioconductor-tcgawgbsdata.hg19



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-tcgawgbsdata.hg19.
