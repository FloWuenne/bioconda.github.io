======== ===========
Home     https://www.cgat.org/downloads/public/cgatpipelines/documentation

License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/cgat-pipelines-nosetests
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cgat-pipelines-nosetests

and update with::

   conda update cgat-pipelines-nosetests



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cgat-pipelines-nosetests.
