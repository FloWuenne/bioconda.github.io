picopore
========

|downloads|

A tool for reducing the size of Oxford Nanopore Technologies\' datasets without losing information.

============= ===========
Home          https://github.com/scottgigante/picopore
Versions      1.2.0, 1.1.5, 1.1.4, 1.1.2, 1.0.0, 0.2.2, 0.2.1
License       GNU General Public License (GPL)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//picopore/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install picopore

and update with::

   conda update picopore



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/picopore.
