bioconductor-clusterseq
=======================

|downloads|

Identification of clusters of co\-expressed genes based on their expression across multiple \(replicated\) biological samples.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/clusterSeq.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-clusterseq



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-clusterseq

and update with::

   conda update bioconductor-clusterseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-clusterseq.
