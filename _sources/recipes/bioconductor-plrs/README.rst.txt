bioconductor-plrs
=================

|downloads|

The present package implements a flexible framework for modeling the relationship between DNA copy number and gene expression data using Piecewise Linear Regression Splines \(PLRS\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/plrs.html
Versions      1.22.0
License       GPL (>=2.0)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-plrs/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-plrs

and update with::

   conda update bioconductor-plrs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-plrs.
