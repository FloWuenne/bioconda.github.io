bioconductor-platecore
======================

|downloads|

Provides basic S4 data structures and routines for analyzing plate based flow cytometry data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/plateCore.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-platecore/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-platecore

and update with::

   conda update bioconductor-platecore



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-platecore.
