bioconductor-ideoviz
====================

|downloads|

Plots data associated with arbitrary genomic intervals along chromosomal ideogram.

============= ===========
Home          https://bioconductor.org/packages/release/bioc/html/IdeoViz.html
Versions      1.10.0, 1.16.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ideoviz



Links         biotools: :biotools:`ideoviz`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ideoviz

and update with::

   conda update bioconductor-ideoviz



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ideoviz.
