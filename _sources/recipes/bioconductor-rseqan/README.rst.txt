Headers from the SeqAn C\+\+ library for easy of usage in R.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rseqan

and update with::

   conda update bioconductor-rseqan
