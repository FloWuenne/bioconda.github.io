.. _`r-rismed`:

r-rismed
========

|downloads|

A set of tools to extract bibliographic content from the National Center for Biotechnology Information \(NCBI\) databases\, including PubMed. The name RISmed is a portmanteau of RIS \(for Research Information Systems\, a common tag format for bibliographic data\) and PubMed.

============= ===========
Home          https://CRAN.R-project.org/package=RISmed
Versions      2.1.7
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//r-rismed/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rismed

and update with::

   conda update r-rismed



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rismed.
