.. _`seqmap`:

seqmap
======

|downloads|

SeqMap is a tool for mapping large amount of oligonucleotide to the genome.

============= ===========
Home          http://www-personal.umich.edu/~jianghui/seqmap/
Versions      1.0.13
License       Custom OSS
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/seqmap



Links         doi: :doi:`https://doi.org/10.1093/bioinformatics/btn429`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install seqmap

and update with::

   conda update seqmap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/seqmap.
