Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dsh-bio

and update with::

   conda update dsh-bio

A Docker container is available at https://quay.io/repository/biocontainers/dsh-bio.
