perl-test-exec
==============

Test that some code calls exec without terminating testing

Home          https://metacpan.org/pod/Test::Exec
Versions      0.04

Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-test-exec

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-test-exec

and update with::

   conda update perl-test-exec
