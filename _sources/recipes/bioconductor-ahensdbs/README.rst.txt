.. _`bioconductor-ahensdbs`:

bioconductor-ahensdbs
=====================

|downloads|

Supplies AnnotationHub with EnsDb Ensembl\-based annotation databases for all species. EnsDb SQLite databases are generated separately from Ensembl MySQL databases using functions from the ensembldb package employing the Ensembl Perl API.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/AHEnsDbs.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ahensdbs



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ahensdbs

and update with::

   conda update bioconductor-ahensdbs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ahensdbs.
