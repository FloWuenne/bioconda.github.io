cmv
===

|downloads|

A collection of tools for the visualisation of Hidden Markov Models \(HMMV\) and RNA\-family models \(CMV\).

============= ===========
Home          https://github.com/eggzilla/cmv
Versions      1.0.8, 1.0.7, 1.0.6, 1.0.5, 1.0.2
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/cmv/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cmv

and update with::

   conda update cmv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cmv.
