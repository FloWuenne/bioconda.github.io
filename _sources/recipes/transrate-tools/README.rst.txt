transrate-tools
===============

Command\-line tools used by transrate for processing bam files.

License       MIT

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install transrate-tools

and update with::

   conda update transrate-tools
