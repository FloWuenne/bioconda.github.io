crimson
=======

|downloads|

Bioinformatics tool outputs converter to JSON or YAML\.

======== ===========
Home     https://github.com/bow/crimson
Versions 0.3.0
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/crimson

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install crimson

and update with::

   conda update crimson

A Docker container is available at https://quay.io/repository/biocontainers/crimson.
