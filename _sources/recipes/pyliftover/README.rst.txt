pyliftover
==========

|downloads|

Pure\-python implementation of UCSC \`\`liftOver\`\` genome coordinate conversion.

============= ===========
Home          https://github.com/konstantint/pyliftover
Versions      0.3
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//pyliftover/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pyliftover

and update with::

   conda update pyliftover



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pyliftover.
