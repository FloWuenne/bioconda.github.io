bioconductor-mipp
=================

|downloads|

This package finds optimal sets of genes that seperate samples into two or more classes.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/MiPP.html
Versions 1.48.0, 1.50.0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mipp
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mipp

and update with::

   conda update bioconductor-mipp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mipp.
