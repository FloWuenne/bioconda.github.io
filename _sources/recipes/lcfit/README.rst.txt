lcfit
=====

|downloads|

Likelihood curve fitting by nonlinear least squares\.

============= ===========
Home          https://github.com/matsengrp/lcfit
Versions      0.4
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/lcfit



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install lcfit

and update with::

   conda update lcfit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/lcfit.
