r-fnn
=====

|downloads|

Cover-tree and kd-tree fast k-nearest neighbor search algorithms and related applications including KNN classification, regression and information measures are implemented.

======== ===========
Home     https://CRAN.R-project.org/package=FNN
Versions 1.1
License  GPL (>= 2.1)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-fnn
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-fnn

and update with::

   conda update r-fnn



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-fnn.
