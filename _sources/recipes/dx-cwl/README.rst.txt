Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dx-cwl

and update with::

   conda update dx-cwl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/dx-cwl.
