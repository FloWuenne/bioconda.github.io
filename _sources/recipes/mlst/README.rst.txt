mlst
====

|downloads|

Scan contig files against PubMLST typing schemes

============= ===========
Home          https://github.com/tseemann/mlst
Versions      2.10, 2.11, 2.12, 2.6, 2.9
License       GPL2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/mlst



Links         biotools: :biotools:`mlst`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mlst

and update with::

   conda update mlst



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mlst.
