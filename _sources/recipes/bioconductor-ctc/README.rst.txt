bioconductor-ctc
================

|downloads|

Tools for export and import classification trees and clusters to other programs

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/ctc.html
Versions      1.52.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ctc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ctc

and update with::

   conda update bioconductor-ctc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ctc.
