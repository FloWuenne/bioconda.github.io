bioconductor-xmapbridge
=======================

|downloads|

xmapBridge can plot graphs in the X\:Map genome browser. This package exports plotting files in a suitable format.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/xmapbridge.html
Versions      1.38.0, 1.36.0, 1.34.0
License       LGPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-xmapbridge



Links         biotools: :biotools:`xmapbridge`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-xmapbridge

and update with::

   conda update bioconductor-xmapbridge



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-xmapbridge.
