bioconductor-cycle
==================

|downloads|

Package for assessing the statistical significance of periodic expression based on Fourier analysis and comparison with data generated by different background models

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/cycle.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-cycle/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cycle

and update with::

   conda update bioconductor-cycle

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cycle.
