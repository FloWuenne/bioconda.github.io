bioconductor-msnbase
====================

|downloads|

MSnbase provides infrastructure for manipulation\, processing and visualisation of mass spectrometry and proteomics data\, ranging from raw to quantitative and annotated data.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/MSnbase.html
Versions      2.2.0, 2.4.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-msnbase



Links         biotools: :biotools:`msnbase`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-msnbase

and update with::

   conda update bioconductor-msnbase



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-msnbase.
