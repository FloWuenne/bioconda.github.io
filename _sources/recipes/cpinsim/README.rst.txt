.. _`cpinsim`:

cpinsim
=======

|downloads|

CPINSim is a package for the simulation of protein complex assembly with constrained
protein interaction networks\.


======== ===========
Home     https://github.com/BiancaStoecker/cpinsim
Versions 0.5.2
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/cpinsim

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cpinsim

and update with::

   conda update cpinsim



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cpinsim.
