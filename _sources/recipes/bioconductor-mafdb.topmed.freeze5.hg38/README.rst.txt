Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mafdb.topmed.freeze5.hg38

and update with::

   conda update bioconductor-mafdb.topmed.freeze5.hg38
