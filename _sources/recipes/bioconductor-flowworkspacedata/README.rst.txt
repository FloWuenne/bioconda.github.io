bioconductor-flowworkspacedata
==============================

|downloads|

The necessary external data to run the flowWorkspace and openCyto vignette is found in this package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/flowWorkspaceData.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowworkspacedata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowworkspacedata

and update with::

   conda update bioconductor-flowworkspacedata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowworkspacedata.
