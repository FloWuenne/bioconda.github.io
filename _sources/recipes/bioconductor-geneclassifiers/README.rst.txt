bioconductor-geneclassifiers
============================

|downloads|

This packages aims for easy accessible application of classifiers which have been published in literature using an ExpressionSet as input.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/geneClassifiers.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-geneclassifiers/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-geneclassifiers

and update with::

   conda update bioconductor-geneclassifiers



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-geneclassifiers.
