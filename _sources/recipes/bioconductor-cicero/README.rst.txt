bioconductor-cicero
===================

|downloads|

Cicero computes putative cis\-regulatory maps from single\-cell chromatin accessibility data. It also extends monocle 2 for use in chromatin accessibility data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/cicero.html
Versions      
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-cicero/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cicero

and update with::

   conda update bioconductor-cicero



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cicero.
