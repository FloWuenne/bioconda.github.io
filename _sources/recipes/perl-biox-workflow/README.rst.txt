Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-biox-workflow

and update with::

   conda update perl-biox-workflow



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-biox-workflow.
