bioconductor-chipqc
===================

|downloads|

Quality metrics for ChIPseq data\.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/ChIPQC.html
Versions      1.10.3, 1.12.3, 1.14.0
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-chipqc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-chipqc

and update with::

   conda update bioconductor-chipqc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-chipqc.
