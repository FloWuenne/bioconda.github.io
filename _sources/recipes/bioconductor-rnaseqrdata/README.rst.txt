bioconductor-rnaseqrdata
========================

|downloads|

RNASeqRData is a helper experiment package for vignette demonstration purpose in RNASeqR software package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/RNASeqRData.html
Versions      1.0.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-rnaseqrdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rnaseqrdata

and update with::

   conda update bioconductor-rnaseqrdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rnaseqrdata.
