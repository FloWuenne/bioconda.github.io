Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cutadapt

and update with::

   conda update cutadapt



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cutadapt.
