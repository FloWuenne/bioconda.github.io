pmdtools

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pmdtools

and update with::

   conda update pmdtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pmdtools.
