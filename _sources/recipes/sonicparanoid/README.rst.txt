Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sonicparanoid

and update with::

   conda update sonicparanoid



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sonicparanoid.
