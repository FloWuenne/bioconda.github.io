pybedtools
==========

|downloads|

Wraps BEDTools for use in Python and adds many additional features.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pybedtools

and update with::

   conda update pybedtools
