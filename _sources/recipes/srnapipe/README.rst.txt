Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install srnapipe

and update with::

   conda update srnapipe

A Docker container is available at https://quay.io/repository/biocontainers/srnapipe.
