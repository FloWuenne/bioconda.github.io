alder
=====

|downloads|

The ALDER software computes the weighted linkage disequilibrium \(LD\) statistic for making inference about population admixture

============= ===========
Home          http://cb.csail.mit.edu/cb/alder/
Versions      1.03
License       Custom OSS
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/alder/meta.yaml



Links         biotools: :biotools:`alder`, doi: :doi:`10.1534/genetics.112.147330`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install alder

and update with::

   conda update alder



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/alder.
