perl-image-exiftool
===================

|downloads|

ExifTool is a platform\-independent Perl library plus a command\-line application for reading\, writing and editing meta information in a wide variety of files.

============= ===========
Home          http://metacpan.org/pod/Image::ExifTool
Versions      10.40, 11.01, 11.11
License       perl_5
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-image-exiftool



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-image-exiftool

and update with::

   conda update perl-image-exiftool



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-image-exiftool.
