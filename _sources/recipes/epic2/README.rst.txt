epic2
=====

Ultraperformant Chip\-Seq broad domain finder based on SICER.

============= ===========

Versions      0.0.16, 0.0.15
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//epic2/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install epic2

and update with::

   conda update epic2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/epic2.
