ncrf
====

|downloads|

Noise\-Cancelling Repeat Finder\, Uncovering tandem repeats in error\-prone long\-read sequencing data.

============= ===========
Home          https://github.com/makovalab-psu/NoiseCancellingRepeatFinder
Versions      1.00.06, 1.0.4
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//ncrf/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ncrf

and update with::

   conda update ncrf



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ncrf.
