r-genometricorr
===============

|downloads|

Genometric Correlation \(GenometriCorr\) is an R package for spatial correlation of genome\-wide interval datasets.

============= ===========
Home          http://genometricorr.sourceforge.net
Versions      1.1.17
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-genometricorr



Links         doi: :doi:`10.1371/journal.pcbi.1002529`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-genometricorr

and update with::

   conda update r-genometricorr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-genometricorr.
