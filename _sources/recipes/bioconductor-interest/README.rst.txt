bioconductor-interest
=====================

|downloads|

This package performs Intron\-Exon Retention analysis on RNA\-seq data \(.bam files\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/IntEREst.html
Versions      1.4.1, 1.2.2
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-interest



Links         doi: :doi:`10.18129/B9.bioc.IntEREst`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-interest

and update with::

   conda update bioconductor-interest



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-interest.
