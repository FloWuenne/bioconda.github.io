r-mmgenome
==========

|downloads|

Tools for extracting individual genomes from metagenomes

============= ===========
Home          https://github.com/MadsAlbertsen/mmgenome
Versions      0.7.1, 0.6.3
License       AGPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//r-mmgenome/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-mmgenome

and update with::

   conda update r-mmgenome



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-mmgenome.
