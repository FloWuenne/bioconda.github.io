bioconductor-illumina450probevariants.db
========================================

|downloads|

Includes details on variants for each probe on the 450k bead chip for each of the four populations \(Asian\, American\, African and European\)

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/Illumina450ProbeVariants.db.html
Versions      1.18.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-illumina450probevariants.db/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-illumina450probevariants.db

and update with::

   conda update bioconductor-illumina450probevariants.db



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-illumina450probevariants.db.
