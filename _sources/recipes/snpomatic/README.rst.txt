snpomatic
=========

|downloads|

SNP-o-matic is a fast, stringent short-read mapping software.

======== ===========
Home     https://github.com/magnusmanske/snpomatic
Versions 1.0
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/snpomatic
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install snpomatic

and update with::

   conda update snpomatic



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/snpomatic.
