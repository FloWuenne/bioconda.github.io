.. _`bioconductor-annotationtools`:

bioconductor-annotationtools
============================

|downloads|

Functions to annotate microarrays, find orthologs, and integrate heterogeneous gene expression profiles using annotation and other molecular biology information available as flat file database (plain text files).

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/annotationTools.html
Versions 1.50.0, 1.52.0
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-annotationtools
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-annotationtools

and update with::

   conda update bioconductor-annotationtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-annotationtools.
