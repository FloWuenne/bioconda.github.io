r-ncbit
=======

|downloads|

making NCBI taxonomic data locally available and searchable as an R object

============= ===========
Home          https://CRAN.R-project.org/package=ncbit
Versions      2013.03.29
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-ncbit



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-ncbit

and update with::

   conda update r-ncbit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-ncbit.
