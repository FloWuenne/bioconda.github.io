bioconductor-probamr
====================

|downloads|

Mapping PSMs back to genome. The package builds SAM file from shotgun proteomics data The package also provides function to prepare annotation from GTF file.

======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/proBAMr.html
Versions 1.10.0, 1.8.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-probamr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-probamr

and update with::

   conda update bioconductor-probamr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-probamr.
