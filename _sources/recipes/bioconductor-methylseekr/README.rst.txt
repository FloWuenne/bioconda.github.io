bioconductor-methylseekr
========================

|downloads|

This is a package for the discovery of regulatory regions from Bis\-seq data

============= ===========

Versions      1.20.0, 1.18.0, 1.16.0
License       GPL (>=2)

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-methylseekr

and update with::

   conda update bioconductor-methylseekr

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-methylseekr.
