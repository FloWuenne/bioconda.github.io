bioconductor-flowqb
===================

|downloads|

flowQB is a fully automated R Bioconductor package to calculate automatically the detector efficiency \(Q\)\, optical background \(B\) and intrinsic CV of the beads.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/flowQB.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowqb



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowqb

and update with::

   conda update bioconductor-flowqb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowqb.
