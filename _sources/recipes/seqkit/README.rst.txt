Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install seqkit

and update with::

   conda update seqkit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/seqkit.
