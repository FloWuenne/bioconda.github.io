bioconductor-basespacer
=======================

|downloads|

A rich R interface to Illumina's BaseSpace cloud computing environment, enabling the fast development of data analysis and visualisation tools.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/BaseSpaceR.html
Versions 1.20.0
License  Apache License 2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-basespacer
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-basespacer

and update with::

   conda update bioconductor-basespacer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-basespacer.
