trawler
=======

|downloads|

Trawler is a motif discovery tool used to identify enriched motifs in a set of sequenced regions of DNA.

============= ===========
Home          https://trawler.erc.monash.edu.au/help.html
Versions      2.0
License       GPLv2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//trawler/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install trawler

and update with::

   conda update trawler



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/trawler.
