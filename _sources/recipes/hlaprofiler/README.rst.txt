hlaprofiler
===========

|downloads|

HLAProfiler uses k\-mer profiles to predict HLA types from paired\-end RNA\-seq data.

============= ===========
Home          https://github.com/ExpressionAnalysis/HLAProfiler
Versions      1.0.5, 1.0.4, 1.0.3, 1.0.2, 1.0.1
License       Custom non-commercial license
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/hlaprofiler/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hlaprofiler

and update with::

   conda update hlaprofiler



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hlaprofiler.
