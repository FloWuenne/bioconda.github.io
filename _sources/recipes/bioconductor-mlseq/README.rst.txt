.. _`bioconductor-mlseq`:

bioconductor-mlseq
==================

|downloads|

This package applies several machine learning methods\, including SVM\, bagSVM\, Random Forest and CART\, to RNA\-Seq data.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/MLSeq.html
Versions      1.18.0
License       GPL(>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mlseq



Links         biotools: :biotools:`mlseq`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mlseq

and update with::

   conda update bioconductor-mlseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mlseq.
