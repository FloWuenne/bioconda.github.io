bcalm
=====

|downloads|

BCALM 2 is a bioinformatics tool for constructing the compacted de Bruijn graph from sequencing data.

============= ===========
Home          https://github.com/GATB/bcalm
Versions      2.2.1, 2.2.0
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bcalm/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bcalm

and update with::

   conda update bcalm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bcalm.
