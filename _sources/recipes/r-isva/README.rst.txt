r-isva
======

|downloads|

Independent Surrogate Variable Analysis is an algorithm for feature selection in the presence of potential confounding factors \(see Teschendorff AE et al 2011\, \<doi\: 10.1093\/bioinformatics\/btr171\>\).

============= ===========
Home          https://CRAN.R-project.org/package=isva
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-isva



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-isva

and update with::

   conda update r-isva



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-isva.
