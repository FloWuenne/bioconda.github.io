fsnviz
======

|downloads|

Tool for plotting gene fusion events detected by various tools using Circos.

============= ===========
Home          https://github.com/bow/fsnviz
Versions      0.3.0, 0.2.0, 0.1.0
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//fsnviz/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fsnviz

and update with::

   conda update fsnviz



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/fsnviz.
