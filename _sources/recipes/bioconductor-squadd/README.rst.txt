.. _`bioconductor-squadd`:

bioconductor-squadd
===================

|downloads|

This package SQUADD is a SQUAD add-on. It permits to generate SQUAD simulation matrix, prediction Heat-Map and Correlation Circle from PCA analysis.

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/SQUADD.html
Versions 1.26.0, 1.28.0
License  GPL (>=2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-squadd
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-squadd

and update with::

   conda update bioconductor-squadd



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-squadd.
