bioconductor-methylpipe
=======================

|downloads|

Memory efficient analysis of base resolution DNA methylation data in both the CpG and non\-CpG sequence context. Integration of DNA methylation data derived from any methodology providing base\- or low\-resolution data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/methylPipe.html
Versions      1.14.0, 1.12.0, 1.10.0
License       GPL(>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-methylpipe/meta.yaml



Links         biotools: :biotools:`methylpipe`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-methylpipe

and update with::

   conda update bioconductor-methylpipe

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-methylpipe.
