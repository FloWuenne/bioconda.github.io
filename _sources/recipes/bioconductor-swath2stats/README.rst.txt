.. _`bioconductor-swath2stats`:

bioconductor-swath2stats
========================

|downloads|

This package is intended to transform SWATH data from the OpenSWATH software into a format readable by other statistics packages while performing filtering, annotation and FDR estimation.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/SWATH2stats.html
Versions 1.6.1
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-swath2stats
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-swath2stats

and update with::

   conda update bioconductor-swath2stats



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-swath2stats.
