Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hicexplorer

and update with::

   conda update hicexplorer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hicexplorer.
