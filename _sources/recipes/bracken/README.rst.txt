.. _`bracken`:

bracken
=======

|downloads|

Bracken \(Bayesian Reestimation of Abundance with KrakEN\) is a highly accurate statistical method that computes the abundance of species in DNA sequences from a metagenomics sample.

============= ===========
Home          https://github.com/jenniferlu717/Bracken
Versions      1.0.0
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bracken



Links         biotools: :biotools:`Bracken`, doi: :doi:`10.7717/peerj-cs.104`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bracken

and update with::

   conda update bracken



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bracken.
