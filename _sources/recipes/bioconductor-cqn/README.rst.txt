bioconductor-cqn
================

|downloads|

A normalization tool for RNA\-Seq data\, implementing the conditional quantile normalization method.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/cqn.html
Versions      1.28.0, 1.26.0, 1.24.0, 1.22.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-cqn/meta.yaml

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cqn

and update with::

   conda update bioconductor-cqn



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cqn.
