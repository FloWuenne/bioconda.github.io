bioconductor-nucler
===================

|downloads|

Nucleosome positioning for Tiling Arrays and NGS experiments.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/nucleR.html
Versions 2.8.0
License  LGPL (>= 3)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-nucler
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-nucler

and update with::

   conda update bioconductor-nucler



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-nucler.
