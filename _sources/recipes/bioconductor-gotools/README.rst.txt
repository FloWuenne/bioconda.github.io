bioconductor-gotools
====================

|downloads|

Wraper functions for description/comparison of oligo ID list using Gene Ontology database

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/goTools.html
Versions 1.52.0
License  GPL-2
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-gotools
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-gotools

and update with::

   conda update bioconductor-gotools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-gotools.
