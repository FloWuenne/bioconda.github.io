bioconductor-compartmap
=======================

|downloads|

Compartmap performs shrunken A\/B compartment inference from ATAC\-seq and methylation arrays.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/compartmap.html
Versions      1.0.2
License       GPL-3 + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-compartmap/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-compartmap

and update with::

   conda update bioconductor-compartmap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-compartmap.
