Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bwa

and update with::

   conda update bwa



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bwa.
