wes-service-client
==================

|downloads|

Implementation of the GA4GH Workflow Execution Service\, a REST service for running workflows\; client support only

============= ===========
Home          https://github.com/common-workflow-language/workflow-service
Versions      2.7, 2.5
License       Apache License 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/wes-service-client



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install wes-service-client

and update with::

   conda update wes-service-client



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/wes-service-client.
