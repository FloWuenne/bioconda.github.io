graftm
======

|downloads|

GraftM is a pipeline used for identifying and classifying marker gene reads from metagenomic datasets

============= ===========
Home          http://geronimp.github.io/graftM
Versions      0.11.1, 0.10.1
License       GPL3+
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/graftm/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install graftm

and update with::

   conda update graftm
