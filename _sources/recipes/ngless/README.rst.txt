License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ngless



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ngless

and update with::

   conda update ngless



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ngless.
