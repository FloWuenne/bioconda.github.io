cromshell
=========

|downloads|

Command\-line interface to the Cromwell workflow manager

============= ===========
Home          https://github.com/broadinstitute/cromshell
Versions      0.2.3
License       BSD-3-Clause
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/cromshell



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cromshell

and update with::

   conda update cromshell
