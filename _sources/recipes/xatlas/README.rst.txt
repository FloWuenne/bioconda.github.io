.. _`xatlas`:

xatlas
======

|downloads|

xAtlas is a fast and retrainable small variant caller that has been developed at the Baylor College of Medicine Human Genome Sequencing Center.

============= ===========
Home          https://github.com/jfarek/xatlas
Versions      0.2.1, 0.1
License       BSD-3-Clause
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/xatlas



Links         doi: :doi:`10.1101/295071`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install xatlas

and update with::

   conda update xatlas



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/xatlas.
