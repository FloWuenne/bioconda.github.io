bioconductor-omiccircos
=======================

|downloads|

OmicCircos is an R application and package for generating high\-quality circular plots for omics data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/OmicCircos.html
Versions      1.20.0, 1.18.0, 1.16.0, 1.14.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-omiccircos/meta.yaml



Links         biotools: :biotools:`omiccircos`, doi: :doi:`10.4137/cin.s13495`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-omiccircos

and update with::

   conda update bioconductor-omiccircos



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-omiccircos.
