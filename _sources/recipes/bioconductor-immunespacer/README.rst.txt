.. _`bioconductor-immunespacer`:

bioconductor-immunespacer
=========================

|downloads|

Provides a convenient API for accessing data sets within ImmuneSpace \(www.immunespace.org\)\, the data repository and analysis platform of the Human Immunology Project Consortium \(HIPC\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/ImmuneSpaceR.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-immunespacer



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-immunespacer

and update with::

   conda update bioconductor-immunespacer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-immunespacer.
