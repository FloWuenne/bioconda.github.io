bioconductor-rqc
================

|downloads|

Rqc is an optimised tool designed for quality control and assessment of high\-throughput sequencing data. It performs parallel processing of entire files and produces a report which contains a set of high\-resolution graphics.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/Rqc.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-rqc/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rqc

and update with::

   conda update bioconductor-rqc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rqc.
