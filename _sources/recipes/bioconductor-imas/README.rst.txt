Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-imas

and update with::

   conda update bioconductor-imas



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-imas.
