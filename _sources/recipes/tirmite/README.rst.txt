tirmite
=======

|downloads|

Map TIR\-pHMM models to genomic sequences for annotation of MITES and complete DNA\-Transposons\.

============= ===========
Home          https://github.com/Adamtaranto/TIRmite
Versions      1.1.0
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/tirmite



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install tirmite

and update with::

   conda update tirmite



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/tirmite.
