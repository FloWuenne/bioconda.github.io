prestor
=======

|downloads|

A prototype package for generating quality control plots from pRESTO output.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install prestor

and update with::

   conda update prestor
