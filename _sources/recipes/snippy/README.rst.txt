Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install snippy

and update with::

   conda update snippy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/snippy.
