bioconductor-icnv
=================

|downloads|

Integrative copy number variation \(CNV\) detection from multiple platform and experimental design.

Versions

Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-icnv/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-icnv

and update with::

   conda update bioconductor-icnv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-icnv.
