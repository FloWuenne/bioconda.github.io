Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/cap3

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cap3

and update with::

   conda update cap3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cap3.
