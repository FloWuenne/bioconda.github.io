bioconductor-mirnatap.db
========================

|downloads|

This package holds the database for miRNAtap\.

============= ===========
Home          http://bioconductor.org/packages/3.6/data/annotation/html/miRNAtap.db.html
Versions      0.99.10
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mirnatap.db



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mirnatap.db

and update with::

   conda update bioconductor-mirnatap.db



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mirnatap.db.
