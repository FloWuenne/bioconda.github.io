cmappy
======

|downloads|

Assorted tools for interacting with .gct\, .gctx\, .grp\, and .gmt files as well as other Connectivity Map \(Broad Institute\) data\/tools

============= ===========
Home          https://github.com/cmap/cmapPy
Versions      1.0.9, 1.1.1, 2.0.1, 2.1.0, 2.2.0, 2.2.1, 2.2.2, 3.0.0, 3.1.1, 3.2.0, 3.3.0
License       BSD 3-clause
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/cmappy



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cmappy

and update with::

   conda update cmappy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cmappy.
