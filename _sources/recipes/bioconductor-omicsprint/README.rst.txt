bioconductor-omicsprint
=======================

|downloads|

omicsPrint provides functionality for cross omic genetic fingerprinting\, for example\, to verify sample relationships between multiple omics data types\, i.e. genomic\, transcriptomic and epigenetic \(DNA methylation\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/omicsPrint.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-omicsprint/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-omicsprint

and update with::

   conda update bioconductor-omicsprint



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-omicsprint.
