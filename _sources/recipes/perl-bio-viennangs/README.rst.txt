Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-bio-viennangs

and update with::

   conda update perl-bio-viennangs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-bio-viennangs.
