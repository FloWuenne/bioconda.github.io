bioconductor-qdnaseq.mm10
=========================

|downloads|

This package provides QDNAseq bin annotations for the mouse genome build mm10.

======== ===========
Home     http://bioconductor.org/packages/3.6/data/experiment/html/QDNAseq.mm10.html
Versions 1.4.0, 1.6.0, 1.8.0
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-qdnaseq.mm10
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-qdnaseq.mm10

and update with::

   conda update bioconductor-qdnaseq.mm10



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-qdnaseq.mm10.
