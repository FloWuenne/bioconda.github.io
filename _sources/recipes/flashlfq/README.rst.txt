Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install flashlfq

and update with::

   conda update flashlfq

A Docker container is available at https://quay.io/repository/biocontainers/flashlfq.
