r-adephylo
==========

|downloads|

Multivariate tools to analyze comparative data\, i.e. a phylogeny and some traits measured for each taxa.

============= ===========
Home          https://CRAN.R-project.org/package=adephylo
Versions      1.1_11
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/r-adephylo/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-adephylo

and update with::

   conda update r-adephylo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-adephylo.
