bioconductor-bsseqdata
======================

|downloads|

Example whole genome bisulfite data for the bsseq package

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/bsseqData.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-bsseqdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsseqdata

and update with::

   conda update bioconductor-bsseqdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsseqdata.
