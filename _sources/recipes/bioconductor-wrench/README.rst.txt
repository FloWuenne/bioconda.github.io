bioconductor-wrench
===================

|downloads|

Wrench is a package for normalization sparse genomic count data\, like that arising from 16s metagenomic surveys.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/Wrench.html
Versions      1.0.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-wrench/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-wrench

and update with::

   conda update bioconductor-wrench



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-wrench.
