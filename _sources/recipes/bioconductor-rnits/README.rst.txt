============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rnits

and update with::

   conda update bioconductor-rnits



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rnits.
