bioconductor-junctionseq
========================

|downloads|

A Utility for Detection and Visualization of Differential Exon or Splice\-Junction Usage in RNA\-Seq data.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/JunctionSeq.html
Versions      1.8.0
License       file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-junctionseq



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-junctionseq

and update with::

   conda update bioconductor-junctionseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-junctionseq.
