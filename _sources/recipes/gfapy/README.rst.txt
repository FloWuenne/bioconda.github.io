Library for handling data in the GFA1 and GFA2 formats

======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gfapy

and update with::

   conda update gfapy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gfapy.
