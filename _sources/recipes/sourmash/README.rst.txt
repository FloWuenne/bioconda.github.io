Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sourmash

and update with::

   conda update sourmash



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sourmash.
