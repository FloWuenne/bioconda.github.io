sourmash
========

Compute and compare MinHash signatures for DNA data sets.

Versions 1.0, 2.0.0a1, 2.0.0a2
License  BSD License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/sourmash
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sourmash

and update with::

   conda update sourmash



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sourmash.
