Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install samtools

and update with::

   conda update samtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/samtools.
