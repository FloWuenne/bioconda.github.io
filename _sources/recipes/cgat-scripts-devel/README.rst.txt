Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install cgat-scripts-devel

and update with::

   conda update cgat-scripts-devel



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/cgat-scripts-devel.
