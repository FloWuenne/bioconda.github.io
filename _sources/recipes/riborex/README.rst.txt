riborex
=======

|downloads|

Riborex is a R package for identification of differential translation from Ribo\-seq data.

============= ===========
Home          https://github.com/smithlabcode/riborex
Versions      2.3.4
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/riborex/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install riborex

and update with::

   conda update riborex
