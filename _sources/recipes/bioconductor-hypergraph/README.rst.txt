bioconductor-hypergraph
=======================

|downloads|

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-hypergraph

and update with::

   conda update bioconductor-hypergraph
