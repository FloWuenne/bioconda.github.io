Differential expression analysis of RNA\-seq using the Poisson\-Tweedie family of distributions.

Versions      1.26.0, 1.24.0, 1.22.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-tweedeseq

and update with::

   conda update bioconductor-tweedeseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-tweedeseq.
