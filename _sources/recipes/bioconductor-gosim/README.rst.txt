bioconductor-gosim
==================

|downloads|

This package implements several functions useful for computing similarities between GO terms and gene products based on their GO annotation. Moreover it allows for computing a GO enrichment analysis

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/GOSim.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-gosim/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-gosim

and update with::

   conda update bioconductor-gosim



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-gosim.
