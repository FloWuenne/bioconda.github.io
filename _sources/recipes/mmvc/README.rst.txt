mmvc
====

|downloads|

Call variants based on a Bayesian multinomial mixture model.

============= ===========
Home          https://github.com/veg/mmvc
Versions      1.0.2
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/mmvc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mmvc

and update with::

   conda update mmvc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mmvc.
