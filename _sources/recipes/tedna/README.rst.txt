Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install tedna

and update with::

   conda update tedna



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/tedna.
