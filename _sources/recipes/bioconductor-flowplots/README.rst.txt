.. _`bioconductor-flowplots`:

bioconductor-flowplots
======================

|downloads|

Graphical displays with embedded statistical tests for gated ICS flow cytometry data\, and a data class which stores \"stacked\" data and has methods for computing summary measures on stacked data\, such as marginal and polyfunctional degree data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/flowPlots.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowplots



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowplots

and update with::

   conda update bioconductor-flowplots



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowplots.
