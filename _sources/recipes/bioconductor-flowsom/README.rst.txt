bioconductor-flowsom
====================

|downloads|

FlowSOM offers visualization options for cytometry data\, by using Self\-Organizing Map clustering and Minimal Spanning Trees.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/FlowSOM.html
Versions      1.12.0, 1.10.0, 1.8.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowsom



Links         biotools: :biotools:`flowsom`, doi: :doi:`10.1002/cyto.a.22625`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowsom

and update with::

   conda update bioconductor-flowsom



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowsom.
