domclust
========

|downloads|

Effective tool for orthologous grouping in multiple genomes

============= ===========
Home          http://mbgd.genome.ad.jp/domclust/
Versions      1.2.8a, 1.0
License       GPL3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/domclust



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install domclust

and update with::

   conda update domclust



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/domclust.
