bioconductor-cctutorial
=======================

|downloads|

This is a data package that accompanies a ChIP\-chip tutorial\, which has been published in PLoS Computational Biology. The data and source code in this package allow the reader to completely reproduce the steps in the tutorial.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/ccTutorial.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-cctutorial



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cctutorial

and update with::

   conda update bioconductor-cctutorial



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cctutorial.
