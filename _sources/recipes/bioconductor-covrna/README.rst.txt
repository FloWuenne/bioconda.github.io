bioconductor-covrna
===================

|downloads|

This package provides the analysis methods fourthcorner and RLQ analysis for large-scale transcriptomic data.

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/covRNA.html
Versions 1.4.0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-covrna
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-covrna

and update with::

   conda update bioconductor-covrna



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-covrna.
