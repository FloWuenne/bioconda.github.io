Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gprofiler-official

and update with::

   conda update gprofiler-official



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gprofiler-official.
