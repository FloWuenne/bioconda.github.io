.. _`sshmm`:

sshmm
=====

|downloads|

ssHMM is an RNA motif finder that recovers sequence\-structure motifs from RNA\-binding protein data\, such as CLIP\-Seq data.

============= ===========
Home          https://github.molgen.mpg.de/heller/ssHMM
Versions      1.0.7
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//sshmm/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sshmm

and update with::

   conda update sshmm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sshmm.
