afplot
======

|downloads|

Plot allele frequencies in VCF files

======== ===========
Home     https://github.com/sndrtj/afplot
Versions 0.2.1
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/afplot

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install afplot

and update with::

   conda update afplot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/afplot.
