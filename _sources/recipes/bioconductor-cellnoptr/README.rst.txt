bioconductor-cellnoptr
======================

|downloads|

This package does optimisation of boolean logic networks of signalling pathways based on a previous knowledge network and a set of data upon perturbation of the nodes in the network.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/CellNOptR.html
Versions 1.22.0
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-cellnoptr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cellnoptr

and update with::

   conda update bioconductor-cellnoptr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cellnoptr.
