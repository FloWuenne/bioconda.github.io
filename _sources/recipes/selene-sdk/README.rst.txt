selene-sdk
==========

Framework for developing sequence\-level deep learning networks.

============= ===========
Home          https://github.com/FunctionLab/selene
Versions      0.2.0, 0.1.3, 0.1.2, 0.0.1
License       BSD 3-clause clear
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/selene-sdk



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install selene-sdk

and update with::

   conda update selene-sdk



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/selene-sdk.
