.. _`irida-linker`:

irida-linker
============

|downloads|

The NGS Archive Linker is a Perl script used to generate a structure of links for files stored in the IRIDA platform.

============= ===========
Home          https://github.com/phac-nml/irida-linker
Versions      1.0.2, 1.0.1
License       Apache-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/irida-linker



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install irida-linker

and update with::

   conda update irida-linker



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/irida-linker.
