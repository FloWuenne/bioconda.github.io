Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-canine2.db

and update with::

   conda update bioconductor-canine2.db

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-canine2.db.
