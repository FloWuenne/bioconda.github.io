bioconductor-multiscan
======================

|downloads|

Estimates gene expressions from several laser scans of the same microarray

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/multiscan.html
Versions 1.36.0, 1.38.0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-multiscan

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-multiscan

and update with::

   conda update bioconductor-multiscan



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-multiscan.
