mpa-portable
============

|downloads|

MPA Portable is a light\-weight and stand\-alone software for the
identification of proteins and in\-depth analysis of metaproteomics.


============= ===========
Home          https://github.com/compomics/meta-proteome-analyzer
Versions      1.4.1
License       Apache License, Version 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/mpa-portable/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mpa-portable

and update with::

   conda update mpa-portable


Notes
-----
mpa\-protable is Java program that comes with a custom wrapper shell script.
