alignlib-lite
=============

|downloads|

Simple wrapper around alignlib C\+\+ library for sequence alignment

============= ===========
Home          http://sourceforge.net/projects/alignlib/
Versions      0.3, 0.2.3
License       GPL-2.0-or-later
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/alignlib-lite/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install alignlib-lite

and update with::

   conda update alignlib-lite



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/alignlib-lite.
