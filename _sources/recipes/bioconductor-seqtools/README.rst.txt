.. _`bioconductor-seqtools`:

bioconductor-seqtools
=====================

|downloads|

Analyze read length, phred scores and alphabet frequency and DNA k-mers on uncompressed and compressed fastq files.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/seqTools.html
Versions 1.10.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-seqtools
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-seqtools

and update with::

   conda update bioconductor-seqtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-seqtools.
