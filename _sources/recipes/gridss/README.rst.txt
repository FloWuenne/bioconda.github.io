Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gridss

and update with::

   conda update gridss



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gridss.
