Biosequence analysis using profile hidden Markov models

============= ===========

License       GPLv2

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hmmer2

and update with::

   conda update hmmer2
