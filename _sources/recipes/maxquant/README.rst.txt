maxquant
========

|downloads|

MaxQuant is a quantitative proteomics software package designed for analyzing large mass\-spectrometric data sets. License restricted.

============= ===========
Home          http://www.coxdocs.org/doku.php?id=maxquant:start
Versions      1.6.3.4, 1.6.2.10
License       http://www.coxdocs.org/lib/exe/fetch.php?media=license_agreement.pdf
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/maxquant/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install maxquant

and update with::

   conda update maxquant



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/maxquant.
