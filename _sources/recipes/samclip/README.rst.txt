samclip
=======

|downloads|

Filter SAM file for soft and hard clipped alignments

============= ===========
Home          https://github.com/tseemann/samclip
Versions      0.2
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//samclip/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install samclip

and update with::

   conda update samclip



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/samclip.
