assemblerflow
=============

|downloads|

A Nextflow pipeline assembler for genomics. Pick your modules. Assemble them. Run the pipeline.

============= ===========
Home          https://github.com/ODiogoSilva/assemblerflow
Versions      1.1.0.post3, 1.1.0.post1, 1.0.1

Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/assemblerflow

Documentation http://assemblerflow.readthedocs.io/en/latest/


Development   http://assemblerflow.readthedocs.io/en/latest/


============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install assemblerflow

and update with::

   conda update assemblerflow



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/assemblerflow.
