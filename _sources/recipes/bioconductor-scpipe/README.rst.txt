.. _`bioconductor-scpipe`:

bioconductor-scpipe
===================

|downloads|

to process single cell RNA\-seq data from fastq to gene counting matrix\. it can process data generated by CEL\-seq\, MARS\-seq\, Drop\-seq and SMART\-seq\.

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/scPipe.html
Versions 1.0.0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-scpipe

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-scpipe

and update with::

   conda update bioconductor-scpipe



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-scpipe.
