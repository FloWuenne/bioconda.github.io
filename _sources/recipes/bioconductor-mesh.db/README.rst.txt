bioconductor-mesh.db
====================

|downloads|

A set of annotation maps describing the entire MeSH assembled using data from MeSH.

============= ===========
Home          http://bioconductor.org/packages/3.7/data/annotation/html/MeSH.db.html
Versions      1.10.0, 1.9.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mesh.db



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mesh.db

and update with::

   conda update bioconductor-mesh.db



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mesh.db.
