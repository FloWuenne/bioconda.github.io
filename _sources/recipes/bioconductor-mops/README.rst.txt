Versions      1.10.0, 1.12.0
License       GPL-3

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mops

and update with::

   conda update bioconductor-mops



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mops.
