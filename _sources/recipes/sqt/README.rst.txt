Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sqt

and update with::

   conda update sqt



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sqt.
