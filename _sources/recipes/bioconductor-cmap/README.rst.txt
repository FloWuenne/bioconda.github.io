bioconductor-cmap
=================

|downloads|

Annotation data file for cMAP assembled using data from public data repositories

============= ===========
Home          http://bioconductor.org/packages/3.6/data/annotation/html/cMAP.html
Versions      1.15.1
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-cmap



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cmap

and update with::

   conda update bioconductor-cmap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cmap.
