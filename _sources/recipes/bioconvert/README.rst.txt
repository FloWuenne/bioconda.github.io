License  BSD License

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconvert

and update with::

   conda update bioconvert

A Docker container is available at https://quay.io/repository/biocontainers/bioconvert.
