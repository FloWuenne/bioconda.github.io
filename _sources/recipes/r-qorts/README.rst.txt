r-qorts
=======

|downloads|

The QoRTs software package is a fast\, efficient\, and portable multifunction toolkit designed to assist in the analysis\, quality control\, and data management of RNA\-Seq datasets.

============= ===========
Home          https://github.com/hartleys/QoRTs
Versions      1.3.0
License       Public Domain
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//r-qorts/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-qorts

and update with::

   conda update r-qorts


Notes
-----
This package is a requirement of the main QoRTs package\; \"qorts\".



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-qorts.
