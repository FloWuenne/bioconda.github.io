.. _`gembs`:

gembs
=====

|downloads|

gemBS is a bioinformatics pipeline designed for high throughput analysis of DNA methylation from Whole Genome Bisulfite Sequencing data \(WGBS\).

============= ===========
Home          https://github.com/heathsc/gemBS
Versions      3.2.0
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//gembs/meta.yaml



Links         doi: :doi:`10.1101/201988`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gembs

and update with::

   conda update gembs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gembs.
