.. _`masurca`:

masurca
=======

|downloads|

MaSuRCA \(Maryland Super\-Read Celera Assembler\) genome assembly software.
MaSuRCA requires Illumina data\, and supports third\-generation PacBio\/Nanopore
MinION reads for hybrid assembly.


============= ===========
Home          http://masurca.blogspot.co.uk/
Versions      3.2.7
License       GPL-3.0-only
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/masurca



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install masurca

and update with::

   conda update masurca



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/masurca.
