ldhelmet
========

|downloads|

Software program for statistical inference of fine\-scale crossover recombination rates from population genetic data.

============= ===========
Home          http://sourceforge.net/projects/ldhelmet/
Versions      1.10
License       GPL3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ldhelmet



Links         biotools: :biotools:`LDhelmet`, doi: :doi:`10.1371/journal.pgen.1003090`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ldhelmet

and update with::

   conda update ldhelmet



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ldhelmet.
