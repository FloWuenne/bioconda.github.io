Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install plasmidtron

and update with::

   conda update plasmidtron



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/plasmidtron.
