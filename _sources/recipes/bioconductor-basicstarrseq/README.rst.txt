bioconductor-basicstarrseq
==========================

|downloads|

Basic peak calling on STARR\-seq data based on a method introduced in \"Genome\-Wide Quantitative Enhancer Activity Maps Identified by STARR\-seq\" Arnold et al. Science. 2013 Mar 1\;339\(6123\)\:1074\-7. doi\: 10.1126\/science. 1232542. Epub 2013 Jan 17.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/BasicSTARRseq.html
Versions      1.10.0
License       LGPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-basicstarrseq/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-basicstarrseq

and update with::

   conda update bioconductor-basicstarrseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-basicstarrseq.
