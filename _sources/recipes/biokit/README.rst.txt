Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install biokit

and update with::

   conda update biokit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/biokit.
