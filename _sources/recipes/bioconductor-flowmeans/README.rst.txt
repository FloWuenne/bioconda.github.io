bioconductor-flowmeans
======================

|downloads|

Identifies cell populations in Flow Cytometry data using non\-parametric clustering and segmented\-regression\-based change point detection. Note\: R 2.11.0 or newer is required.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/flowMeans.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-flowmeans/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowmeans

and update with::

   conda update bioconductor-flowmeans



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowmeans.
