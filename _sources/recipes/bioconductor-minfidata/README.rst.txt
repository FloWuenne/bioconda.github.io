bioconductor-minfidata
======================

|downloads|

Data from 6 samples across 2 groups from 450k methylation arrays.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/minfiData.html
Versions      0.28.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-minfidata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-minfidata

and update with::

   conda update bioconductor-minfidata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-minfidata.
