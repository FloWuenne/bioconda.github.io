Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nasp

and update with::

   conda update nasp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nasp.
