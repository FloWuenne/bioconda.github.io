.. _`vcf-annotator`:

vcf-annotator
=============

|downloads|

Use the reference GenBank file to add biological annotations to the variant calls in a VCF.

============= ===========
Home          https://github.com/rpetit3/vcf-annotator
Versions      0.5
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/vcf-annotator



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vcf-annotator

and update with::

   conda update vcf-annotator



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vcf-annotator.
