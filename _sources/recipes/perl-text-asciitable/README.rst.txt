perl-text-asciitable
====================

|downloads|

Create a nice formatted table using ASCII characters\.

============= ===========
Home          http://metacpan.org/pod/Text::ASCIITable
Versions      0.22
License       perl_5
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-text-asciitable/0.22



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-text-asciitable

and update with::

   conda update perl-text-asciitable



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-text-asciitable.
