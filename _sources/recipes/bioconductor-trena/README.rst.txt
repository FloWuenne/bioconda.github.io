bioconductor-trena
==================

|downloads|

Methods for reconstructing transcriptional regulatory networks\, especially in species for which genome\-wide TF binding site information is available.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/trena.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-trena/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-trena

and update with::

   conda update bioconductor-trena



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-trena.
