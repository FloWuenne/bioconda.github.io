bioconductor-h5vc
=================

|downloads|

This package contains functions to interact with tally data from NGS experiments that is stored in HDF5 files.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/h5vc.html
Versions      
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-h5vc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-h5vc

and update with::

   conda update bioconductor-h5vc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-h5vc.
