.. _`primer3`:

primer3
=======

|downloads|

Design PCR primers from DNA sequence. From mispriming libraries to sequence quality data to the generation of internal oligos\, primer3 does it.

============= ===========
Home          https://github.com/primer3-org/primer3
Versions      2.4.1a, 2.4.0, 2.3.7, 2.0.0a, 1.1.4
License       GPLv2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/primer3



Links         biotools: :biotools:`primer3`, doi: :doi:`10.1093/nar/gks596`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install primer3

and update with::

   conda update primer3



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/primer3.
