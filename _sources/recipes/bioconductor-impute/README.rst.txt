Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-impute

and update with::

   conda update bioconductor-impute



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-impute.
