r-som
=====

|downloads|

Self-Organizing Map (with application in gene clustering).

======== ===========
Home     https://CRAN.R-project.org/package=som
Versions 0.3_5.1
License  GPL (>= 3)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-som
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-som

and update with::

   conda update r-som



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-som.
