.. _`sparcc`:

sparcc
======

|downloads|

SparCC is a python module for computing correlations in compositional data \(16S\, metagenomics\, etc\).

============= ===========
Home          https://bitbucket.org/yonatanf/sparcc
Versions      0.1.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/sparcc



Links         doi: :doi:`10.1371/journal.pcbi.1002687`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sparcc

and update with::

   conda update sparcc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sparcc.
