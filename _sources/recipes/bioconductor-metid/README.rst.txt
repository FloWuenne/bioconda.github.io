bioconductor-metid
==================

|downloads|

This package uses an innovative network\-based approach that will enhance our ability to determine the identities of significant ions detected by LC\-MS.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/MetID.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-metid



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-metid

and update with::

   conda update bioconductor-metid



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-metid.
