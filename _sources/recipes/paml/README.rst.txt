paml
====

|downloads|

A package of programs for phylogenetic analyses of DNA or protein sequences using maximum likelihood.

============= ===========
Home          http://abacus.gene.ucl.ac.uk/software/paml.html
Versions      4.9
License       GNU General Public License v3 (GPLv3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/paml/4.9



Links         biotools: :biotools:`paml`, doi: :doi:`10.1093/bioinformatics/13.5.555`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install paml

and update with::

   conda update paml



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/paml.
