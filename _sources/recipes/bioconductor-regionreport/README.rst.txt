.. _`bioconductor-regionreport`:

bioconductor-regionreport
=========================

|downloads|

Generate HTML or PDF reports to explore a set of regions such as the results from annotation\-agnostic expression analysis of RNA\-seq data at base\-pair resolution performed by derfinder. You can also create reports for DESeq2 or edgeR results.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/regionReport.html
Versions      1.16.1
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-regionreport/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-regionreport

and update with::

   conda update bioconductor-regionreport



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-regionreport.
