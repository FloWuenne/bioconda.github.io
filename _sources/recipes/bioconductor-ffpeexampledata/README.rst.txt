.. _`bioconductor-ffpeexampledata`:

bioconductor-ffpeexampledata
============================

|downloads|

A subset of GSE17565 \(April et al. 2009\) containing 32 FFPE samples of Burkitts Lymphoma and Breast Adenocarcinoma\, with a dilution series in technical duplicate.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/ffpeExampleData.html
Versions      
License       GPL (>2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ffpeexampledata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ffpeexampledata

and update with::

   conda update bioconductor-ffpeexampledata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ffpeexampledata.
