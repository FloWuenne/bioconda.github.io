metavelvet-sl-pipeline
======================

|downloads|

Perl libraries that run the full pipeline for metavelvet\-sl

Versions      1.0
License

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install metavelvet-sl-pipeline

and update with::

   conda update metavelvet-sl-pipeline



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/metavelvet-sl-pipeline.
