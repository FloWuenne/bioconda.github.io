enabrowsertools
===============

|downloads|

enaBrowserTools is a set of scripts that interface with the ENA web services to download data from ENA easily

============= ===========
Home          https://github.com/enasequence/enaBrowserTools
Versions      1.5.4
License       Apache-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/enabrowsertools/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install enabrowsertools

and update with::

   conda update enabrowsertools
