bioconductor-rexposome
======================

|downloads|

Package that allows to explore the exposome and to perform association analyses between exposures and health outcomes.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/rexposome.html
Versions      1.4.0
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-rexposome/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rexposome

and update with::

   conda update bioconductor-rexposome



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rexposome.
