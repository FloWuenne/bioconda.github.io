Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install itsxpress

and update with::

   conda update itsxpress

A Docker container is available at https://quay.io/repository/biocontainers/itsxpress.
