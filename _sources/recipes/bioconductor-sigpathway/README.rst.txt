.. _`bioconductor-sigpathway`:

bioconductor-sigpathway
=======================

|downloads|

Conducts pathway analysis by calculating the NT\_k and NE\_k statistics as described in Tian et al\. \(2005\)

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/sigPathway.html
Versions      1.44.1, 1.46.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-sigpathway



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-sigpathway

and update with::

   conda update bioconductor-sigpathway



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-sigpathway.
