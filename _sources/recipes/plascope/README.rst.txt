plascope
========

|downloads|

PlaScope is a targeted approach to assess the plasmidome of bacteria.

License       GPL (>=3)

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install plascope

and update with::

   conda update plascope
