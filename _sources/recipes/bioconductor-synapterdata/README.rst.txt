bioconductor-synapterdata
=========================

|downloads|

Data independant acquisition of UPS1 protein mix in an E. coli background obtained on a Waters Synapt G2 instrument.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/synapterdata.html
Versions      1.20.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-synapterdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-synapterdata

and update with::

   conda update bioconductor-synapterdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-synapterdata.
