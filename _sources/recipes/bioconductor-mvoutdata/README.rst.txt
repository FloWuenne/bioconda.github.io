bioconductor-mvoutdata
======================

|downloads|

affy and illumina raw data for assessing outlier detector performance

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/mvoutData.html
Versions      1.18.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-mvoutdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mvoutdata

and update with::

   conda update bioconductor-mvoutdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mvoutdata.
