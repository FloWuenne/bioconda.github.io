libmems
=======

|downloads|

libMems is a freely available software development library to support DNA string matching and comparative genomics.

============= ===========
Home          http://darlinglab.org/mauve/
Versions      1.6.0
License       GPL-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//libmems/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install libmems

and update with::

   conda update libmems



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/libmems.
