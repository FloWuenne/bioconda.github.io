bioconductor-ogsa
=================

|downloads|

OGSA provides a global estimate of pathway deregulation in cancer subtypes by integrating the estimates of significance for individual pathway members that have been identified by outlier analysis.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/OGSA.html
Versions      1.10.0, 1.8.0, 1.6.0
License       GPL (== 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ogsa



Links         biotools: :biotools:`ogsa`, doi: :doi:`10.1109/TCBB.2013.153`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ogsa

and update with::

   conda update bioconductor-ogsa



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ogsa.
