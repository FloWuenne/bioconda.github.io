Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install phylopandas

and update with::

   conda update phylopandas



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/phylopandas.
