.. _`bioconductor-ace`:

bioconductor-ace
================

|downloads|

Uses segmented copy number data to estimate tumor cell percentage and produce copy number plots displaying absolute copy numbers.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/ACE.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ace



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ace

and update with::

   conda update bioconductor-ace



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ace.
