Data needed by the affycomp package.

Versions

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-affycompdata

and update with::

   conda update bioconductor-affycompdata

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-affycompdata.
