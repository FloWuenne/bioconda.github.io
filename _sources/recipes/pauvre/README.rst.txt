Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pauvre

and update with::

   conda update pauvre



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pauvre.
