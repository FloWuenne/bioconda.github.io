bioconductor-chromstardata
==========================

|downloads|

ChIP\-seq data for demonstration purposes in the chromstaR package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/chromstaRData.html
Versions      1.6.0, 1.4.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-chromstardata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-chromstardata

and update with::

   conda update bioconductor-chromstardata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-chromstardata.
