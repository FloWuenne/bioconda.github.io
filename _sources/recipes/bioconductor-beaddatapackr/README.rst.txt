bioconductor-beaddatapackr
==========================

|downloads|

Provides functionality for the compression and decompression of raw bead\-level data from the Illumina BeadArray platform

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/BeadDataPackR.html
Versions      1.28.0, 1.30.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-beaddatapackr



Links         biotools: :biotools:`beaddatapackr`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-beaddatapackr

and update with::

   conda update bioconductor-beaddatapackr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-beaddatapackr.
