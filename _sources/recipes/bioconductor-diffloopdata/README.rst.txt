bioconductor-diffloopdata
=========================

|downloads|

ChIA\-PET example datasets and additional data for use with the diffloop package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/diffloopdata.html
Versions      
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-diffloopdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-diffloopdata

and update with::

   conda update bioconductor-diffloopdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-diffloopdata.
