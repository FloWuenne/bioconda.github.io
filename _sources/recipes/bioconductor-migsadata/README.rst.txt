bioconductor-migsadata
======================

|downloads|

MIGSA vignette data. The MIGSAdata package provides several data objects needed by MIGSA package to correctly generate its vignette\, and follow it step by step.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/MIGSAdata.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-migsadata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-migsadata

and update with::

   conda update bioconductor-migsadata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-migsadata.
