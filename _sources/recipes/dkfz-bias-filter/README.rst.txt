dkfz-bias-filter
================

|downloads|

The DKFZ bias filter flags SNVs that appear to be biased based on the variant read support

============= ===========
Home          https://github.com/eilslabs/DKFZBiasFilter
Versions      1.2.3a
License       GPLv3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//dkfz-bias-filter/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dkfz-bias-filter

and update with::

   conda update dkfz-bias-filter



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/dkfz-bias-filter.
