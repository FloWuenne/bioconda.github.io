.. _`bioconductor-mulder2012`:

bioconductor-mulder2012
=======================

|downloads|

This package provides functions to reproduce results and figures in Mulder K. et. al. published in Nature Cell Biology 2012 and Wang X. et. al. published in PLoS Computational Biology 2012.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/Mulder2012.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-mulder2012/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mulder2012

and update with::

   conda update bioconductor-mulder2012



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mulder2012.
