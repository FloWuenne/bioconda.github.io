bioconductor-rchyoptimyx
========================

|downloads|

Constructs a hierarchy of cells using flow cytometry for maximization of an external variable \(e.g.\, a clinical outcome or a cytokine response\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/RchyOptimyx.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rchyoptimyx



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rchyoptimyx

and update with::

   conda update bioconductor-rchyoptimyx

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rchyoptimyx.
