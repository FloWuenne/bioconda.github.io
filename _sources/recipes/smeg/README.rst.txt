smeg
====

|downloads|

Strain\-level MEtagenomic Growth estimation \(SMEG\) measures growth rates of microbial strains from complex metagenomic dataset at low coverage \(1X\)

============= ===========
Home          https://github.com/ohlab/SMEG
Versions      1.0.1
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/smeg/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install smeg

and update with::

   conda update smeg



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/smeg.
