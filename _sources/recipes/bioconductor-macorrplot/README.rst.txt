bioconductor-macorrplot
=======================

|downloads|

Graphically displays correlation in microarray data that is due to insufficient normalization

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/maCorrPlot.html
Versions      1.46.0, 1.48.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-macorrplot



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-macorrplot

and update with::

   conda update bioconductor-macorrplot



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-macorrplot.
