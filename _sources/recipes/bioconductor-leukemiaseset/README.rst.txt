.. _`bioconductor-leukemiaseset`:

bioconductor-leukemiaseset
==========================

|downloads|

Expressionset containing gene expresion data from 60 bone marrow samples of patients with one of the four main types of leukemia \(ALL\, AML\, CLL\, CML\) or non\-leukemia.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/leukemiasEset.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-leukemiaseset



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-leukemiaseset

and update with::

   conda update bioconductor-leukemiaseset



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-leukemiaseset.
