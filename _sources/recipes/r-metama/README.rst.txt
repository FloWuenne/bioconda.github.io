r-metama
========

|downloads|

Combines either p\-values or modified effect sizes from different studies to find differentially expressed genes

============= ===========
Home          https://CRAN.R-project.org/package=metaMA
Versions      3.1.2
License       GPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//r-metama/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-metama

and update with::

   conda update r-metama



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-metama.
