.. _`bioconductor-ffpe`:

bioconductor-ffpe
=================

|downloads|

Identify low\-quality data using metrics developed for expression data derived from Formalin\-Fixed\, Paraffin\-Embedded \(FFPE\) data.  Also a function for making Concordance at the Top plots \(CAT\-plots\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/ffpe.html
Versions      1.24.0, 1.22.0
License       GPL (>2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ffpe



Links         biotools: :biotools:`ffpe`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ffpe

and update with::

   conda update bioconductor-ffpe



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ffpe.
