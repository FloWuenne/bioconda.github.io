primerprospector
================

|downloads|

Primer Prospector is a pipeline of programs to design and analyze PCR primers\.

======== ===========
Home     http://pprospector.sourceforge.net/
Versions 1.0.1
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/primerprospector

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install primerprospector

and update with::

   conda update primerprospector



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/primerprospector.
