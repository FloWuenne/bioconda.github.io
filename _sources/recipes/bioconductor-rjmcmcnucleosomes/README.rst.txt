bioconductor-rjmcmcnucleosomes
==============================

|downloads|

This package does nucleosome positioning using informative Multinomial\-Dirichlet prior in a t\-mixture with reversible jump estimation of nucleosome positions for genome\-wide profiling.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/RJMCMCNucleosomes.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rjmcmcnucleosomes



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rjmcmcnucleosomes

and update with::

   conda update bioconductor-rjmcmcnucleosomes

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rjmcmcnucleosomes.
