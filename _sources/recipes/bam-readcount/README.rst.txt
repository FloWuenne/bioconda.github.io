.. _`bam-readcount`:

bam-readcount
=============

|downloads|

bam\-readcount generates metrics at single nucleotide positions.

============= ===========
Home          https://github.com/genome/bam-readcount
Versions      0.8
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bam-readcount



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bam-readcount

and update with::

   conda update bam-readcount



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bam-readcount.
