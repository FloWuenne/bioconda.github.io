bioconductor-ga4ghclient
========================

|downloads|

GA4GHclient provides an easy way to access public data servers through Global Alliance for Genomics and Health \(GA4GH\) genomics API. It provides low\-level access to GA4GH API and translates response data into Bioconductor\-based class objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/GA4GHclient.html
Versions      1.6.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-ga4ghclient/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ga4ghclient

and update with::

   conda update bioconductor-ga4ghclient



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ga4ghclient.
