bioconductor-genomewidesnp6crlmm
================================

|downloads|

Package with metadata for fast genotyping Affymetrix GenomeWideSnp\_6 arrays using the \'crlmm\' package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/genomewidesnp6Crlmm.html
Versions      1.0.7
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-genomewidesnp6crlmm/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genomewidesnp6crlmm

and update with::

   conda update bioconductor-genomewidesnp6crlmm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genomewidesnp6crlmm.
