bioconductor-mus.musculus
=========================

|downloads|

Contains the Mus.musculus object to access data from several related annotation packages.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/Mus.musculus.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mus.musculus



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mus.musculus

and update with::

   conda update bioconductor-mus.musculus



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mus.musculus.
