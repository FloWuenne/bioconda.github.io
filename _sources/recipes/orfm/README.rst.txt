orfm
====

|downloads|

OrfM is a simple and not slow ORF caller

======== ===========
Home     https://github.com/wwood/OrfM
Versions 0.6.1
License  GPL3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/orfm

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install orfm

and update with::

   conda update orfm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/orfm.
