talloc
======

|downloads|

talloc is a hierarchical\, reference counted memory pool system with destructors.

============= ===========
Home          https://talloc.samba.org/talloc/doc/html/index.html`
Versions      2.1.9
License       LGPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/talloc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install talloc

and update with::

   conda update talloc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/talloc.
