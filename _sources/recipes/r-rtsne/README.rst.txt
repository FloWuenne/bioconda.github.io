.. _`r-rtsne`:

r-rtsne
=======

|downloads|

An R wrapper around the fast T-distributed Stochastic Neighbor Embedding implementation by Van der Maaten.

======== ===========
Home     https://cran.r-project.org/web/packages/Rtsne/index.html
Versions 0.13
License  BSD 3-clause
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-rtsne
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-rtsne

and update with::

   conda update r-rtsne



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-rtsne.
