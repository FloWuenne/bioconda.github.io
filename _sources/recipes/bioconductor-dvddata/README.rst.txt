bioconductor-dvddata
====================

|downloads|

Data package which provides default drug and disease expression profiles for the DvD package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/DvDdata.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/bioconductor-dvddata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-dvddata

and update with::

   conda update bioconductor-dvddata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-dvddata.
