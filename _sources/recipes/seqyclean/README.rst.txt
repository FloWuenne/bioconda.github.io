.. _`seqyclean`:

seqyclean
=========

|downloads|

Main purpose of this software is to pre\-process NGS data in order to prepare for downstream analysis.

============= ===========
Home          https://github.com/ibest/seqyclean
Versions      1.10.07
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/seqyclean



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install seqyclean

and update with::

   conda update seqyclean



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/seqyclean.
