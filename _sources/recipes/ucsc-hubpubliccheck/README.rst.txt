ucsc-hubpubliccheck

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-hubpubliccheck

and update with::

   conda update ucsc-hubpubliccheck
