bioconductor-stringdb
=====================

|downloads|

The STRINGdb package provides a R interface to the STRING protein\-protein interactions database \(http\:\/\/www.string\-db.org\).

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/STRINGdb.html
Versions      1.20.0, 1.18.0, 1.16.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-stringdb



Links         biotools: :biotools:`stringdb`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-stringdb

and update with::

   conda update bioconductor-stringdb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-stringdb.
