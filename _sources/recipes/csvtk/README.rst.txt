Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install csvtk

and update with::

   conda update csvtk



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/csvtk.
