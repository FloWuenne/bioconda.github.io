r-qqman
=======

|downloads|

Create Q\-Q and manhattan plots for GWAS data from PLINK results.

============= ===========
Home          https://CRAN.R-project.org/package=qqman
Versions      0.1.4, 0.1.2
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/r-qqman/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-qqman

and update with::

   conda update r-qqman
