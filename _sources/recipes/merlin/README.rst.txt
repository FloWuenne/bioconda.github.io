.. _`merlin`:

merlin
======

|downloads|

MERLIN uses sparse trees to represent gene flow in pedigrees and is a fast pedigree analysis package

======== ===========
Home     http://csg.sph.umich.edu/abecasis/merlin
Versions 1.1.2
License  OpenSource
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/merlin

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install merlin

and update with::

   conda update merlin



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/merlin.
