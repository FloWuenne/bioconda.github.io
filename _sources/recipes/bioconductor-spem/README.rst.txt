bioconductor-spem
=================

|downloads|

This package can optimize the parameter in S\-system models given time series data

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/SPEM.html
Versions      1.20.0, 1.18.0, 1.16.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-spem/meta.yaml

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-spem

and update with::

   conda update bioconductor-spem



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-spem.
