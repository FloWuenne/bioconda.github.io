bioconductor-pint
=================

|downloads|

Pairwise data integration for functional genomics\, including tools for DNA\/RNA\/miRNA dependency screens\.

Versions      1.28.0

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-pint

and update with::

   conda update bioconductor-pint



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-pint.
