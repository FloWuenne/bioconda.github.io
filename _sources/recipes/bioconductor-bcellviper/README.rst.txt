bioconductor-bcellviper
=======================

|downloads|

This package provides a human B\-cell context\-specific transcriptional regulatory network and a human normal B\-cells dataset for the examples in package viper.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/bcellViper.html
Versions      
License       GPL (>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-bcellviper/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bcellviper

and update with::

   conda update bioconductor-bcellviper



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bcellviper.
