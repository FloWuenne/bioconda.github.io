perl-symbol
===========

|downloads|

manipulate Perl symbols and their names

======== ===========
Home     http://metacpan.org/pod/Symbol
Versions 1.07
License  perl_5
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-symbol/1.07

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-symbol

and update with::

   conda update perl-symbol



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-symbol.
