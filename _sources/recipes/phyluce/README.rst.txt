Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install phyluce

and update with::

   conda update phyluce



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/phyluce.
