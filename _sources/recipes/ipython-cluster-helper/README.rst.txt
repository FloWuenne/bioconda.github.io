ipython-cluster-helper
======================

|downloads|

Tool to easily start up an IPython cluster on different schedulers

======== ===========
Home     https://github.com/roryk/ipython-cluster-helper
Versions 0.5.0, 0.5.1, 0.5.2, 0.5.3, 0.5.4, 0.5.5, 0.5.6, 0.5.7, 0.5.8, 0.5.9, 0.6.0, 0.6.1
License  MIT
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ipython-cluster-helper

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ipython-cluster-helper

and update with::

   conda update ipython-cluster-helper



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ipython-cluster-helper.
