bioconductor-genotypeeval
=========================

|downloads|

Takes in a gVCF or VCF and reports metrics to assess quality of calls.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/genotypeeval.html
Versions      
License       file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-genotypeeval



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genotypeeval

and update with::

   conda update bioconductor-genotypeeval



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genotypeeval.
