tantan
======

|downloads|

tantan is a tool for masking simple regions \(low complexity and short\-period tandem repeats\) in biological sequences.

============= ===========
Home          http://cbrc3.cbrc.jp/~martin/tantan/
Versions      13
License       GPLv3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/tantan/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install tantan

and update with::

   conda update tantan



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/tantan.
