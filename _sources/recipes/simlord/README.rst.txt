.. _`simlord`:

simlord
=======

|downloads|

SimLoRD is a read simulator for long reads from third generation sequencing. Currently\, it supports the Pacific Biosciences SMRT error model.

============= ===========
Home          https://bitbucket.org/genomeinformatics/simlord/
Versions      1.0.2, 1.0.1, 0.7.3
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/simlord/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install simlord

and update with::

   conda update simlord



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/simlord.
