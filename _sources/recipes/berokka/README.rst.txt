berokka
=======

|downloads|

Trim\, circularise and orient long read bacterial genome assemblies.

============= ===========
Home          https://github.com/tseemann/berokka
Versions      0.1, 0.2
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/berokka



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install berokka

and update with::

   conda update berokka



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/berokka.
