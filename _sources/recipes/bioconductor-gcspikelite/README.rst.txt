Spike\-in data for GC\/MS data and methods within flagme

Versions      1.14.0, 1.16.0
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-gcspikelite



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-gcspikelite

and update with::

   conda update bioconductor-gcspikelite



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-gcspikelite.
