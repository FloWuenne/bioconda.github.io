perl-test2-suite

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-test2-suite

and update with::

   conda update perl-test2-suite



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-test2-suite.
