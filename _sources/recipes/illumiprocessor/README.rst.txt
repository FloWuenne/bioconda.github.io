illumiprocessor
===============

|downloads|

illumiprocessor is a tool to batch process illumina sequencing reads using the excellent trimmomatic package.

============= ===========
Home          https://github.com/faircloth-lab/illumiprocessor
Versions      2.0.9
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/illumiprocessor/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install illumiprocessor

and update with::

   conda update illumiprocessor



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/illumiprocessor.
