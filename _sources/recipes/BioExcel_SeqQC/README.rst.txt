Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioexcel_seqqc

and update with::

   conda update bioexcel_seqqc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioexcel_seqqc.
