bioconductor-riboseqr
=====================

|downloads|

Plotting functions\, frameshift detection and parsing of sequencing data from ribosome profiling experiments\.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/riboSeqR.html
Versions      1.10.0, 1.12.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-riboseqr



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-riboseqr

and update with::

   conda update bioconductor-riboseqr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-riboseqr.
