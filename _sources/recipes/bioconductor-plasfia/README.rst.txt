bioconductor-plasfia
====================

|downloads|

Positive Ionization FIA\-HRMS data of human plasma spiked with a pool of 40 compounds acquired in FIA\-HRMS mode on an orbitrap fusion. plasFIA also include the result of the processing by the proFIA package with adapted parameters for an Orbitrap Fusion.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/plasFIA.html
Versions      
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-plasfia/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-plasfia

and update with::

   conda update bioconductor-plasfia



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-plasfia.
