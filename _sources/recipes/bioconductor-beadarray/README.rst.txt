bioconductor-beadarray
======================

|downloads|

The package is able to read bead\-level data \(raw TIFFs and text files\) output by BeadScan as well as bead\-summary data from BeadStudio.  Methods for quality assessment and low\-level analysis are provided.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/beadarray.html
Versions      2.26.1, 2.28.0, 2.30.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-beadarray



Links         biotools: :biotools:`beadarray`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-beadarray

and update with::

   conda update bioconductor-beadarray



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-beadarray.
