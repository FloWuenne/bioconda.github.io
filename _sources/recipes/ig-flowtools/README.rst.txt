Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ig-flowtools

and update with::

   conda update ig-flowtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ig-flowtools.
