vcfpy
=====

|downloads|

Python 3 VCF library with good support for both reading and writing

============= ===========
Home          https://github.com/bihealth/vcfpy
Versions      0.11.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/vcfpy



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vcfpy

and update with::

   conda update vcfpy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vcfpy.
