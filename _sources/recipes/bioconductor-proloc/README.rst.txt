bioconductor-proloc
===================

|downloads|

The pRoloc package implements machine learning and visualisation methods for the analysis and interogation of quantitiative mass spectrometry data to reliably infer protein sub\-cellular localisation\.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/pRoloc.html
Versions      1.18.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-proloc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-proloc

and update with::

   conda update bioconductor-proloc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-proloc.
