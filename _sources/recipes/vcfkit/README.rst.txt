vcfkit
======

|downloads|

Assorted utilities for the variant call format

======== ===========
Home     https://github.com/AndersenLab/VCF-kit
Versions 0.0.4, 0.1.6
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/vcfkit
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vcfkit

and update with::

   conda update vcfkit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vcfkit.
