bioconductor-dbchip
===================

|downloads|

DBChIP detects differentially bound sharp binding sites across multiple conditions\, with or without matching control samples.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/DBChIP.html
Versions      1.24.0, 1.22.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-dbchip



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-dbchip

and update with::

   conda update bioconductor-dbchip



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-dbchip.
