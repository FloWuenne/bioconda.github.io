Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mintmap

and update with::

   conda update mintmap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mintmap.
