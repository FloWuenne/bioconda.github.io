.. _`msisensor`:

msisensor
=========

|downloads|

MSIsensor is a C\+\+ program to detect replication slippage variants at microsatellite regions\, and differentiate them as somatic or germline.

============= ===========
Home          https://github.com/ding-lab/msisensor
Versions      0.5
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//msisensor/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install msisensor

and update with::

   conda update msisensor



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/msisensor.
