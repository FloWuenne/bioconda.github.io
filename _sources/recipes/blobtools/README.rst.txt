blobtools

Modular command\-line solution for visualisation\, quality control and taxonomic partitioning of genome datasets

License       GPLv3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/blobtools



Links         doi: :doi:`10.12688/f1000research.12232.1`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install blobtools

and update with::

   conda update blobtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/blobtools.
