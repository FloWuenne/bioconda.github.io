bioconductor-rbowtie2
=====================

|downloads|

This package provides an R wrapper of the popular bowtie2 sequencing reads aligner and AdapterRemoval\, a convenient tool for rapid adapter trimming\, identification\, and read merging.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/Rbowtie2.html
Versions      
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rbowtie2



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rbowtie2

and update with::

   conda update bioconductor-rbowtie2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rbowtie2.
