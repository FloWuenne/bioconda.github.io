bioconductor-tcgabiolinksgui.data
=================================

|downloads|

Supporting data for the TCGAbiolinksGUI package. It includes the following objects\: glioma.gcimp.model\, glioma.idhwt.model glioma.idhmut.model\,glioma.idh.mode\, probes2rm\, maf.tumor\,GDCdisease.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-tcgabiolinksgui.data

and update with::

   conda update bioconductor-tcgabiolinksgui.data

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-tcgabiolinksgui.data.
