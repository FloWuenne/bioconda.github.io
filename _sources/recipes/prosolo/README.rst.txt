prosolo
=======

|downloads|

A highly sensitive and accurate Bayesian caller for variants in single cell sequencing data.

============= ===========
Home          https://github.com/prosolo/prosolo/tree/v0.6.1
Versions      0.6.1, 0.6.0, 0.5.0, 0.4.0, 0.3.1, 0.2.0
License       GPLv3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//prosolo/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install prosolo

and update with::

   conda update prosolo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/prosolo.
