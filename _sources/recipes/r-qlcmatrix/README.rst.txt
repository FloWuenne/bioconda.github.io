r-qlcmatrix
===========

|downloads|

Extension of the functionality of the Matrix package for using sparse matrices. Some of the functions are very general, while other are highly specific for special data format as used for quantitative language comparison (QLC).

======== ===========
Home     https://CRAN.R-project.org/package=qlcMatrix
Versions 0.9.5
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-qlcmatrix
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-qlcmatrix

and update with::

   conda update r-qlcmatrix



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-qlcmatrix.
