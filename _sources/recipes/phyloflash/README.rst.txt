.. _`phyloflash`:

phyloflash
==========

|downloads|

phyloFlash is a pipeline to rapidly reconstruct the SSU rRNAs and explore
phylogenetic composition of an illumina (meta)genomic dataset.


======== ===========
Home     https://github.com/HRGV/phyloFlash
Versions 2.0beta6, 3.0b1
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/phyloflash
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install phyloflash

and update with::

   conda update phyloflash



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/phyloflash.
