secimtools
==========

Metabolomics tools from the SECIM project

============= ===========

Versions      1.0.0
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/secimtools/meta.yaml



============= ===========

SECIMTools project aims to develop a suite of tools for processing of metabolomics data\, which can be run in a standalone mode or via Galaxy Genomics Framework.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install secimtools

and update with::

   conda update secimtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/secimtools.
