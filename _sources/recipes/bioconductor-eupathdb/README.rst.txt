.. _`bioconductor-eupathdb`:

bioconductor-eupathdb
=====================

|downloads|

Brings together annotation resources from the various EuPathDB databases \(PlasmoDB\, ToxoDB\, TriTrypDB\, etc.\) and makes them available in R using the AnnotationHub framework.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/EuPathDB.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-eupathdb/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-eupathdb

and update with::

   conda update bioconductor-eupathdb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-eupathdb.
