Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genemeta

and update with::

   conda update bioconductor-genemeta



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genemeta.
