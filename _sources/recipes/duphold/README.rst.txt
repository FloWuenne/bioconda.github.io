.. _`duphold`:

duphold
=======

|downloads|

SV callers like lumpy look at split\-reads and pair distances to find structural variants. This tool is a fast way to add depth information to those calls.

============= ===========
Home          https://github.com/brentp/duphold
Versions      0.0.2, 0.0.3, 0.0.6, 0.0.9
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/duphold



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install duphold

and update with::

   conda update duphold



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/duphold.
