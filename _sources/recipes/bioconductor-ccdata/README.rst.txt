bioconductor-ccdata
===================

|downloads|

This package contains microarray gene expression data generated from the Connectivity Map build 02 and LINCS l1000. The data are used by the ccmap package to find drugs and drug combinations to mimic or reverse a gene expression signature.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/ccdata.html
Versions      1.8.0
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-ccdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ccdata

and update with::

   conda update bioconductor-ccdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ccdata.
