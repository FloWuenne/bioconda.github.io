gvcfgenotyper
=============

|downloads|

A utility for merging and genotyping Illumina\-style GVCFs.

============= ===========
Home          https://github.com/Illumina/gvcfgenotyper
Versions      2018.10.15
License       Apache 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/gvcfgenotyper



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gvcfgenotyper

and update with::

   conda update gvcfgenotyper



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gvcfgenotyper.
