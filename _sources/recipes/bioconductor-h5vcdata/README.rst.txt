bioconductor-h5vcdata
=====================

|downloads|

This package contains the data used in the vignettes and examples of the \'h5vc\' package

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/h5vcData.html
Versions      2.2.0
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-h5vcdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-h5vcdata

and update with::

   conda update bioconductor-h5vcdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-h5vcdata.
