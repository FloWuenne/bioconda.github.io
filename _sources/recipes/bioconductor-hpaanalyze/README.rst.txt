bioconductor-hpaanalyze
=======================

|downloads|

Provide functions for retrieving\, exploratory analyzing and visualizing the Human Protein Atlas data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/HPAanalyze.html
Versions      
License       GPL-3 + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-hpaanalyze



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-hpaanalyze

and update with::

   conda update bioconductor-hpaanalyze



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-hpaanalyze.
