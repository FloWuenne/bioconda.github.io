Comparing runs of Oxford Nanopore sequencing data and alignments

Versions 0.12.4, 0.5.0, 0.7.0
License  MIT License
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/nanocomp
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanocomp

and update with::

   conda update nanocomp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanocomp.
