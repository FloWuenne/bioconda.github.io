bioconductor-mspuritydata
=========================

|downloads|

Fragmentation spectral libraries and data to test the msPurity package

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/msPurityData.html
Versions      1.10.0, 1.8.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-mspuritydata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mspuritydata

and update with::

   conda update bioconductor-mspuritydata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mspuritydata.
