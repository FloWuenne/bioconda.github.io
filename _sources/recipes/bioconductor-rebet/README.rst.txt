bioconductor-rebet
==================

|downloads|

There is an increasing focus to investigate the association between rare variants and diseases. The REBET package implements the subREgion\-based BurdEn Test which is a powerful burden test that simultaneously identifies susceptibility loci and sub\-regions.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/REBET.html
Versions      1.0.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-rebet/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rebet

and update with::

   conda update bioconductor-rebet



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rebet.
