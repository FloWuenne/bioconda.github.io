loompy

Work with .loom files for single\-cell RNA\-seq data

Versions      1.0.2, 1.1.0, 2.0.9
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/loompy



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install loompy

and update with::

   conda update loompy

A Docker container is available at https://quay.io/repository/biocontainers/loompy.
