Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install epa-ng

and update with::

   conda update epa-ng



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/epa-ng.
