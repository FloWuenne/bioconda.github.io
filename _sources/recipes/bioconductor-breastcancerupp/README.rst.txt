bioconductor-breastcancerupp
============================

|downloads|

Gene expression data from a breast cancer study published by Miller et al. in 2005\, provided as an eSet.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/breastCancerUPP.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-breastcancerupp



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-breastcancerupp

and update with::

   conda update bioconductor-breastcancerupp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-breastcancerupp.
