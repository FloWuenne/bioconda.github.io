bioconductor-readat
===================

|downloads|

This package contains functionality to import\, transform and annotate data from ADAT files generated by the SomaLogic SOMAscan platform.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/readat.html
Versions      
License       GPL-3

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-readat

and update with::

   conda update bioconductor-readat

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-readat.
