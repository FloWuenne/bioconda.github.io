bioconductor-qualifier
======================

|downloads|

Provides quality control and quality assessment tools for gated flow cytometry data.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/QUALIFIER.html
Versions      1.20.0, 1.22.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-qualifier



Links         biotools: :biotools:`qualifier`, doi: :doi:`10.1186/1471-2105-13-252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-qualifier

and update with::

   conda update bioconductor-qualifier



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-qualifier.
