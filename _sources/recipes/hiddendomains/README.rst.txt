hiddendomains
=============

|downloads|

hiddenDomains is a suite of programs used to identify significant enrichment of ChIP\-seq reads that span large domains.

============= ===========
Home          http://hiddendomains.sourceforge.net/
Versions      3.0
License       GNU GPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//hiddendomains/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hiddendomains

and update with::

   conda update hiddendomains



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hiddendomains.
