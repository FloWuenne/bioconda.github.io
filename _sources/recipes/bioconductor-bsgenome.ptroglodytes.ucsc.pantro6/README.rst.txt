bioconductor-bsgenome.ptroglodytes.ucsc.pantro6
===============================================

|downloads|

Full genome sequences for Pan troglodytes \(Chimp\) as provided by UCSC \(panTro6\, Jan. 2018\) and stored in Biostrings objects.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.ptroglodytes.ucsc.pantro6

and update with::

   conda update bioconductor-bsgenome.ptroglodytes.ucsc.pantro6

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.ptroglodytes.ucsc.pantro6.
