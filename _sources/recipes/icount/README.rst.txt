icount
======

|downloads|

Computational pipeline for analysis of iCLIP data

============= ===========
Home          https://github.com/tomazc/iCount
Versions      2.0.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/icount/meta.yaml



============= ===========

iCount is a Python module and associated command\-line interface \(CLI\)\, which provides all the commands needed to process iCLIP data on protein\-RNA interactions.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install icount

and update with::

   conda update icount



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/icount.
