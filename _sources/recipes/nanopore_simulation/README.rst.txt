nanopore_simulation
===================

|downloads|

Nanopore SimulatION is a tool for simulating an Oxfornd Nanopore Technologies MinION device for bioinformatic development.

============= ===========
Home          https://github.com/crohrandt/nanopore_simulation
Versions      0.3
License       MPL-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/nanopore_simulation/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanopore_simulation

and update with::

   conda update nanopore_simulation



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanopore_simulation.
