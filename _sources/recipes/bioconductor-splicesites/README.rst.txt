bioconductor-splicesites
========================

|downloads|

Performs splice centered analysis on RNA\-seq data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/spliceSites.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-splicesites



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-splicesites

and update with::

   conda update bioconductor-splicesites



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-splicesites.
