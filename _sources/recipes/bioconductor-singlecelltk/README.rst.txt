bioconductor-singlecelltk
=========================

|downloads|

Run common single cell analysis directly through your browser including differential expression\, downsampling analysis\, and clustering.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/singleCellTK.html
Versions      1.2.3
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-singlecelltk/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-singlecelltk

and update with::

   conda update bioconductor-singlecelltk



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-singlecelltk.
