red
===

|downloads|

Red \(RepeatsDetector\)\: an intelligent\, rapid\, accurate tool for detecting repeats de\-novo on the genomic scale.

============= ===========
Home          http://toolsmith.ens.utulsa.edu
Versions      2015.05.22
License       Custom OSS
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//red/meta.yaml



Links         doi: :doi:`10.1186/s12859-015-0654-5`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install red

and update with::

   conda update red



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/red.
