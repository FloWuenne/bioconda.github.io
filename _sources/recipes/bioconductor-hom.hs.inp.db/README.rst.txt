Homology information for Homo Sapiens from Inparanoid assembled using data from public repositories

Home          http://bioconductor.org/packages/3.7/data/annotation/html/hom.Hs.inp.db.html
Versions      3.1.2
License       Artistic-2.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-hom.hs.inp.db

and update with::

   conda update bioconductor-hom.hs.inp.db
