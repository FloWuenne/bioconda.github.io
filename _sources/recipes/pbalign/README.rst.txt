pbalign
=======

|downloads|

Python wrapper for producing PBBAM valid alignments

============= ===========
Home          https://github.com/PacificBiosciences/pbalign
Versions      0.3.2, 0.3.1
License       BSD-3-Clause-Clear
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//pbalign/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pbalign

and update with::

   conda update pbalign



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pbalign.
