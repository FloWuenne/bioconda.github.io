r-corbi
=======

|downloads|

Provides a bundle of basic and fundamental bioinformatics tools\, such as network querying and alignment\, subnetwork extraction and search\, network biomarker identification.

============= ===========
Home          https://github.com/wulingyun/Corbi
Versions      0.4_2
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/r-corbi/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-corbi

and update with::

   conda update r-corbi
