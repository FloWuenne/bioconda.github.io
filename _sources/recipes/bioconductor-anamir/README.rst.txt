bioconductor-anamir
===================

|downloads|

This package is intended to identify potential interactions of miRNA\-target gene interactions from miRNA and mRNA expression data. It contains functions for statistical test\, databases of miRNA\-target gene interaction and functional analysis.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/anamiR.html
Versions      1.10.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-anamir/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-anamir

and update with::

   conda update bioconductor-anamir

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-anamir.
