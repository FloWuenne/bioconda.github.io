bioconductor-normqpcr
=====================

|downloads|

Functions for the selection of optimal reference genes and the normalisation of real\-time quantitative PCR data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/NormqPCR.html
Versions      1.28.0
License       LGPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-normqpcr/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-normqpcr

and update with::

   conda update bioconductor-normqpcr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-normqpcr.
