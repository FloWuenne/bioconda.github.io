bioconductor-dyndoc
===================

|downloads|

A set of functions to create and interact with dynamic documents and vignettes.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/DynDoc.html
Versions      1.56.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-dyndoc



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-dyndoc

and update with::

   conda update bioconductor-dyndoc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-dyndoc.
