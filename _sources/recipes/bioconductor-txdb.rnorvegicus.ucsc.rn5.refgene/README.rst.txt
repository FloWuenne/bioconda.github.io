Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-txdb.rnorvegicus.ucsc.rn5.refgene

and update with::

   conda update bioconductor-txdb.rnorvegicus.ucsc.rn5.refgene
