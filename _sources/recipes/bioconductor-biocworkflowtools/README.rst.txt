bioconductor-biocworkflowtools
==============================

|downloads|

Provides functions to ease the transition between Rmarkdown and LaTeX documents when authoring a Bioconductor Workflow.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/BiocWorkflowTools.html
Versions      1.2.0, 1.4.0
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-biocworkflowtools



Links         biotools: :biotools:`BiocWorkflowTools`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-biocworkflowtools

and update with::

   conda update bioconductor-biocworkflowtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-biocworkflowtools.
