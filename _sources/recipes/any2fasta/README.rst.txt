any2fasta
=========

|downloads|

Convert various sequence formats to FASTA

============= ===========
Home          https://github.com/tseemann/any2fasta
Versions      0.4.2
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/any2fasta



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install any2fasta

and update with::

   conda update any2fasta



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/any2fasta.
