Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install rgt

and update with::

   conda update rgt



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/rgt.
