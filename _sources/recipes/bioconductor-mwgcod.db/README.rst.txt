bioconductor-mwgcod.db
======================

|downloads|

Codelink Mouse Whole Genome Bioarray \(\~36 000 mouse gene targets\) annotation data \(chip mwgcod\) assembled using data from public repositories

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/mwgcod.db.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mwgcod.db



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mwgcod.db

and update with::

   conda update bioconductor-mwgcod.db



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mwgcod.db.
