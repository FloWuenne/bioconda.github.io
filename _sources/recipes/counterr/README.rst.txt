.. _`counterr`:

counterr
========

|downloads|

Counterr is a light\-weight command line tool that computes errors in sequencing data by comparing the reads to a reference genome.

============= ===========
Home          https://github.com/dayzerodx/counterr
Versions      0.1
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//counterr/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install counterr

and update with::

   conda update counterr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/counterr.
