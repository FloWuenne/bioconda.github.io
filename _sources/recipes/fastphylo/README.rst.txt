.. _`fastphylo`:

fastphylo
=========

|downloads|

Fastphylo is software project containing the implementations of the algorithms \"Fast Computation of Distance Estimators\" and \"Fast Neighbor Joining\".

============= ===========
Home          https://github.com/arvestad/FastPhylo
Versions      1.0.3
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/fastphylo/meta.yaml



Links         doi: :doi:`10.1186/1471-2105-14-334`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fastphylo

and update with::

   conda update fastphylo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/fastphylo.
