bioconductor-bsgenome.drerio.ucsc.danrer7
=========================================

|downloads|

Full genome sequences for Danio rerio \(Zebrafish\) as provided by UCSC \(danRer7\, Jul. 2010\) and stored in Biostrings objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Drerio.UCSC.danRer7.html
Versions      1.4.0
License       Artistic-2.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.drerio.ucsc.danrer7

and update with::

   conda update bioconductor-bsgenome.drerio.ucsc.danrer7
