.. _`rnablueprint`:

rnablueprint
============

|downloads|

The RNAblueprint library solves the problem of uniformly sampling RNA\/DNA sequences compatible to multiple structural constraints and sequence constraints.

============= ===========
Home          https://github.com/ViennaRNA/RNAblueprint
Versions      1.2.2, 1.2
License       GPL3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//rnablueprint/meta.yaml



Links         doi: :doi:`10.1093/bioinformatics/btx263`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install rnablueprint

and update with::

   conda update rnablueprint



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/rnablueprint.
