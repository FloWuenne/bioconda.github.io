======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanoplotter

and update with::

   conda update nanoplotter



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanoplotter.
