bioconductor-biodist
====================

|downloads|

A collection of software tools for calculating distance measures\.

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/bioDist.html
Versions 1.48.0, 1.50.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-biodist

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-biodist

and update with::

   conda update bioconductor-biodist



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-biodist.
