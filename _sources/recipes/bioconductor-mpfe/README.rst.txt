bioconductor-mpfe
=================

|downloads|

Estimate distribution of methylation patterns from a table of counts from a bisulphite sequencing experiment given a non\-conversion rate and read error rate.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/MPFE.html
Versions      1.18.0, 1.16.0, 1.14.0, 1.12.0
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-mpfe/meta.yaml



Links         biotools: :biotools:`mpfe`, doi: :doi:`10.1186/s12859-015-0600-6`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mpfe

and update with::

   conda update bioconductor-mpfe



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mpfe.
