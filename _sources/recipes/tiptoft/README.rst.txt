tiptoft
=======

|downloads|

Predict plasmids from uncorrected long read data.

============= ===========
Home          https://github.com/andrewjpage/tiptoft
Versions      1.0.0, 0.1.4
License       GPL3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/tiptoft


Development   https://github.com/andrewjpage/tiptoft


============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install tiptoft

and update with::

   conda update tiptoft



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/tiptoft.
