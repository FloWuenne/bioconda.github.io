bioconductor-genomicdatacommons
===============================

|downloads|

Programmatically access the NIH \/ NCI Genomic Data Commons RESTful service\.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/GenomicDataCommons.html
Versions      1.0.5, 1.2.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-genomicdatacommons



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genomicdatacommons

and update with::

   conda update bioconductor-genomicdatacommons



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genomicdatacommons.
