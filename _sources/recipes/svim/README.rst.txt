svim
====

|downloads|

SVIM is a structural variant caller for long reads.

============= ===========
Home          https://github.com/eldariont/svim
Versions      0.4.1, 0.4.2
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/svim



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install svim

and update with::

   conda update svim



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/svim.
