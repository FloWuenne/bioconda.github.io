bioconductor-lungexpression
===========================

|downloads|

Data from three large lung cancer studies provided as ExpressionSets

Home          https://bioconductor.org/packages/3.8/data/experiment/html/lungExpression.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-lungexpression/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-lungexpression

and update with::

   conda update bioconductor-lungexpression



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-lungexpression.
