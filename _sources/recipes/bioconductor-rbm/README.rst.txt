bioconductor-rbm
================

|downloads|

Use A Resampling\-Based Empirical Bayes Approach to Assess Differential Expression in Two\-Color Microarrays and RNA\-Seq data sets.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/RBM.html
Versions      1.10.0, 1.8.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rbm



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rbm

and update with::

   conda update bioconductor-rbm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rbm.
