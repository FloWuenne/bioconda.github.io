bioconductor-maqcexpression4plex
================================

|downloads|

Data from human \(HG18\) 4plex NimbleGen array. It has 24k genes with 3 60mer probes per gene.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/maqcExpression4plex.html
Versions      
License       GPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-maqcexpression4plex/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-maqcexpression4plex

and update with::

   conda update bioconductor-maqcexpression4plex



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-maqcexpression4plex.
