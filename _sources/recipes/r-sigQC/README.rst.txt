.. _`r-sigqc`:

r-sigqc
=======

|downloads|

Provides gene signature quality control metrics in publication ready plots. Namely\, enables the visualization of properties such as expression\, variability\, correlation\, and comparison of methods of standardisation and scoring metrics.

============= ===========
Home          https://CRAN.R-project.org/package=sigQC
Versions      0.1.21, 0.1.14
License       file LICENSE (Restricts use)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-sigQC



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-sigqc

and update with::

   conda update r-sigqc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-sigqc.
