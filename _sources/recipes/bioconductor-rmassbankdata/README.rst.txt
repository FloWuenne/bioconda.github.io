.. _`bioconductor-rmassbankdata`:

bioconductor-rmassbankdata
==========================

|downloads|

Example spectra\, example compound list\(s\) and an example annotation list for a narcotics dataset\; required to test RMassBank. The package is described in the man page for RMassBankData. Includes new XCMS test data.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/RMassBankData.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rmassbankdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rmassbankdata

and update with::

   conda update bioconductor-rmassbankdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rmassbankdata.
