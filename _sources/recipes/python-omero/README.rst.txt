python-omero
============

|downloads|

Client library offering helper methods to access an OMERO server.

============= ===========
Home          https://www.openmicroscopy.org/
Versions      5.2.7, 5.2.8, 5.3.0, 5.3.1, 5.3.2, 5.3.3, 5.4.0, 5.4.1, 5.4.2, 5.4.3, 5.4.4, 5.4.5, 5.4.6, 5.4.7
License       GPL 2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/python-omero



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install python-omero

and update with::

   conda update python-omero



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/python-omero.
