regtools
========

|downloads|

Tools that integrate DNA\-seq and RNA\-seq data to help interpret mutations in a regulatory and splicing context.

Home          https://github.com/griffithlab/regtools/
Versions      0.5.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/regtools/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install regtools

and update with::

   conda update regtools
