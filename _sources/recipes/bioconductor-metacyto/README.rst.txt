bioconductor-metacyto
=====================

|downloads|

This package provides functions for preprocessing\, automated gating and meta\-analysis of cytometry data. It also provides functions that facilitate the collection of cytometry data from the ImmPort database.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/MetaCyto.html
Versions      1.4.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-metacyto/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-metacyto

and update with::

   conda update bioconductor-metacyto



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-metacyto.
