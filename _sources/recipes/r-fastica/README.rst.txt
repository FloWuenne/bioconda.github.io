r-fastica
=========

|downloads|

Implementation of FastICA algorithm to perform Independent Component Analysis (ICA) and Projection Pursuit.

======== ===========
Home     https://CRAN.R-project.org/package=fastICA
Versions 1.2_0
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-fastica
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-fastica

and update with::

   conda update r-fastica



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-fastica.
