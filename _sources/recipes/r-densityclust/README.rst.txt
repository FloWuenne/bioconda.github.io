.. _`r-densityclust`:

r-densityclust
==============

|downloads|

An implementation of the clustering algorithm described by Alex Rodriguez and Alessandro Laio (Science, 2014 vol. 344), along with tools to inspect and visualize the results.

======== ===========
Home     https://CRAN.R-project.org/package=densityClust
Versions 0.2.1
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-densityclust
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-densityclust

and update with::

   conda update r-densityclust



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-densityclust.
