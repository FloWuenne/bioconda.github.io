Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install rapmap

and update with::

   conda update rapmap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/rapmap.
