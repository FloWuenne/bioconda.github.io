.. _`bioconductor-brainstars`:

bioconductor-brainstars
=======================

|downloads|

This package can search and get gene expression data and plots from BrainStars (B*). BrainStars is a quantitative expression database of the adult mouse brain. The database has genome-wide expression profile at 51 adult mouse CNS regions.

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/BrainStars.html
Versions 1.22.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-brainstars
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-brainstars

and update with::

   conda update bioconductor-brainstars



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-brainstars.
