bioconductor-nnnorm
===================

|downloads|

This package allows to detect and correct for spatial and intensity biases with two\-channel microarray data. The normalization method implemented in this package is based on robust neural networks fitting.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/nnNorm.html
Versions      2.40.0, 2.42.0, 2.44.0
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-nnnorm



Links         biotools: :biotools:`nnnorm`, doi: :doi:`10.1093/bioinformatics/bti397`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-nnnorm

and update with::

   conda update bioconductor-nnnorm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-nnnorm.
