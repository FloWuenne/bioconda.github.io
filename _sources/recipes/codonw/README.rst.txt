.. _`codonw`:

codonw
======

|downloads|

CodonW is a programme designed to simplify the Multivariate analysis \(correspondence analysis\) of codon and amino acid usage.

============= ===========
Home          http://codonw.sourceforge.net
Versions      1.4.4
License       GPLv2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/codonw



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install codonw

and update with::

   conda update codonw



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/codonw.
