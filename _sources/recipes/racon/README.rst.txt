racon
=====

|downloads|

Ultrafast consensus module for raw de novo genome assembly of long uncorrected reads.

============= ===========
Home          https://github.com/isovic/racon
Versions      0.5.0, 1.0.0, 1.0.1, 1.1.0, 1.2.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/racon



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install racon

and update with::

   conda update racon



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/racon.
