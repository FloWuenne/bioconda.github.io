bioconductor-bsgenome.drerio.ucsc.danrer6
=========================================

|downloads|

Full genome sequences for Danio rerio \(Zebrafish\) as provided by UCSC \(danRer6\, Dec. 2008\) and stored in Biostrings objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Drerio.UCSC.danRer6.html
Versions      1.4.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-bsgenome.drerio.ucsc.danrer6/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.drerio.ucsc.danrer6

and update with::

   conda update bioconductor-bsgenome.drerio.ucsc.danrer6



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.drerio.ucsc.danrer6.
