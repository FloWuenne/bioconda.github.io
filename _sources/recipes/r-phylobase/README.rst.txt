r-phylobase
===========

|downloads|

Provides a base S4 class for comparative methods\, incorporating one or more trees and trait data.

============= ===========
Home          https://github.com/fmichonneau/phylobase
Versions      0.8.4
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-phylobase



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-phylobase

and update with::

   conda update r-phylobase



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-phylobase.
