.. _`r-tcr`:

r-tcr
=====

|downloads|

Platform for the advanced analysis of T cell receptor and Immunoglobulin repertoires data and visualisation of the analysis results.

============= ===========
Home          http://imminfo.github.io/tcr/
Versions      
License       Apache License 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-tcr



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-tcr

and update with::

   conda update r-tcr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-tcr.
