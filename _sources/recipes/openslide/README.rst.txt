.. _`openslide`:

openslide
=========

|downloads|

OpenSlide is a C library that provides a simple interface to read whole\-slide images \(also known as virtual slides\).

============= ===========
Home          http://openslide.org/
Versions      3.4.1
License       GNU LGPL 2.1
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/openslide



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install openslide

and update with::

   conda update openslide



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/openslide.
