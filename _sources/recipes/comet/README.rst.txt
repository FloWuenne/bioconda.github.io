comet-ms
========

|downloads|

Comet is a command line tool that does MS\/MS database search.

============= ===========
Home          http://comet-ms.sourceforge.net/
Versions      2016013, 2018012
License       Apache License 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/comet



Links         doi: :doi:`10.1007/s13361-015-1179-x`, doi: :doi:`10.1002/pmic.201200439`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install comet-ms

and update with::

   conda update comet-ms



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/comet-ms.
