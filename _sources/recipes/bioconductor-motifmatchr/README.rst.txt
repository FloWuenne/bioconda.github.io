bioconductor-motifmatchr
========================

|downloads|

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-motifmatchr

and update with::

   conda update bioconductor-motifmatchr
