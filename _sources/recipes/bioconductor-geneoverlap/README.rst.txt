bioconductor-geneoverlap
========================

|downloads|

Test two sets of gene lists and visualize the results.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/GeneOverlap.html
Versions      1.16.0, 1.14.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-geneoverlap



Links         biotools: :biotools:`geneoverlap`, doi: :doi:`10.1167/iovs.16-20618`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-geneoverlap

and update with::

   conda update bioconductor-geneoverlap



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-geneoverlap.
