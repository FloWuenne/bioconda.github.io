bioconductor-affylmgui
======================

|downloads|

A Graphical User Interface \(GUI\) for analysis of Affymetrix microarray gene expression data using the affy and limma packages.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/affylmGUI.html
Versions      1.56.0
License       GPL (>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-affylmgui/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-affylmgui

and update with::

   conda update bioconductor-affylmgui



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-affylmgui.
