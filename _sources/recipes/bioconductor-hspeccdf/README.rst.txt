bioconductor-hspeccdf
=====================

|downloads|

A package containing an environment representing the HGU133Plus2\_Hs\_Hspec.cdf file.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/hspeccdf.html
Versions      
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-hspeccdf



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-hspeccdf

and update with::

   conda update bioconductor-hspeccdf

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-hspeccdf.
