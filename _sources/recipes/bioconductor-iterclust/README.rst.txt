Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-iterclust

and update with::

   conda update bioconductor-iterclust



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-iterclust.
