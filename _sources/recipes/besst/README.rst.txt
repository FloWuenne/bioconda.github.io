License       GPL-3.0

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install besst

and update with::

   conda update besst



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/besst.
