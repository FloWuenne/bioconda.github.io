.. _`bioconductor-flowpeaks`:

bioconductor-flowpeaks
======================

|downloads|

A fast and automatic clustering to classify the cells into subpopulations based on finding the peaks from the overall density function generated by K\-means.

============= ===========
Home          https://bioconductor.org/packages/release/bioc/html/flowPeaks.html
Versions      1.20.0
License       Artistic-1.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-flowpeaks



Links         biotools: :biotools:`flowpeaks`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowpeaks

and update with::

   conda update bioconductor-flowpeaks



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowpeaks.
