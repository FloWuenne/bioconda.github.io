Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install lace

and update with::

   conda update lace

Notes
-----

Prefix with 'Lace_' for the script name that is rather generic: 'Checker.py'.

|docker|

A Docker container is available at https://quay.io/repository/biocontainers/lace.
