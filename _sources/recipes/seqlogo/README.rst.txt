seqlogo
=======

|downloads|

Python port of the R Bioconductor \`seqlogo\` package

============= ===========
Home          https://github.com/betteridiot/seqlogo
Versions      0.1.12
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/seqlogo



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install seqlogo

and update with::

   conda update seqlogo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/seqlogo.
