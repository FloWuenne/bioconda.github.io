bioconductor-homo.sapiens
=========================

|downloads|

Contains the Homo.sapiens object to access data from several related annotation packages.

======== ===========
Home     http://bioconductor.org/packages/3.6/data/annotation/html/Homo.sapiens.html
Versions 1.3.1
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-homo.sapiens
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-homo.sapiens

and update with::

   conda update bioconductor-homo.sapiens



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-homo.sapiens.
