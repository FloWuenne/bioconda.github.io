bioconductor-phosphonormalizer
==============================

|downloads|

It uses the overlap between enriched and non\-enriched datasets to compensate for the bias introduced in global phosphorylation after applying median normalization.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/phosphonormalizer.html
Versions      1.4.0, 1.2.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-phosphonormalizer/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-phosphonormalizer

and update with::

   conda update bioconductor-phosphonormalizer



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-phosphonormalizer.
