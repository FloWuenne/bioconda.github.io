bioconductor-frmatools
======================

|downloads|

Tools for advanced use of the frma microarray package.

============= ===========
Home          https://www.bioconductor.org/packages/release/bioc/html/frmaTools.html
Versions      1.28.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-frmatools



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-frmatools

and update with::

   conda update bioconductor-frmatools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-frmatools.
