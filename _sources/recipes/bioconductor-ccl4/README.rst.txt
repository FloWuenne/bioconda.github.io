bioconductor-ccl4
=================

|downloads|

NChannelSet for rat hepatocytes treated with Carbon Tetrachloride \(CCl4\) data from LGC company.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/CCl4.html
Versions      1.20.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-ccl4/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ccl4

and update with::

   conda update bioconductor-ccl4



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ccl4.
