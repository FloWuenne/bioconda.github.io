diamond_add_taxonomy
====================

|downloads|

Utility to work with NCBI taxonomy database including tool to annotate DIAMOND results with taxonomy lineage

============= ===========
Home          https://github.com/pvanheus/diamond_add_taxonomy
Versions      
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/diamond_add_taxonomy



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install diamond_add_taxonomy

and update with::

   conda update diamond_add_taxonomy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/diamond_add_taxonomy.
