.. _`bioconductor-minionsummarydata`:

bioconductor-minionsummarydata
==============================

|downloads|

Summarised MinION sequencing data for Salmonella Typhi published by Ashton et al. in 2015. Three replicate runs are each provided as Fast5Summary objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/minionSummaryData.html
Versions      
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/bioconductor-minionsummarydata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-minionsummarydata

and update with::

   conda update bioconductor-minionsummarydata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-minionsummarydata.
