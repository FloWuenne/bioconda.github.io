.. _`bioconductor-prolocdata`:

bioconductor-prolocdata
=======================

|downloads|

Mass\-spectrometry based spatial proteomics data sets and protein complex separation data. Also contains the time course expression experiment from Mulvey et al. 2015.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/pRolocdata.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-prolocdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-prolocdata

and update with::

   conda update bioconductor-prolocdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-prolocdata.
