.. _`bioconductor-shinymethyl`:

bioconductor-shinymethyl
========================

|downloads|

Interactive tool for visualizing Illumina methylation array data. Both the 450k and EPIC array are supported.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/shinyMethyl.html
Versions      1.14.0, 1.16.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-shinymethyl



Links         biotools: :biotools:`shinymethyl`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-shinymethyl

and update with::

   conda update bioconductor-shinymethyl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-shinymethyl.
