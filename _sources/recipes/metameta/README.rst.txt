Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install metameta

and update with::

   conda update metameta



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/metameta.
