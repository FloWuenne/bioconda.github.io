======== ===========
Home     http://bioconductor.org/packages/release/bioc/html/MassSpecWavelet.html
Versions 1.40.0, 1.42.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-massspecwavelet

and update with::

   conda update bioconductor-massspecwavelet

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-massspecwavelet.
