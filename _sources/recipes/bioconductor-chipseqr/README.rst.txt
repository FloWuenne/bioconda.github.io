bioconductor-chipseqr
=====================

|downloads|

ChIPseqR identifies protein binding sites from ChIP\-seq and nucleosome positioning experiments. The model used to describe binding events was developed to locate nucleosomes but should flexible enough to handle other types of experiments as well.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/ChIPseqR.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-chipseqr



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-chipseqr

and update with::

   conda update bioconductor-chipseqr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-chipseqr.
