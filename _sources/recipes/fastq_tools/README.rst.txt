fastq-tools
===========

|downloads|

A collection of fastq manipulation scripts written in C for speed.

============= ===========
Home          http://homes.cs.washington.edu/~dcjones/fastq-tools/
Versions      0.8
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/fastq_tools/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fastq-tools

and update with::

   conda update fastq-tools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/fastq-tools.
