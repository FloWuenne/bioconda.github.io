.. _`r-exactranktests`:

r-exactranktests
================

|downloads|

Computes exact conditional p-values and quantiles using an implementation of the Shift-Algorithm by Streitberg & Roehmel.

======== ===========
Home     https://cran.rstudio.com/web/packages/exactRankTests/index.html
Versions 0.8_29
License  GPL-2|GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-exactranktests
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-exactranktests

and update with::

   conda update r-exactranktests



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-exactranktests.
