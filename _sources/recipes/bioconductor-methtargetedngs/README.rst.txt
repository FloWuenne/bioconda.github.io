bioconductor-methtargetedngs
============================

|downloads|

Perform step by step methylation analysis of Next Generation Sequencing data.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/MethTargetedNGS.html
Versions      1.10.0, 1.12.0, 1.8.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-methtargetedngs



Links         biotools: :biotools:`methtargetedngs`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-methtargetedngs

and update with::

   conda update bioconductor-methtargetedngs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-methtargetedngs.
