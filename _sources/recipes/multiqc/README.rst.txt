Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install multiqc

and update with::

   conda update multiqc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/multiqc.
