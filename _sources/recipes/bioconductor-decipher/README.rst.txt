bioconductor-decipher
=====================

|downloads|

A toolset for deciphering and managing biological sequences.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/DECIPHER.html
Versions      2.8.1, 2.6.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-decipher



Links         biotools: :biotools:`DECIPHER`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-decipher

and update with::

   conda update bioconductor-decipher



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-decipher.
