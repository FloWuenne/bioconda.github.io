genomepy
========

|downloads|

Download genomes the easy way.

============= ===========
Home          https://github.com/simonvh/genomepy
Versions      0.3.1, 0.4.0, 0.5.0
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/genomepy



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install genomepy

and update with::

   conda update genomepy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/genomepy.
