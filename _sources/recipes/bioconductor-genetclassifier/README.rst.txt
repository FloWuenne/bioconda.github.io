.. _`bioconductor-genetclassifier`:

bioconductor-genetclassifier
============================

|downloads|

Comprehensive package to automatically train and validate a multi\-class SVM classifier based on gene expression data. Provides transparent selection of gene markers\, their coexpression networks\, and an interface to query the classifier.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/geNetClassifier.html
Versions      1.20.0, 1.18.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-genetclassifier



Links         biotools: :biotools:`genetclassifier`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genetclassifier

and update with::

   conda update bioconductor-genetclassifier



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genetclassifier.
