bioconductor-snapcgh
====================

|downloads|

Methods for segmenting\, normalising and processing aCGH data\; including plotting functions for visualising raw and segmented data for individual and multiple arrays.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/snapCGH.html
Versions      
License       GPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-snapcgh



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-snapcgh

and update with::

   conda update bioconductor-snapcgh



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-snapcgh.
