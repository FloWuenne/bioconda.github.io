.. _`clinvar-tsv`:

clinvar-tsv
===========

|downloads|

A Snakemake\-based program to download ClinVar and convert to easy\-to\-use TSV files.

============= ===========
Home          https://github.com/bihealth/clinvar-tsv
Versions      0.1.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/clinvar-tsv



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install clinvar-tsv

and update with::

   conda update clinvar-tsv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/clinvar-tsv.
