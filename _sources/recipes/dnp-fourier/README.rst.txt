Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dnp-fourier

and update with::

   conda update dnp-fourier

A Docker container is available at https://quay.io/repository/biocontainers/dnp-fourier.
