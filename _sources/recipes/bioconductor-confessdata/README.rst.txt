bioconductor-confessdata
========================

|downloads|

Example text\-converted C01 image files for use in the CONFESS Bioconductor package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/CONFESSdata.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-confessdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-confessdata

and update with::

   conda update bioconductor-confessdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-confessdata.
