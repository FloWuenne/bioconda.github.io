bioconductor-gatingmldata
=========================

|downloads|

Test data and XML files for testing compliance of the flowUtils\/flowCore packages with Gating\-ML \(1.5 and 2.0\) standards.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/gatingMLData.html
Versions      2.22.0
License       GPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-gatingmldata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-gatingmldata

and update with::

   conda update bioconductor-gatingmldata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-gatingmldata.
