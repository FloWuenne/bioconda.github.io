License  Artistic-2.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-lumihumanall.db

and update with::

   conda update bioconductor-lumihumanall.db
