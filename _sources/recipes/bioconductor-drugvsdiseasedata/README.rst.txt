bioconductor-drugvsdiseasedata
==============================

|downloads|

Data package which provides default disease expression profiles\, clusters and annotation information for use with the DrugVsDisease package.

============= ===========
Home          http://bioconductor.org/packages/3.7/data/experiment/html/DrugVsDiseasedata.html
Versions      1.16.0, 1.14.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-drugvsdiseasedata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-drugvsdiseasedata

and update with::

   conda update bioconductor-drugvsdiseasedata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-drugvsdiseasedata.
