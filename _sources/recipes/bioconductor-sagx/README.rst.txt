bioconductor-sagx
=================

|downloads|

A package for retrieval\, preparation and analysis of data from the Affymetrix GeneChip. In particular the issue of identifying differentially expressed genes is addressed.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/SAGx.html
Versions      1.54.0, 1.52.0, 1.50.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-sagx



Links         biotools: :biotools:`sagx`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-sagx

and update with::

   conda update bioconductor-sagx



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-sagx.
