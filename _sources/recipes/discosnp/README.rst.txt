Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install discosnp

and update with::

   conda update discosnp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/discosnp.
