.. _`bioconductor-igc`:

bioconductor-igc
================

|downloads|

This package is intended to identify differentially expressed genes driven by Copy Number Alterations from samples with both gene expression and CNA data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/iGC.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-igc/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-igc

and update with::

   conda update bioconductor-igc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-igc.
