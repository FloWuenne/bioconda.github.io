dcmtk
=====

|downloads|

DCMTK is a collection of libraries and applications implementing large parts the DICOM standard

============= ===========
Home          http://dicom.offis.de
Versions      3.6.1
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/dcmtk/meta.yaml

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dcmtk

and update with::

   conda update dcmtk
