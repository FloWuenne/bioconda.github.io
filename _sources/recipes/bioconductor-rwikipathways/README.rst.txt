==========================

Use this package to interface with the WikiPathways API.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rwikipathways

and update with::

   conda update bioconductor-rwikipathways

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rwikipathways.
