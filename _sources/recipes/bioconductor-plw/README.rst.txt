Probe level Locally moderated Weighted median\-t \(PLW\) and Locally Moderated Weighted\-t \(LMW\).

Versions      1.36.0, 1.38.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-plw

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-plw

and update with::

   conda update bioconductor-plw

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-plw.
