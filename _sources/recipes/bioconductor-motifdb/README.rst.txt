bioconductor-motifdb
====================

|downloads|

More than 8000 annotated position frequency matrices from 13 public sources\, for multiple organisms.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/MotifDb.html
Versions      1.16.0, 1.18.0, 1.20.0, 1.22.0
License       Artistic-2.0 | file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-motifdb



Links         biotools: :biotools:`motifdb`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-motifdb

and update with::

   conda update bioconductor-motifdb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-motifdb.
