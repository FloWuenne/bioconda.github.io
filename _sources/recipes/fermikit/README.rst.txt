fermikit
========

|downloads|

FermiKit is a de novo assembly based variant calling pipeline for deep Illumina resequencing data\.

============= ===========
Home          https://github.com/lh3/fermikit
Versions      0.14.dev1
License       Unknown
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/fermikit



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fermikit

and update with::

   conda update fermikit



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/fermikit.
