meta-sparse
===========

|downloads|

SPARSE indexes reference genomes in public databases into hierarchical clusters and uses it to predict origins of metagenomic reads.

============= ===========
Home          https://github.com/zheminzhou/SPARSE/
Versions      0.1.12, 0.1.11, 0.1.2
License       GNU General Public License v3 (GPLv3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/meta-sparse



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install meta-sparse

and update with::

   conda update meta-sparse



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/meta-sparse.
