bioconductor-refplus
====================

|downloads|

The package contains functions for pre\-processing Affymetrix data using the RMA\+ and the RMA\+\+ methods\.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/RefPlus.html
Versions      1.48.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-refplus



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-refplus

and update with::

   conda update bioconductor-refplus



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-refplus.
