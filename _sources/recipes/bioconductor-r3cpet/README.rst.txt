bioconductor-r3cpet
===================

|downloads|

The package provides a method to infer the set of proteins that are more probably to work together to maintain chormatin interaction given a ChIA\-PET experiment results.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/R3CPET.html
Versions      
License       GPL (>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-r3cpet



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-r3cpet

and update with::

   conda update bioconductor-r3cpet



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-r3cpet.
