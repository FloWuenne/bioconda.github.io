bioconductor-beachmat
=====================

|downloads|

Provides a consistent C\+\+ class interface for a variety of commonly used matrix types\, including sparse and HDF5\-backed matrices.

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/beachmat.html
Versions      1.2.1, 1.0.1
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-beachmat



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-beachmat

and update with::

   conda update bioconductor-beachmat



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-beachmat.
