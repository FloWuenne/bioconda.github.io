bioconductor-treeio
===================

|downloads|

Base classes and functions for parsing and exporting phylogenetic trees.

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/treeio.html
Versions 1.0.2
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-treeio
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-treeio

and update with::

   conda update bioconductor-treeio



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-treeio.
