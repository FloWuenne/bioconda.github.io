bioconductor-h10kcod.db
=======================

|downloads|

Codelink UniSet Human I Bioarray \(\~10 000 human genes\) annotation data \(chip h10kcod\) assembled using data from public repositories

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/h10kcod.db.html
Versions      3.4.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-h10kcod.db/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-h10kcod.db

and update with::

   conda update bioconductor-h10kcod.db



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-h10kcod.db.
