.. _`bioconductor-geuvadistranscriptexpr`:

bioconductor-geuvadistranscriptexpr
===================================

|downloads|

Provides transcript expression and bi\-allelic genotypes corresponding to the chromosome 19 for CEU individuals from the GEUVADIS project\, Lappalainen et al.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/GeuvadisTranscriptExpr.html
Versions      
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-geuvadistranscriptexpr



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-geuvadistranscriptexpr

and update with::

   conda update bioconductor-geuvadistranscriptexpr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-geuvadistranscriptexpr.
