bioconductor-customprodb
========================

|downloads|

Generate customized protein sequence database from RNA\-Seq data for proteomics search

============= ===========
Home          http://bioconductor.org/packages/release/bioc/html/customProDB.html
Versions      1.22.0, 1.14.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//customprodb/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-customprodb

and update with::

   conda update bioconductor-customprodb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-customprodb.
