.. _`htseq`:

htseq
=====

|downloads|

HTSeq is a Python library to facilitate processing and analysis of data from high\-throughput sequencing \(HTS\) experiments\.

============= ===========
Home          https://github.com/simon-anders/htseq
Versions      0.6.1, 0.6.1.post1, 0.6.1p1, 0.7.2, 0.9.1
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/htseq



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install htseq

and update with::

   conda update htseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/htseq.
