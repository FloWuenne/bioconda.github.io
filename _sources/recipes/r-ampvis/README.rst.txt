A package to visualise amplicon data

============= ===========

Versions      1.27.0
License       AGPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-ampvis



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-ampvis

and update with::

   conda update r-ampvis

A Docker container is available at https://quay.io/repository/biocontainers/r-ampvis.
