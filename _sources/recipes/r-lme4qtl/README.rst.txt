r-lme4qtl
=========

|downloads|

Linear mixed models \(lme4\) with flexible covariance structure for qtl and association analysis.

============= ===========
Home          https://CRAN.R-project.org/package=lme4qtl
Versions      0.1.10
License       GPL (>= 3)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-lme4qtl



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-lme4qtl

and update with::

   conda update r-lme4qtl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-lme4qtl.
