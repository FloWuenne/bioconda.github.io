graphaligner
============

Sequence to graph aligner for long reads

============= ===========
Home          https://github.com/maickrau/GraphAligner
Versions      1.0.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/graphaligner



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install graphaligner

and update with::

   conda update graphaligner

A Docker container is available at https://quay.io/repository/biocontainers/graphaligner.
