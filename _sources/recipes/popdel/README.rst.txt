popdel
======

|downloads|

Fast structural deletion calling on population\-scale short read paired\-end data.

============= ===========
Home          https://github.com/kehrlab/PopDel
Versions      1.0.1
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/popdel



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install popdel

and update with::

   conda update popdel



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/popdel.
