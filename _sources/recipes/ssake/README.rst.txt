ssake
=====

|downloads|

SSAKE is a genomics application for de novo assembly of millions of very short DNA sequences.

Home          https://github.com/warrenlr/SSAKE
Versions      4.0
License       GPL-2.0

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ssake

and update with::

   conda update ssake
