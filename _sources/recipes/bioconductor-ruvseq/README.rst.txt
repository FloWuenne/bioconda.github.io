bioconductor-ruvseq
===================

|downloads|

This package implements the remove unwanted variation \(RUV\) methods of Risso et al. \(2014\) for the normalization of RNA\-Seq read counts between samples.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/RUVSeq.html
Versions      1.14.0, 1.12.0, 1.10.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/bioconductor-ruvseq/meta.yaml



Links         biotools: :biotools:`ruvseq`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ruvseq

and update with::

   conda update bioconductor-ruvseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ruvseq.
