bioconductor-breakpointr
========================

|downloads|

This package implements functions for finding breakpoints\, plotting and export of Strand\-seq data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/breakpointR.html
Versions      1.0.0
License       file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-breakpointr/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-breakpointr

and update with::

   conda update bioconductor-breakpointr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-breakpointr.
