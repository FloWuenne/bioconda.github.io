bioconductor-bsgenome.rnorvegicus.ucsc.rn5
==========================================

|downloads|

Full genome sequences for Rattus norvegicus \(Rat\) as provided by UCSC \(rn5\, Mar. 2012\) and stored in Biostrings objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Rnorvegicus.UCSC.rn5.html
Versions      1.4.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-bsgenome.rnorvegicus.ucsc.rn5/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.rnorvegicus.ucsc.rn5

and update with::

   conda update bioconductor-bsgenome.rnorvegicus.ucsc.rn5



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.rnorvegicus.ucsc.rn5.
