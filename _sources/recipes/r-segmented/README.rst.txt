r-segmented
===========

|downloads|

Given a regression model, segmented `updates' the model by adding one or more segmented (i.e., piecewise-linear) relationships. Several variables with multiple breakpoints are allowed.

======== ===========
Home     https://cran.r-project.org/web/packages/segmented/
Versions 0.5_1.4
License  GPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-segmented
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-segmented

and update with::

   conda update r-segmented



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-segmented.
