bioconductor-chimeraviz
=======================

|downloads|

chimeraviz manages data from fusion gene finders and provides useful visualization tools.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/chimeraviz.html
Versions      1.6.2, 1.0.4
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-chimeraviz



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-chimeraviz

and update with::

   conda update bioconductor-chimeraviz



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-chimeraviz.
