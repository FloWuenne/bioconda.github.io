.. _`bioconductor-mirbaseconverter`:

bioconductor-mirbaseconverter
=============================

|downloads|

A comprehensive tool for converting and retrieving the miRNA Name\, Accession\, Sequence\, Version\, History and Family information in different miRBase versions. It can process a huge number of miRNAs in a short time without other depends.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/miRBaseConverter.html
Versions      1.6.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-mirbaseconverter/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mirbaseconverter

and update with::

   conda update bioconductor-mirbaseconverter



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mirbaseconverter.
