.. _`ddrage`:

ddrage
======

|downloads|

Simulator for ddRADseq \(double digest restriction site associated DNA sequencing\) datasets. Generates reads \(FASTQ format\) that can be analyzed and validated using a ground truth file \(YAML\).

============= ===========
Home          https://bitbucket.org/genomeinformatics/rage
Versions      1.1.4, 1.2.2, 1.3.0, 1.3.1, 1.4.0, 1.5.1, 1.5.2, 1.6.1
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ddrage



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ddrage

and update with::

   conda update ddrage



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ddrage.
