.. _`r-qpcr`:

r-qpcr
======

|downloads|

Model fitting\, optimal model selection and calculation of various features that are essential in the analysis of quantitative real\-time polymerase chain reaction \(qPCR\).

============= ===========
Home          https://CRAN.R-project.org/package=qpcR
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//r-qpcr/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-qpcr

and update with::

   conda update r-qpcr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-qpcr.
