.. _`comparems2`:

comparems2
==========

|downloads|

A simple tool developed to compare\, globally\, all MS\/MS spectra between two datasets \(in Mascot Generic Format or MGF\).

============= ===========
Home          http://www.ms-utils.org/compareMS2.html
Versions      1
License       GPL3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//comparems2/meta.yaml



Links         biotools: :biotools:`comparems2`, doi: :doi:`10.1002/rcm.6162`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install comparems2

and update with::

   conda update comparems2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/comparems2.
