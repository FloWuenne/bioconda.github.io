extract-sv-reads
================

|downloads|

Tool for extracting splitter or discordant reads from a BAM or CRAM file.

============= ===========
Home          https://github.com/hall-lab/extract_sv_reads
Versions      1.3.0, 1.2.1, 1.1.2, 1.1.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/extract-sv-reads/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install extract-sv-reads

and update with::

   conda update extract-sv-reads



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/extract-sv-reads.
