bioconductor-multimed
=====================

|downloads|

Implements methods for testing multiple mediators

============= ===========
Home          http://bioconductor.org/packages/3.7/bioc/html/MultiMed.html
Versions      2.2.0, 1.12.0, 1.10.0
License       GPL (>= 2) + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-multimed

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-multimed

and update with::

   conda update bioconductor-multimed



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-multimed.
