bioconductor-citruscdf
======================

|downloads|

A package containing an environment representing the Citrus.cdf file.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/citruscdf.html
Versions      
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-citruscdf



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-citruscdf

and update with::

   conda update bioconductor-citruscdf



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-citruscdf.
