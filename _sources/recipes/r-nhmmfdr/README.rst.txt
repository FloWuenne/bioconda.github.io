.. _`r-nhmmfdr`:

r-nhmmfdr
=========

|downloads|

The NHMMfdr package implements the non-homogeneous Hidden Markov Model based FDR control as described in Kuan et al., 2011 for multiple comparison adjustment under dependence. It allows for prior information to be incorporated in the model to improve detection of significant tests.

======== ===========
Home     http://www.ams.sunysb.edu/~pfkuan/NHMMfdr/
Versions 1.0.6
License  GPL (>= 2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-nhmmfdr
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-nhmmfdr

and update with::

   conda update r-nhmmfdr

A Docker container is available at https://quay.io/repository/biocontainers/r-nhmmfdr.
