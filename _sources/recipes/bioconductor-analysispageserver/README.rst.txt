bioconductor-analysispageserver
===============================

|downloads|

AnalysisPageServer is a modular system that enables sharing of customizable R analyses via the web.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/AnalysisPageServer.html
Versions      1.16.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-analysispageserver/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-analysispageserver

and update with::

   conda update bioconductor-analysispageserver



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-analysispageserver.
