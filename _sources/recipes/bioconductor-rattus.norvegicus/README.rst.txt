bioconductor-rattus.norvegicus
==============================

|downloads|

Contains the Rattus.norvegicus object to access data from several related annotation packages.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/Rattus.norvegicus.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rattus.norvegicus



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rattus.norvegicus

and update with::

   conda update bioconductor-rattus.norvegicus



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rattus.norvegicus.
