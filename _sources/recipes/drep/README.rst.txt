drep
====

|downloads|

De\-replication of microbial genomes assembled from multiple samples

============= ===========
Home          https://github.com/MrOlm/drep
Versions      2.0.5, 2.2.3
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/drep



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install drep

and update with::

   conda update drep



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/drep.
