bioconductor-apeglm
===================

|downloads|

apeglm provides Bayesian shrinkage estimators for effect sizes for a variety of GLM models\, using approximation of the posterior for individual coefficients.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/apeglm.html
Versions      1.2.1, 1.0.3
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-apeglm



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-apeglm

and update with::

   conda update bioconductor-apeglm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-apeglm.
