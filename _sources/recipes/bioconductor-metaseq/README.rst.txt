bioconductor-metaseq
====================

|downloads|

The probabilities by one-sided NOISeq are combined by Fisher's method or Stouffer's method

======== ===========
Home     http://bioconductor.org/packages/3.5/bioc/html/metaSeq.html
Versions 1.16.0
License  Artistic-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-metaseq
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-metaseq

and update with::

   conda update bioconductor-metaseq



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-metaseq.
