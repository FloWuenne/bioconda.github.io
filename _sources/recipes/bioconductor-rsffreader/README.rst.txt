bioconductor-rsffreader
=======================

|downloads|

rSFFreader reads sequence\, qualities and clip point values from sff files generated by Roche 454 and Life Sciences Ion Torrent sequencers into similar classes as are present for fastq files.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/rSFFreader.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-rsffreader/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rsffreader

and update with::

   conda update bioconductor-rsffreader



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rsffreader.
