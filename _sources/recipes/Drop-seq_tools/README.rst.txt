dropseq_tools
=============

|downloads|

Package for the analysis of Drop\-seq data developed by Jim Nemesh in the McCarroll Lab


============= ===========
Home          http://mccarrolllab.com/dropseq/
Versions      1.13, 2.0.0
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/Drop-seq_tools



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install dropseq_tools

and update with::

   conda update dropseq_tools


Notes
-----
Drop\-seq\_tools utilities are wrapper shell scripts. To get help on individual tool\, use e.g. \`PolyATrimmer \-\- \-\-help\`



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/dropseq_tools.
