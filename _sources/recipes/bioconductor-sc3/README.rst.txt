Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-sc3

and update with::

   conda update bioconductor-sc3

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-sc3.
