bioconductor-erccdashboard
==========================

|downloads|

Technical performance metrics for differential gene expression experiments using External RNA Controls Consortium \(ERCC\) spike\-in ratio mixtures.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/erccdashboard.html
Versions      1.16.0
License       GPL (>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-erccdashboard/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-erccdashboard

and update with::

   conda update bioconductor-erccdashboard



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-erccdashboard.
