segmentation-fold

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install segmentation-fold

and update with::

   conda update segmentation-fold



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/segmentation-fold.
