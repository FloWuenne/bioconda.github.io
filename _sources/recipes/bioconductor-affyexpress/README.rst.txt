.. _`bioconductor-affyexpress`:

bioconductor-affyexpress
========================

|downloads|

The purpose of this package is to provide a comprehensive and easy\-to\-use tool for quality assessment and to identify differentially expressed genes in the Affymetrix gene expression data\.

======== ===========
Home     http://bioconductor.org/packages/3.6/bioc/html/AffyExpress.html
Versions 1.42.0, 1.44.0
License  LGPL
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-affyexpress

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-affyexpress

and update with::

   conda update bioconductor-affyexpress



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-affyexpress.
