binsanity
=========

|downloads|

Method to cluster contigs based a biphasic method with coverage and composition

============= ===========
Home          https://github.com/edgraham/BinSanity
Versions      0.2.6.1, 0.2.6.3
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/binsanity



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install binsanity

and update with::

   conda update binsanity



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/binsanity.
