bioconductor-tomatocdf
======================

|downloads|

A package containing an environment representing the Tomato.cdf file.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/tomatocdf.html
Versions      
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-tomatocdf



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-tomatocdf

and update with::

   conda update bioconductor-tomatocdf



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-tomatocdf.
