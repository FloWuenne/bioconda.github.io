Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install iqtree

and update with::

   conda update iqtree



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/iqtree.
