Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-txdb.mmusculus.ucsc.mm9.knowngene

and update with::

   conda update bioconductor-txdb.mmusculus.ucsc.mm9.knowngene
