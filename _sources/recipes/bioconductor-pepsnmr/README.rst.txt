bioconductor-pepsnmr
====================

|downloads|

This package provides R functions for common pre\-procssing steps that are applied on 1H\-NMR data. It also provides a function to read the FID signals directly in the Bruker format.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/PepsNMR.html
Versions      1.0.1
License       GPL-2 | file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-pepsnmr/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-pepsnmr

and update with::

   conda update bioconductor-pepsnmr

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-pepsnmr.
