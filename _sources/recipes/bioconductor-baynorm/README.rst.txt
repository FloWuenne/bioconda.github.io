bioconductor-baynorm
====================

|downloads|

bayNorm is used for normalizing single\-cell RNA\-seq data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/bayNorm.html
Versions      1.0.6
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-baynorm/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-baynorm

and update with::

   conda update bioconductor-baynorm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-baynorm.
