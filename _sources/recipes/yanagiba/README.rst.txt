yanagiba
========

|downloads|

Filter short or low quality Oxford Nanopore reads which have been basecalled with Albacore.

============= ===========
Home          https://github.com/Adamtaranto/Yanagiba
Versions      1.0.0, 0.1.0
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/yanagiba



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install yanagiba

and update with::

   conda update yanagiba



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/yanagiba.
