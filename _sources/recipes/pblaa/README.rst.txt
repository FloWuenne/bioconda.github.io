pblaa
=====

|downloads|

PacBio tool to deconvolute mixtures of alleles and loci into phased consensus sequences.

============= ===========
Home          https://github.com/PacificBiosciences/pbbioconda
Versions      2.4.2
License       BSD-3-Clause-Clear
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//pblaa/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pblaa

and update with::

   conda update pblaa



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pblaa.
