bioconductor-cghmcr
===================

|downloads|

This package provides functions to identify genomic regions of interests based on segmented copy number data from multiple samples.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/cghMCR.html
Versions      1.38.0, 1.36.0, 1.34.0
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/bioconductor-cghmcr/meta.yaml



Links         biotools: :biotools:`cghmcr`, doi: :doi:`10.1093/bioinformatics/btv298`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cghmcr

and update with::

   conda update bioconductor-cghmcr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cghmcr.
