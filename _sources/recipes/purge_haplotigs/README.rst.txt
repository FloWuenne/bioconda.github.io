purge_haplotigs
===============

|downloads|

Pipeline to help with curating heterozygous diploid genome assemblies.

============= ===========
Home          https://bitbucket.org/mroachawri/purge_haplotigs/
Versions      1.0.1
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/purge_haplotigs



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install purge_haplotigs

and update with::

   conda update purge_haplotigs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/purge_haplotigs.
