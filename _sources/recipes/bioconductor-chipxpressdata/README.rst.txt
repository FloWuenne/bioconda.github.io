.. _`bioconductor-chipxpressdata`:

bioconductor-chipxpressdata
===========================

|downloads|

Contains pre\-built mouse \(GPL1261\) and human \(GPL570\) database of gene expression profiles to be used for ChIPXpress ranking\.

======== ===========
Home     http://bioconductor.org/packages/3.6/data/experiment/html/ChIPXpressData.html
Versions 1.16.0
License  GPL (>=2)
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-chipxpressdata

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-chipxpressdata

and update with::

   conda update bioconductor-chipxpressdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-chipxpressdata.
