bioconductor-mutationalpatterns
===============================

|downloads|

R package for extracting and visualizing mutational patterns in base substitution catalogues

============= ===========
Home          http://bioconductor.org/packages/release/bioc/html/MutationalPatterns.html
Versions      1.4.1, 1.4.3
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-mutationalpatterns



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mutationalpatterns

and update with::

   conda update bioconductor-mutationalpatterns



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mutationalpatterns.
