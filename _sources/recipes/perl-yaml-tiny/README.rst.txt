perl-yaml-tiny
==============

|downloads|

Read\/Write YAML files with as little code as possible

============= ===========
Home          http://search.cpan.org/dist/YAML-Tiny/lib/YAML/Tiny.pm
Versions      1.70, 1.73
License       perl
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-yaml-tiny



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install perl-yaml-tiny

and update with::

   conda update perl-yaml-tiny



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/perl-yaml-tiny.
