r-pack
======

|downloads|

Functions to easily convert data to binary formats other programs/machines can understand.

======== ===========
Home     https://CRAN.R-project.org/package=pack
Versions 0.1_1
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-pack
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-pack

and update with::

   conda update r-pack



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-pack.
