bioconductor-grndata
====================

|downloads|

Simulated expression data for five large Gene Regulatory Networks from different simulators

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/grndata.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-grndata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-grndata

and update with::

   conda update bioconductor-grndata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-grndata.
