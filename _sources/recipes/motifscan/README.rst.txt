motifscan
=========

|downloads|

A motif discovery tool to search for candidate targets of given motifs.

Home          https://github.com/shao-lab/MotifScan
Versions      1.1.2, 1.1
License       BSD License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/motifscan/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install motifscan

and update with::

   conda update motifscan



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/motifscan.
