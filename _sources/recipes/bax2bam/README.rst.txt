.. _`bax2bam`:

bax2bam
=======

|downloads|

bax2bam converts the legacy PacBio basecall format \(bax.h5\) into the BAM basecall format

============= ===========
Home          https://github.com/PacificBiosciences/bax2bam
Versions      0.0.9
License       BSD
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bax2bam



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bax2bam

and update with::

   conda update bax2bam



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bax2bam.
