bioconductor-mogsa
==================

|downloads|

This package provide a method for doing gene set analysis based on multiple omics data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/mogsa.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-mogsa/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mogsa

and update with::

   conda update bioconductor-mogsa



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-mogsa.
