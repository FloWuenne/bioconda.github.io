mqc

qualtiy control tool to assess the mapping quality of a ribosome profiling experiment

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mqc

and update with::

   conda update mqc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mqc.
