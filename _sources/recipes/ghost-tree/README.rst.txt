ghost-tree
==========

|downloads|

ghost\-tree is a bioinformatics tool that combines sequence data from two
genetic marker databases into one phylogenetic tree that can be used for
diversity analyses.


============= ===========
Home          https://github.com/JTFouquier/ghost-tree
Versions      0.2.2
License       BSD license
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/ghost-tree



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ghost-tree

and update with::

   conda update ghost-tree



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ghost-tree.
