Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.osativa.msu.msu7

and update with::

   conda update bioconductor-bsgenome.osativa.msu.msu7
