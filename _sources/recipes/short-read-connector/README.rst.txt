shortreadsconnector
===================

Short read connector enables the comparisons of two read sets

Versions 1.1.0, 1.1.3
License  GNU Affero General Public License v3.0

======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install shortreadsconnector

and update with::

   conda update shortreadsconnector



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/shortreadsconnector.
