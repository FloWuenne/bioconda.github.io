Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install salmon

and update with::

   conda update salmon



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/salmon.
