bioconductor-ceu1kg
===================

|downloads|

CEU genotypes from 1000 genomes \"low coverage\" pilot phase I \(approx 8 million SNP calls released July 2010\)\; includes Wellcome trust GENEVAR expression for 43 indiv

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/ceu1kg.html
Versions      0.20.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-ceu1kg/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ceu1kg

and update with::

   conda update bioconductor-ceu1kg



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ceu1kg.
