Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install genomicconsensus

and update with::

   conda update genomicconsensus

A Docker container is available at https://quay.io/repository/biocontainers/genomicconsensus.
