bioconductor-rrdpdata
=====================

|downloads|

Database used by the default RDP Classifier

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/rRDPData.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-rrdpdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-rrdpdata

and update with::

   conda update bioconductor-rrdpdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-rrdpdata.
