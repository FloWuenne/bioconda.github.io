strelka
=======

|downloads|

Strelka calls somatic and germline small variants from mapped sequencing reads

======== ===========
Home     https://github.com/Illumina/strelka
Versions 2.7.1, 2.8.2, 2.8.4
License  GPLv3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/strelka

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install strelka

and update with::

   conda update strelka



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/strelka.
