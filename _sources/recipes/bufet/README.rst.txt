.. _`bufet`:

bufet
=====

|downloads|

Tool that performs the unbiased miRNA functional enrichment analysis \(Bleazard et al.\) requiring significantly reduced excution times \(less than 10 minutes for 1 million iterations\).

Home          https://github.com/diwis/BUFET/

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bufet

and update with::

   conda update bufet



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bufet.
