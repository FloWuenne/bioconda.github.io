glimmerhmm
==========

|downloads|

GlimmerHMM is a gene finder based on a Generalized Hidden Markov Model \(GHMM\)

============= ===========
Home          https://ccb.jhu.edu/software/glimmerhmm/
Versions      3.0.4
License       Artistic License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/glimmerHMM



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install glimmerhmm

and update with::

   conda update glimmerhmm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/glimmerhmm.
