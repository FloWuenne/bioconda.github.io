bioconductor-varianttoolsdata
=============================

|downloads|

Data from the sequencing of a 50\/50 mixture of HapMap trio samples NA12878 \(CEU\) and NA19240 \(YRI\)\, subset to the TP53 region.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/VariantToolsData.html
Versions      1.6.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-varianttoolsdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-varianttoolsdata

and update with::

   conda update bioconductor-varianttoolsdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-varianttoolsdata.
