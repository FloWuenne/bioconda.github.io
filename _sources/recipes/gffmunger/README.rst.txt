.. _`gffmunger`:

gffmunger
=========

|downloads|

Munges GFF3 files exported from Chado database to make them suitable for loading into WebApollo

============= ===========
Home          https://github.com/sanger-pathogens/gffmunger
Versions      0.0.1
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//gffmunger/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install gffmunger

and update with::

   conda update gffmunger



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/gffmunger.
