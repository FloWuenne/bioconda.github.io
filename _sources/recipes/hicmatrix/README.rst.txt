hicmatrix
=========

|downloads|

Library to manage Hi\-C matrices for HiCExplorer and pyGenomeTracks

============= ===========
Home          https://github.com/deeptools/HiCMatrix
Versions      2.2, 3, 4
License       GPL3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/hicmatrix



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install hicmatrix

and update with::

   conda update hicmatrix



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/hicmatrix.
