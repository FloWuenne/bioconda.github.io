bioformats
==========

|downloads|

A collection of Python classes to handle bioinformatics data.

============= ===========
Home          https://github.com/gtamazian/bioformats
Versions      0.1.15
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioformats



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioformats

and update with::

   conda update bioformats
