kallisto

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install kallisto

and update with::

   conda update kallisto



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/kallisto.
