bioconductor-blma
=================

|downloads|

Suit of tools for bi\-level meta\-analysis. The package can be used in a wide range of applications\, including general hypothesis testings\, differential expression analysis\, functional analysis\, and pathway analysis.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/BLMA.html
Versions      1.6.0
License       GPL (>=2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-blma/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-blma

and update with::

   conda update bioconductor-blma



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-blma.
