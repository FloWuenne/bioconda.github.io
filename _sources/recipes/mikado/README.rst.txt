mikado

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mikado

and update with::

   conda update mikado



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/mikado.
