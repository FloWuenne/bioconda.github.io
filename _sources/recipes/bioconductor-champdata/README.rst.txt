.. _`bioconductor-champdata`:

bioconductor-champdata
======================

|downloads|

Provides datasets needed for ChAMP including a test dataset and blood controls for CNA analysis.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/ChAMPdata.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-champdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-champdata

and update with::

   conda update bioconductor-champdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-champdata.
