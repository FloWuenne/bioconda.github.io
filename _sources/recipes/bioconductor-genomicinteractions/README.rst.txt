bioconductor-genomicinteractions
================================

|downloads|

R package for handling Genomic interaction data\, such as ChIA\-PET\/Hi\-C\, annotating genomic features with interaction information and producing various plots \/ statistics.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/GenomicInteractions.html
Versions      1.10.0, 1.12.0, 1.14.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-genomicinteractions



Links         biotools: :biotools:`genomicinteractions`, doi: :doi:`10.1186/s12864-015-2140-x`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genomicinteractions

and update with::

   conda update bioconductor-genomicinteractions



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genomicinteractions.
