abeona

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install abeona

and update with::

   conda update abeona



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/abeona.
