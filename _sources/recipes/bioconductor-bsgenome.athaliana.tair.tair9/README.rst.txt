bioconductor-bsgenome.athaliana.tair.tair9
==========================================

|downloads|

Full genome sequences for Arabidopsis thaliana as provided by TAIR \(TAIR9 Genome Release\) and stored in Biostrings objects. Note that TAIR10 is an \"annotation release\" based on the same genome assembly as TAIR9.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Athaliana.TAIR.TAIR9.html
Versions      1.3.1000
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-bsgenome.athaliana.tair.tair9/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.athaliana.tair.tair9

and update with::

   conda update bioconductor-bsgenome.athaliana.tair.tair9



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.athaliana.tair.tair9.
