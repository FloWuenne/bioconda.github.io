fasten
======

|downloads|

Perform random operations on fastq files\, using unix streaming. Secure your analysis with Fasten\!

============= ===========
Home          https://github.com/lskatz/fasten
Versions      0.1.13
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/fasten



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fasten

and update with::

   conda update fasten



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/fasten.
