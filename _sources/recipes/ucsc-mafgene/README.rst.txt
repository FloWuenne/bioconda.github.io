Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ucsc-mafgene

and update with::

   conda update ucsc-mafgene



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ucsc-mafgene.
