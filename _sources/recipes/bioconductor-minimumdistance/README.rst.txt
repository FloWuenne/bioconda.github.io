Analysis of de novo copy number variants in trios from high\-dimensional genotyping platforms.

============= ===========

Versions      1.22.0, 1.24.1
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-minimumdistance

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-minimumdistance

and update with::

   conda update bioconductor-minimumdistance



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-minimumdistance.
