Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-a4preproc

and update with::

   conda update bioconductor-a4preproc



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-a4preproc.
