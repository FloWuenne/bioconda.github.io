bioconductor-harmonizedtcgadata
===============================

|downloads|

This package contains the processed harmonized TCGA data of five cancer types used in \"Tianle Ma and Aidong Zhang\, Integrate Multi\-omic Data Using Affinity Network Fusion \(ANF\) for Cancer Patient Clustering\".

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/HarmonizedTCGAData.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-harmonizedtcgadata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-harmonizedtcgadata

and update with::

   conda update bioconductor-harmonizedtcgadata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-harmonizedtcgadata.
