bioconductor-cnvtools
=====================

|downloads|

This package is meant to facilitate the testing of Copy Number Variant data for genetic association\, typically in case\-control studies.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/CNVtools.html
Versions      1.74.0, 1.72.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-cnvtools/meta.yaml



Links         biotools: :biotools:`cnvtools`, doi: :doi:`10.1038/ng.206`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-cnvtools

and update with::

   conda update bioconductor-cnvtools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-cnvtools.
