.. _`bioconductor-keggorthology`:

bioconductor-keggorthology
==========================

|downloads|

graphical representation of the Feb 2010 KEGG Orthology\. The KEGG orthology is a set of pathway IDs that are not to be confused with the KEGG ortholog IDs\.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/keggorthology.html
Versions      2.30.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-keggorthology



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-keggorthology

and update with::

   conda update bioconductor-keggorthology



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-keggorthology.
