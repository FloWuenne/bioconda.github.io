Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install askocli

and update with::

   conda update askocli



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/askocli.
