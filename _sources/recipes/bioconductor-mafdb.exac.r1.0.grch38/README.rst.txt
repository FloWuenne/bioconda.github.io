Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-mafdb.exac.r1.0.grch38

   conda update bioconductor-mafdb.exac.r1.0.grch38
