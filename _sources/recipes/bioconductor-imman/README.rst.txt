bioconductor-imman
==================

|downloads|

Reconstructing Interlog Protein Network \(IPN\) integrated from several Protein protein Interaction Networks \(PPINs\). Using this package\, overlaying different PPINs to mine conserved common networks between diverse species will be applicable.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/IMMAN.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-imman/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-imman

and update with::

   conda update bioconductor-imman



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-imman.
