ectyper
=======

|downloads|

ECTyper is a python program for serotyping E. coli genomes

============= ===========
Home          https://github.com/phac-nml/ecoli_serotyping
Versions      0.8.1, 0.5.4, 0.1
License       Apache 2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//ectyper/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install ectyper

and update with::

   conda update ectyper



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/ectyper.
