bioconductor-genestructuretools
===============================

|downloads|

GeneStructureTools can be used to create in silico alternative splicing events\, and analyse potential effects this has on functional gene products.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/GeneStructureTools.html
Versions      
License       BSD_3_clause + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-genestructuretools/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-genestructuretools

and update with::

   conda update bioconductor-genestructuretools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-genestructuretools.
