bioconductor-difflogo
=====================

|downloads|

DiffLogo is an easy\-to\-use tool to visualize motif differences.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/DiffLogo.html
Versions      
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-difflogo



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-difflogo

and update with::

   conda update bioconductor-difflogo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-difflogo.
