.. _`bioconductor-gatefinder`:

bioconductor-gatefinder
=======================

|downloads|

Given a vector of cluster memberships for a cell population\, identifies a sequence of gates \(polygon filters on 2D scatter plots\) for isolation of that cell type.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/GateFinder.html
Versions      1.2.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-gatefinder/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-gatefinder

and update with::

   conda update bioconductor-gatefinder



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-gatefinder.
