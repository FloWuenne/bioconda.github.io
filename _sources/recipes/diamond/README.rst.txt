Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install diamond

and update with::

   conda update diamond



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/diamond.
