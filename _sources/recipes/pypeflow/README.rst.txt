Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pypeflow

and update with::

   conda update pypeflow



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pypeflow.
