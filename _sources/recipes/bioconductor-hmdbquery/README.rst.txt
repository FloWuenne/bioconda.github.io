bioconductor-hmdbquery
======================

|downloads|

Define utilities for exploration of human metabolome database\, including functions to retrieve specific metabolite entries and data snapshots with pairwise associations \(metabolite\-gene\,\-protein\,\-disease\).

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/hmdbQuery.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-hmdbquery/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-hmdbquery

and update with::

   conda update bioconductor-hmdbquery



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-hmdbquery.
