bioconductor-bsgenome.dmelanogaster.ucsc.dm6
============================================

|downloads|

Full genome sequences for Drosophila melanogaster \(Fly\) as provided by UCSC \(dm6\, Aug. 2014\) and stored in Biostrings objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Dmelanogaster.UCSC.dm6.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-bsgenome.dmelanogaster.ucsc.dm6/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.dmelanogaster.ucsc.dm6

and update with::

   conda update bioconductor-bsgenome.dmelanogaster.ucsc.dm6



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.dmelanogaster.ucsc.dm6.
