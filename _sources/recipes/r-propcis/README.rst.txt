r-propcis
=========

|downloads|

Computes two\-sample confidence intervals for single\, paired and independent proportions.

============= ===========
Home          https://github.com/shearer/PropCIs
Versions      0.3_0
License       GPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-propcis



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-propcis

and update with::

   conda update r-propcis



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-propcis.
