.. _`bioconductor-sispa`:

bioconductor-sispa
==================

|downloads|

Sample Integrated Set Profile Analysis \(SISPA\) is a method designed to define sample groups with similar gene set enrichment profiles.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/SISPA.html
Versions      1.10.0, 1.8.0, 1.6.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-sispa



Links         biotools: :biotools:`sispa`, doi: :doi:`10.1093/nar/gkv1503`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-sispa

and update with::

   conda update bioconductor-sispa



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-sispa.
