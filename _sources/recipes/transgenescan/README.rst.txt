transgenescan
=============

|downloads|

Software tool for finding genes in assembled transcripts from metatranscriptomic sequences.

============= ===========
Home          https://github.com/COL-IU/TransGeneScan
Versions      1.2.1
License       file
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/transgenescan



Links         biotools: :biotools:`TransGeneScan`, doi: :doi:`10.1186/1471-2105-15-S9-S8`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install transgenescan

and update with::

   conda update transgenescan



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/transgenescan.
