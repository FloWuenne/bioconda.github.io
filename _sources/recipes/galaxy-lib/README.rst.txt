galaxy-lib
==========

|downloads|

Subset of Galaxy \(http\:\/\/galaxyproject.org\/\) core code base designed to be used a library.

Home          https://github.com/galaxyproject/galaxy-lib

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install galaxy-lib

and update with::

   conda update galaxy-lib



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/galaxy-lib.
