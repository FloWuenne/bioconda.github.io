r-kriging
=========

|downloads|

Simple and highly optimized ordinary kriging algorithm to plot geographical data

============= ===========
Home          https://CRAN.R-project.org/package=kriging
Versions      1.1
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//r-kriging/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-kriging

and update with::

   conda update r-kriging



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-kriging.
