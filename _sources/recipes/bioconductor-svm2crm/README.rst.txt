====================

Detection of cis\-regulatory elements using svm implemented in LiblineaR.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-svm2crm

and update with::

   conda update bioconductor-svm2crm



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-svm2crm.
