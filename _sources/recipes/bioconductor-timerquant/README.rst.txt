bioconductor-timerquant
=======================

|downloads|

Supplementary Data package for tandem timer methods paper by Barry et al. \(2015\) including TimerQuant shiny applications.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/TimerQuant.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-timerquant



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-timerquant

and update with::

   conda update bioconductor-timerquant



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-timerquant.
