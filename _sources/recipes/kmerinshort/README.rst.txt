kmerinshort
===========

|downloads|

KmerInShort counts kmers from a fasta\/fastq file or list of files\, and outputs results in a text file. It is limited to short kmers \(k\<15\). It is a part of the FEELnc pipeline \(V.Wucher et al.\)
