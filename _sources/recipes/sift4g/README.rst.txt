.. _`sift4g`:

sift4g
======

|downloads|

SIFT 4G is a faster version of SIFT that enables us to scale up and provide SIFT predictions for more organisms.

============= ===========
Home          http://sift.bii.a-star.edu.sg/sift4g/
Versions      2.0.0
License       GPL-3.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//sift4g/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install sift4g

and update with::

   conda update sift4g



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/sift4g.
