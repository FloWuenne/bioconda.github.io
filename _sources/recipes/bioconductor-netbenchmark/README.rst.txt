bioconductor-netbenchmark
=========================

|downloads|

This package implements a benchmarking of several gene network inference algorithms from gene expression data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/netbenchmark.html
Versions      
License       CC BY-NC-SA 4.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-netbenchmark



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-netbenchmark

and update with::

   conda update bioconductor-netbenchmark



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-netbenchmark.
