bctools
=======

|downloads|

Tools for handling barcodes in NGS data.

============= ===========
Home          https://github.com/dmaticzka/bctools
Versions      0.2.1
License       Apache 2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bctools



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bctools

and update with::

   conda update bctools



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bctools.
