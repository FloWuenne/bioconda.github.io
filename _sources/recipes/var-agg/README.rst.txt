.. _`var-agg`:

var-agg
=======

|downloads|

A simple helper for aggregating multi\-sample VCF files into \"site VCF\" files.


============= ===========
Home          https://github.com/bihealth/var-agg
Versions      0.1.1, 0.1.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/var-agg



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install var-agg

and update with::

   conda update var-agg



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/var-agg.
