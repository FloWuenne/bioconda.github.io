bioconductor-yaqcaffy
=====================

|downloads|

Quality control of Affymetrix GeneChip expression data and reproducibility analysis of human whole genome chips with the MAQC reference datasets.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/yaqcaffy.html
Versions      1.40.0, 1.38.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-yaqcaffy



Links         biotools: :biotools:`yaqcaffy`, doi: :doi:`10.1038/nmeth.3252`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-yaqcaffy

and update with::

   conda update bioconductor-yaqcaffy



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-yaqcaffy.
