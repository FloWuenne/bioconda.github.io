binstrings
==========

|downloads|

Convert fasta strings into dinucleotide binary indicator string as 00101...

============= ===========
Home          https://github.com/erinijapranckeviciene/dnpatterntools
Versions      1.0
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/dnp-binstrings



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install binstrings

and update with::

   conda update binstrings



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/binstrings.
