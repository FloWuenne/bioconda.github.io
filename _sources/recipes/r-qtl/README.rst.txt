.. _`r-qtl`:

r-qtl
=====

|downloads|

Analysis of experimental crosses to identify genes \(called quantitative trait loci\, QTLs\) contributing to variation in quantitative traits\.

======== ===========
Home     http://rqtl.org
Versions 1.41_6
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-qtl

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-qtl

and update with::

   conda update r-qtl



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/r-qtl.
