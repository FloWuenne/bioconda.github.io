bioconductor-levi
=================

|downloads|

The tool integrates data from biological networks with transcriptomes\, displaying a heatmap with surface curves to evidence the altered regions.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/levi.html
Versions      1.0.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-levi/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-levi

and update with::

   conda update bioconductor-levi



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-levi.
