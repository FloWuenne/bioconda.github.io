bioconductor-hgfocuscdf
=======================

A package containing an environment representing the HG\-Focus.CDF file.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/annotation/html/hgfocuscdf.html
Versions      2.18.0
License       LGPL
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-hgfocuscdf/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-hgfocuscdf

and update with::

   conda update bioconductor-hgfocuscdf

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-hgfocuscdf.
