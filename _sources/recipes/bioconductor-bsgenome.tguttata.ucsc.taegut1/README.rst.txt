bioconductor-bsgenome.tguttata.ucsc.taegut1
===========================================

|downloads|

Full genome sequences for Taeniopygia guttata \(Zebra finch\) as provided by UCSC \(taeGut1\, Jul. 2008\) and stored in Biostrings objects.

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.tguttata.ucsc.taegut1

and update with::

   conda update bioconductor-bsgenome.tguttata.ucsc.taegut1
