.. _`r-openxlsx`:

r-openxlsx
==========

|downloads|

Simplifies the creation of Excel .xlsx files by providing a high level interface to writing, styling and editing worksheets. Through the use of 'Rcpp', read/write times are comparable to the 'xlsx' and 'XLConnect' packages with the added benefit of removing the dependency on Java.

======== ===========
Home     https
Versions 4.0.17
License  GPL-3
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-openxlsx
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install r-openxlsx

and update with::

   conda update r-openxlsx

A Docker container is available at https://quay.io/repository/biocontainers/r-openxlsx.
