bioconductor-aneufinder
=======================

|downloads|

AneuFinder implements functions for copy\-number detection\, breakpoint detection\, and karyotype and heterogeneity analysis in single\-cell whole genome sequencing and strand\-seq data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/AneuFinder.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-aneufinder



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-aneufinder

and update with::

   conda update bioconductor-aneufinder

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-aneufinder.
