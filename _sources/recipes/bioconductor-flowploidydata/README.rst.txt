bioconductor-flowploidydata
===========================

|downloads|

A collection of raw flow cytometry data for use in vignettes for the flowPloidy package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/flowPloidyData.html
Versions      1.8.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-flowploidydata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowploidydata

and update with::

   conda update bioconductor-flowploidydata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowploidydata.
