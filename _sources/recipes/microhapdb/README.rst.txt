microhapdb
==========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install microhapdb

and update with::

   conda update microhapdb



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/microhapdb.
