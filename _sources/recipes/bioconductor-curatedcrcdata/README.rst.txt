bioconductor-curatedcrcdata
===========================

|downloads|

The curatedCRC package provides relevant functions and data for gene expression analysis in patients with colorectal cancer.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/curatedCRCData.html
Versions      2.14.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-curatedcrcdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-curatedcrcdata

and update with::

   conda update bioconductor-curatedcrcdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-curatedcrcdata.
