bioconductor-tabulamurisdata
============================

|downloads|

Access to processed 10x \(droplet\) and SmartSeq2 \(on FACS\-sorted cells\) single\-cell RNA\-seq data from the Tabula Muris consortium \(http\:\/\/tabula\-muris.ds.czbiohub.org\/\).

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/TabulaMurisData.html
Versions      
License       MIT + file LICENSE
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/bioconductor-tabulamurisdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-tabulamurisdata

and update with::

   conda update bioconductor-tabulamurisdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-tabulamurisdata.
