bioconductor-targetsearchdata
=============================

|downloads|

This package provides example GC\-MS data for TargetSearch Package.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/TargetSearchData.html
Versions      1.20.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-targetsearchdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-targetsearchdata

and update with::

   conda update bioconductor-targetsearchdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-targetsearchdata.
