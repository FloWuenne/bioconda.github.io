Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install kpal

and update with::

   conda update kpal



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/kpal.
