.. _`bioconductor-etec16s`:

bioconductor-etec16s
====================

|downloads|

16S rRNA gene sequencing data to study changes in the faecal microbiota of 12 volunteers during a human challenge study with ETEC \(H10407\) and subsequent treatment with ciprofloxacin.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/etec16s.html
Versions      1.10.0
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-etec16s/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-etec16s

and update with::

   conda update bioconductor-etec16s



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-etec16s.
