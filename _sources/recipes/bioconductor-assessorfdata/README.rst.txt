bioconductor-assessorfdata
==========================

|downloads|

This package provides access to mapping and results objects generated by the AssessORF package\, as well as the genome sequences for the strains corresponding to those objects.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/AssessORFData.html
Versions      1.0.0
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-assessorfdata/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-assessorfdata

and update with::

   conda update bioconductor-assessorfdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-assessorfdata.
