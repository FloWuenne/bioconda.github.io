.. _`tombo`:

tombo
=====

|downloads|

Tombo is a suite of tools primarily for the identification of modified nucleotides from raw nanopore sequencing data.

======== ===========
Home     https://github.com/nanoporetech/tombo
Versions 1.0
License  MPL-2.0
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/tombo
======== ===========

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install tombo

and update with::

   conda update tombo



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/tombo.
