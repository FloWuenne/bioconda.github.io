bioconductor-fantom3and4cage
============================

|downloads|

CAGE \(Cap Analysis Gene Expression\) data from FANTOM3 and FANTOM4 projects produced by RIKEN Omics Science Center.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/FANTOM3and4CAGE.html
Versions      
License       GPL-3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-fantom3and4cage/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-fantom3and4cage

and update with::

   conda update bioconductor-fantom3and4cage



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-fantom3and4cage.
