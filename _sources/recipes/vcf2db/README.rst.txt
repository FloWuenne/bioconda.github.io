Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vcf2db

and update with::

   conda update vcf2db

A Docker container is available at https://quay.io/repository/biocontainers/vcf2db.
