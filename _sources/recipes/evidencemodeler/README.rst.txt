evidencemodeler
===============

|downloads|

Evidence Modeler combines ab intio gene predictions\, protein alignments\, and transcript alignments into weighted consensus gene structures

============= ===========
Home          https://github.com/EVidenceModeler/EVidenceModeler
Versions      v1.1.1
License       BSD 3-Clause
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/evidencemodeler



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install evidencemodeler

and update with::

   conda update evidencemodeler



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/evidencemodeler.
