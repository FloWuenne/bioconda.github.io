bioconductor-flowvs
===================

|downloads|

Per\-channel variance stabilization from a collection of flow cytometry samples by Bertlett test for homogeneity of variances. The approach is applicable to microarrays data as well.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/flowVS.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//bioconductor-flowvs/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-flowvs

and update with::

   conda update bioconductor-flowvs



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-flowvs.
