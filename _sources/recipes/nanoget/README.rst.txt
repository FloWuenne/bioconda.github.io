nanoget
=======

|downloads|

Functions to extract information from Oxford Nanopore sequencing data and alignments.

============= ===========
Home          https://github.com/wdecoster/nanoget
Versions      1.7.6, 1.7.4, 1.5.2, 1.5.0, 1.2.2, 1.0.2, 1.0.0, 0.11.7, 0.11.5
License       MIT License
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//nanoget/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install nanoget

and update with::

   conda update nanoget



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/nanoget.
