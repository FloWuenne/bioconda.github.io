.. _`bioconductor-ruvcorr`:

bioconductor-ruvcorr
====================

|downloads|

RUVcorr allows to apply global removal of unwanted variation \(ridged version of RUV\) to real and simulated gene expression data.

============= ===========
Home          https://bioconductor.org/packages/3.8/bioc/html/RUVcorr.html
Versions      
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-ruvcorr



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-ruvcorr

and update with::

   conda update bioconductor-ruvcorr



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-ruvcorr.
