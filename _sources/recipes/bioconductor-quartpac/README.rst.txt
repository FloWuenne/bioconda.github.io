bioconductor-quartpac
=====================

|downloads|

Identifies clustering of somatic mutations in proteins over the entire quaternary structure.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/QuartPAC.html
Versions      1.10.0
License       GPL-2
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-quartpac



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-quartpac

and update with::

   conda update bioconductor-quartpac



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-quartpac.
