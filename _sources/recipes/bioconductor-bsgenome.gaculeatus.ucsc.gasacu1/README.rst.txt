bioconductor-bsgenome.gaculeatus.ucsc.gasacu1
=============================================

|downloads|

Full genome sequences for Gasterosteus aculeatus \(Stickleback\) as provided by UCSC \(gasAcu1\, Feb. 2006\) and stored in Biostrings objects.

Home          https://bioconductor.org/packages/3.8/data/annotation/html/BSgenome.Gaculeatus.UCSC.gasAcu1.html

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-bsgenome.gaculeatus.ucsc.gasacu1

and update with::

   conda update bioconductor-bsgenome.gaculeatus.ucsc.gasacu1

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-bsgenome.gaculeatus.ucsc.gasacu1.
