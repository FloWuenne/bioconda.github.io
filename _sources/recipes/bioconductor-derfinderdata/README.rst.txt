.. _`bioconductor-derfinderdata`:

bioconductor-derfinderdata
==========================

|downloads|

Processed 22 samples from BrainSpan keeping only the information for chromosome 21. Data is stored in the BigWig format and is used for examples in other packages.

============= ===========
Home          https://bioconductor.org/packages/3.8/data/experiment/html/derfinderData.html
Versions      
License       Artistic-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-derfinderdata



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-derfinderdata

and update with::

   conda update bioconductor-derfinderdata



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-derfinderdata.
