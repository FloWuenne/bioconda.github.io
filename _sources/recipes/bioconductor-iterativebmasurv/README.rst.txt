bioconductor-iterativebmasurv
=============================

|downloads|

The iterative Bayesian Model Averaging \(BMA\) algorithm for survival analysis is a variable selection method for applying survival analysis to microarray data.

============= ===========
Home          http://bioconductor.org/packages/3.6/bioc/html/iterativeBMAsurv.html
Versions      1.36.0, 1.38.0
License       GPL (>= 2)
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/bioconductor-iterativebmasurv



Links         biotools: :biotools:`iterativebmasurv`, doi: :doi:`10.1186/1471-2105-10-72`

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install bioconductor-iterativebmasurv

and update with::

   conda update bioconductor-iterativebmasurv



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/bioconductor-iterativebmasurv.
