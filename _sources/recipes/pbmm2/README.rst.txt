A minimap2 frontend for PacBio native data formats

License       BSD-3-Clause-Clear

Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install pbmm2

and update with::

   conda update pbmm2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/pbmm2.
