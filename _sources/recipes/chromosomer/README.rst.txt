chromosomer
===========

|downloads|

A reference\-assisted assembly tool for producing draft chromosome sequences.

============= ===========
Home          https://github.com/gtamazian/chromosomer
Versions      0.1.4a
License       MIT
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes/chromosomer



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install chromosomer

and update with::

   conda update chromosomer

A Docker container is available at https://quay.io/repository/biocontainers/chromosomer.
