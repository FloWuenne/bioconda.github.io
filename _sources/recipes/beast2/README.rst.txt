beast2
======

|downloads|

BEAST 2 is a cross\-platform program for Bayesian phylogenetic analysis of molecular sequences\.

======== ===========
Home     http://www.beast2.org
Versions 2.4.5
License  LGPLv2.1
Recipe   https://github.com/bioconda/bioconda-recipes/tree/master/recipes/beast2

======== ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install beast2

and update with::

   conda update beast2



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/beast2.
