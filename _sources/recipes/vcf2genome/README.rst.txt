.. _`vcf2genome`:

vcf2genome
==========

|downloads|

A tool to create a draft genome file out of a GATK VCF file and enabling users to filter the VCF in a single step.

============= ===========
Home          https://github.com/apeltzer/vcf2genome
Versions      0.91
License       GPLv3
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//home/circleci/project/docs/bioconda-recipes/recipes/vcf2genome/meta.yaml



============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install vcf2genome

and update with::

   conda update vcf2genome



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/vcf2genome.
