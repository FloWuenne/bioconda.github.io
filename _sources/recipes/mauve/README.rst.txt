mauve
=====

|downloads|

Mauve is a system for constructing multiple genome alignments in the presence of large\-scale evolutionary events such as rearrangement and inversion

============= ===========
Home          http://darlinglab.org/mauve/
Versions      2.4.0.r4736
License       GPL-2.0
Recipe        https://github.com/bioconda/bioconda-recipes/tree/master/recipes//mauve/meta.yaml


Development   https://sourceforge.net/projects/mauve/


============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install mauve

and update with::

   conda update mauve

A Docker container is available at https://quay.io/repository/biocontainers/mauve.
