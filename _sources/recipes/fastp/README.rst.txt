fastp

============= ===========



Installation
------------

.. highlight: bash

With an activated Bioconda channel (see :ref:`set-up-channels`), install with::

   conda install fastp

and update with::

   conda update fastp



|docker|

A Docker container is available at https://quay.io/repository/biocontainers/fastp.
