recipe bioconductor-xcms¶

Framework for processing and visualization of chromatographically separated and single-spectra mass spectral data. Imports from AIA/ANDI NetCDF, mzXML, mzData and mzML files. Preprocesses data for high-throughput, untargeted analyte profiling.

Homepage

https://bioconductor.org/packages/3.9/bioc/html/xcms.html

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-xcms

and update with:

conda update bioconductor-xcms

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-xcms:<tag>

(see bioconductor-xcms/tags for valid values for <tag>)
