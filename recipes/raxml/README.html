Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install raxml

and update with:

conda update raxml

or use the docker container:

docker pull quay.io/biocontainers/raxml:<tag>

(see raxml/tags for valid values for <tag>)
