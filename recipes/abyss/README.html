Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install abyss

and update with:

conda update abyss

or use the docker container:

docker pull quay.io/biocontainers/abyss:<tag>

(see abyss/tags for valid values for <tag>)
