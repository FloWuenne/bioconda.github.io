Installation

With an activated Bioconda channel (see set-up-channels), install with:

conda install mitoz

and update with:

conda update mitoz

or use the docker container:

docker pull quay.io/biocontainers/mitoz:<tag>

(see mitoz/tags for valid values for <tag>)
