Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install khmer

and update with:

conda update khmer

or use the docker container:

docker pull quay.io/biocontainers/khmer:<tag>

(see khmer/tags for valid values for <tag>)
