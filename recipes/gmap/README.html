Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install gmap

and update with:

conda update gmap

or use the docker container:

docker pull quay.io/biocontainers/gmap:<tag>

(see gmap/tags for valid values for <tag>)
