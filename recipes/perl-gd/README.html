package perl-gd¶

Versions:

2.70-0, 2.69-0, 2.68-0, 2.56-9, 2.56-8, 2.56-7, 2.56-6, 2.56-5, 2.56-4, 2.56-3, 2.56-0
Depends:
  • libgd >=2.2.5,<2.3.0a0
  • libstdcxx-ng >=4.9
  • perl >=5.26.2,<5.26.3.0a0
  • zlib >=1.2.11,<1.3.0a0
Required By:
  • circos
  • perl-gdgraph
  • perl-bio-phylo
  • protrac
  • fastx_toolkit
  • perl-gdtextutil
  • perl-pcap
  • perl-bioperl-core

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install perl-gd

and update with:

conda update perl-gd

or use the docker container:

docker pull quay.io/biocontainers/perl-gd:<tag>

(see perl-gd/tags for valid values for <tag>)
