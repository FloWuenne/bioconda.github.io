Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install mafft

and update with:

conda update mafft

or use the docker container:

docker pull quay.io/biocontainers/mafft:<tag>

(see mafft/tags for valid values for <tag>)
