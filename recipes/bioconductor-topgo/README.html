biotools: topgo, doi: 10.1093/bioinformatics/btl140

topGO package provides tools for testing GO terms while accounting for the topology of the GO graph. Different test statistics and different methods for eliminating local similarities and dependencies between GO terms can be implemented and applied.

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-topgo

and update with:

conda update bioconductor-topgo

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-topgo:<tag>

(see bioconductor-topgo/tags for valid values for <tag>)
