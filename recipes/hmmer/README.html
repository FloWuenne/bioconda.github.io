Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install hmmer

and update with:

conda update hmmer

or use the docker container:

docker pull quay.io/biocontainers/hmmer:<tag>

(see hmmer/tags for valid values for <tag>)
