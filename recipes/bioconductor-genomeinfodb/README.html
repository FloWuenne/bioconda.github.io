recipe bioconductor-genomeinfodb

Utilities for manipulating chromosome names, including modifying them to follow a particular naming style

Homepage

https://bioconductor.org/packages/3.12/bioc/html/GenomeInfoDb.html

License

Artistic-2.0

Recipe

/bioconductor-genomeinfodb/meta.yaml

Links

biotools: genomeinfodb, doi: 10.1038/nmeth.3252

Contains data and functions that define and allow translation between different chromosome sequence naming conventions (e.g., "chr1" versus "1"), including a function that attempts to place sequence names in their natural, rather than lexicographic, order.

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-genomeinfodb

and update with:

conda update bioconductor-genomeinfodb

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-genomeinfodb:<tag>

(see bioconductor-genomeinfodb/tags for valid values for <tag>)
