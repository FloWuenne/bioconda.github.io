Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install mauve

and update with:

conda update mauve

or use the docker container:

docker pull quay.io/biocontainers/mauve:<tag>

(see mauve/tags for valid values for <tag>)
