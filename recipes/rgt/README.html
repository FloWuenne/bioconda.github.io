Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install rgt

and update with:

conda update rgt

or use the docker container:

docker pull quay.io/biocontainers/rgt:<tag>

(see rgt/tags for valid values for <tag>)
