Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bwa

and update with:

conda update bwa

or use the docker container:

docker pull quay.io/biocontainers/bwa:<tag>

(see bwa/tags for valid values for <tag>)
