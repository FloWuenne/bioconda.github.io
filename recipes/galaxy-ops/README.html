galaxy-ops¶

Galaxy interval operations

Home https://github.com/galaxyproject/gops
Versions 1.0.0
License Academic Free License version 3.0
Recipe https://github.com/bioconda/bioconda-recipes/tree/master/recipes/galaxy-ops

Installation¶

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install galaxy-ops

and update with:

conda update galaxy-ops

A Docker container is available at https://quay.io/repository/biocontainers/galaxy-ops.
