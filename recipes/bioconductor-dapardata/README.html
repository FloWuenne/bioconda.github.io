package bioconductor-dapardata¶

Versions

1.20.0-0,  1.18.0-0,  1.16.0-0,  1.14.0-1,  1.12.1-0

Depends

  • bioconductor-msnbase >=2.16.0,<2.17.0

  • curl >=7.71.1,<8.0a0

  • r-base >=4.0,<4.1.0a0

  • r-knitr

Required By

  • bioconductor-dapar

  • bioconductor-prostar

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-dapardata

and update with:

conda update bioconductor-dapardata

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-dapardata:<tag>

(see bioconductor-dapardata/tags for valid values for <tag>)
