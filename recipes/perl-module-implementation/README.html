Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install perl-module-implementation

and update with:

conda update perl-module-implementation

or use the docker container:

docker pull quay.io/biocontainers/perl-module-implementation:<tag>

(see perl-module-implementation/tags for valid values for <tag>)
