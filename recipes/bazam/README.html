Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bazam

and update with:

conda update bazam

or use the docker container:

docker pull quay.io/biocontainers/bazam:<tag>

(see bazam/tags for valid values for <tag>)
