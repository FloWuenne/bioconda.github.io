r-car¶

Functions and Datasets to Accompany J. Fox and S. Weisberg, An R Companion to Applied Regression, Second Edition, Sage, 2011.
Home https://r-forge.r-project.org/projects/car/, https://CRAN.R-project.org/package=car, http://socserv.socsci.mcmaster.ca/jfox/Books/Companion/index.html
Versions 2.1_4
License GPL (>= 2)
Recipe https://github.com/bioconda/bioconda-recipes/tree/master/recipes/r-car

Installation¶

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install r-car

and update with:

conda update r-car

A Docker container is available at https://quay.io/repository/biocontainers/r-car.
