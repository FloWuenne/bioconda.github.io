Required By

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install fgmp

and update with:

conda update fgmp

or use the docker container:

docker pull quay.io/biocontainers/fgmp:<tag>

(see fgmp/tags for valid values for <tag>)
