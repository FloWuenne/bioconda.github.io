package perl-xml-dom¶

Versions

1.46-0, 1.45-1, 1.45-0

Depends

  • perl >=5.26.2,<5.26.3.0a0

  • perl-libwww-perl

  • perl-libxml-perl

  • perl-xml-parser

  • perl-xml-regexp

Required By

  • rsat-core

  • perl-cg-pipeline

  • perl-bioperl-core

  • perl-xml-dom-xpath

  • perl-bio-featureio

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install perl-xml-dom

and update with:

conda update perl-xml-dom

or use the docker container:

docker pull quay.io/biocontainers/perl-xml-dom:<tag>

(see perl-xml-dom/tags for valid values for <tag>)
