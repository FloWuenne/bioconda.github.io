Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install perl-biox-workflow-command

and update with:

conda update perl-biox-workflow-command

or use the docker container:

docker pull quay.io/biocontainers/perl-biox-workflow-command:<tag>

(see perl-biox-workflow-command/tags for valid values for <tag>)
