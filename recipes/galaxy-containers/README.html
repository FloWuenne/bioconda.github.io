package galaxy-containers¶

Versions

21.1.0-0,  20.5.0-0,  19.9.0-0

Depends

  • galaxy-util >=21.1

  • python >=3.5

  • requests

Required By

  • planemo

  • galaxy-tool-util

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install galaxy-containers

and update with:

conda update galaxy-containers

or use the docker container:

docker pull quay.io/biocontainers/galaxy-containers:<tag>

(see galaxy-containers/tags for valid values for <tag>)
