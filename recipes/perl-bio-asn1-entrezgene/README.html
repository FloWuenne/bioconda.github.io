perl-bio-asn1-entrezgene¶

Regular expression-based Perl Parser for NCBI Entrez Gene

Home http://search.cpan.org/dist/Bio-ASN1-EntrezGene
Versions 1.70, 1.72
License perl_5
Recipe https://github.com/bioconda/bioconda-recipes/tree/master/recipes/perl-bio-asn1-entrezgene

Installation¶

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install perl-bio-asn1-entrezgene

and update with:

conda update perl-bio-asn1-entrezgene

A Docker container is available at https://quay.io/repository/biocontainers/perl-bio-asn1-entrezgene.
