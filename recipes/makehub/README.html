recipe makehub¶

MakeHub is a command line tool for the fully automatic generation of of track data hubs for visualizing genomes with the UCSC genome browser.

Homepage:https://github.com/Gaius-Augustus/MakeHub
License:GPL / GPL-3.0
Recipe:/makehub/meta.yaml
package makehub¶

Versions:

1.0.3-0
Depends:
  • augustus >=3.3.1
  • biopython
  • python >=3
  • samtools
  • ucsc-bedtobigbed
  • ucsc-fatotwobit
  • ucsc-genepredcheck
  • ucsc-genepredtobed
  • ucsc-genepredtobiggenepred

  • ucsc-hggcpercent

Required By:
  • braker2

Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install makehub

and update with:

conda update makehub

or use the docker container:

docker pull quay.io/biocontainers/makehub:<tag>

(see makehub/tags for valid values for <tag>)
