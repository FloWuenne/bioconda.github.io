Installation

With an activated Bioconda channel (see 2. Set up channels), install with:

conda install bioconductor-txdb.hsapiens.ucsc.hg38.knowngene

and update with:

conda update bioconductor-txdb.hsapiens.ucsc.hg38.knowngene

or use the docker container:

docker pull quay.io/biocontainers/bioconductor-txdb.hsapiens.ucsc.hg38.knowngene:<tag>

(see bioconductor-txdb.hsapiens.ucsc.hg38.knowngene/tags for valid values for <tag>)
