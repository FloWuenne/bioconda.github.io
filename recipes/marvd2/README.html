Installation

With an activated Bioconda channel (see set-up-channels), install with:

conda install marvd2

and update with:

conda update marvd2

or use the docker container:

docker pull quay.io/biocontainers/marvd2:<tag>

(see marvd2/tags for valid values for <tag>)
